# Rüşvet esprileri yüzünden poşet taşıyamıyorum

*Zekai Özçınar 2005.02.07*

<div bgcolor="#FFFEF6">
 <font>
  <img border="0" height="1" src="/web/20051226034739im_/http://www.aksiyon.com.tr/images/blank.gif"/>
 </font>
 <font class="content">
  <p>
   <img border="0" hspace="5" src="http://web.archive.org/web/20051226034739im_/http://www.aksiyon.com.tr/resim/531/21.jpg" vspace="5"/>
  </p>
 </font>
 <font class="content">
  CHP Yüksek Disiplin Kurulu'nun (YDK) adı sanı bilinmeyen üyelerinden biriydi Ergün Aydoğan. Mustafa Sarıgül'ü aklayan kararın ardından, "kendisine poşet içinde rüşvet verildiğini" açıkladı.
 </font>
 <p>
  <font class="content">
   Kurultay sürecini tetiklemekle kalmadı, Türkiye onu konuştu. Ödülü de, yeniden YDK üyeliğine seçilmesi oldu. Aydoğan'la rüşveti konuştuk; ibretlik bir tablo çıktı karşımıza. Bu arada, dürüstlüğünden şüphe edilmez 'memleket insanımın' rüşveti nasıl algıladığı konusunda da fikir sahibi olduk.
   <br>
    <br>
     -Poşeti açtığınızda neler hissettiniz?
     <br/>
     <br/>
     Canım sıkıldı, olduğu gibi koydum. Eşim de, aynı tepkiyi gösterdi. Uyku tutmadı. Sabah olunca da, o arkadaşlara iade ettim.
     <br/>
     <br/>
     -Yüklü bir miktar mıydı?
     <br/>
     <br/>
     Sayma ihtiyacı hissetmedim. Bana ait olmayan şeyi niye sayayım, merak edeyim? Almayı düşünsem sayardım.
     <br/>
     <br/>
     -Dolar olduğu söylendi.
     <br/>
     <br/>
     Öyle bir şey yani. Sonuçta para. Bunun cinsi, olayın magazin tarafı. Hem, paraların çoğu yeşil. Paranın cinsiyle, miktarıyla ilgili değiliz.
     <br/>
     <br/>
     -Baykal'a söylediğinizde ilk tepkisi ne oldu?
     <br/>
     <br/>
     Çok önemli bir görev yaptığımı, partiye sahip çıktığımızı söyledi.
     <br/>
     <br/>
     -Sadece size mi teklif edilmiş?
     <br/>
     <br/>
     Bilemem, yorum da yapamam.
     <br/>
     <br/>
     -İnsanlar, bu davranışınızı nasıl karşıladı?
     <br/>
     <br/>
     "İyi yaptın." diyenlerin sayısı az. "Miktarı az mı buldun? Niye verdin? Alıp istediğin yönde oy kullansaydın. Dürüstlük sana mı kaldı?" dediler. Tabii ki, takdir edenler de çıktı. Yaşadığımız toplum ne kadar dürüstse, siyaset de o kadar dürüst olur.
     <br/>
     <br/>
     -Bir 'enayi' denmediği kalmış, öyle mi?
     <br/>
     <br/>
     Popüler kültür, televole kültürü ortamında, egemen güç para. Her şeyi para belirliyor. Böyle bir toplumda, bu tür düşünceler de  yozlaşma da normal. Bazı insanların zafiyet göstermesi de doğal. Aslında, yaptığımız çok anormal bir davranış değil. Her vatandaşın yapması gerekeni yaptık. Kahramanlık payesi de beklemiyoruz.
     <br/>
     <br/>
     -Trajikomik bir durum söz konusu anlaşılan.
     <br/>
     <br/>
     Toplum öyle yozlaşmış ki, birileri kahramanlaştırıyor, birileri enayileştiriyor. Eğer, 'paranın her kapıyı açtığı' anlayışı yerleşmişse, insanlar bu tür şeylere yöneliyor. Farklı davranırsanız da espri konusu oluyor.
     <br/>
     <br/>
     -Biraz açar mısınız?
     <br/>
     <br/>
     Şimdi, elimde poşet taşımaya çekiniyorum. Poşet, espiri konusu oluyor. "İçinde ne var?" diye soruyorlar, parayı ima ederek. Ben de, gülerek içindekileri gösteriyorum.
     <br/>
     <br/>
     -Artık, poşette hediye de kabul etmezsiniz.
     <br/>
     <br/>
     Herhalde, düşünürüz.
     <br/>
     <br/>
     -Yılın 'dürüstlük' ödülüne aday mısınız?
     <br/>
     <br/>
     Talebimiz yok, gelirse alırız. Poşetin dışında olursa alırız (gülüyor).
     <br/>
     <br/>
     -Bu arada, YDK'nın iş yükü artacak. Kapınız çalınır mı?
     <br/>
     <br/>
     Gelemezler, mümkün mü? Alacakları cevap belli. Bakın, ben YDK üyeliği dahil hiçbir makam için talepte bulunmadım. Sayın genel başkan uygun görmüş.
     <br/>
     <br/>
     -Peki, bu süreçte hiç tehdit aldınız mı?
     <br/>
     <br/>
     Endişemiz oldu; ama tehdit almadık.
     <br/>
     <br/>
     -Kurultayda kan gövdeyi götürdü. Gözler üzerinizdeydi, endişeye kapıldınız mı?
     <br/>
     <br/>
     İlk gün yaşanan olayların tedirginliğiyle otelimizi (Sarıgül taraftarlarının kaldığı bir oteldi) değiştirdik. Ama, fiili bir müdahaleye maruz kalmadım. Bu arada, uyaran arkadaşlar oldu. Kavganın olduğu saatlerde bir mesaj aldım, "Başkan, kendine dikkat et." diye. Karşı taraf dava açmış. Sonucu göreceğiz.
     <br/>
     <br/>
     -O parayı alsanız, bu kadar rahat olamazdınız herhalde.
     <br/>
     <br/>
     Mümkün mü? Vicdanen çok rahatım, hiç bu kadar rahat olmadım. Sonucu ne olursa olsun.
     <br/>
     <br/>
     -Sizce, rüşvet hastalığından kurtulmak mümkün mü?
     <br/>
     <br/>
     İşin ahlâkî, ekonomik ve eğitim boyutu var. İnsanların önce vicdanlarına sorması gerekiyor. Aslında, topyekûn mücadele şart. Tek kişinin münferit olarak ortaya çıkmasıyla bu olay bitmiyor, bitmez de. Polisiye tedbirlerle zaten olmaz. Bir hukukçu arkadaşım, "Polis çağırsanız da bir şey olmaz." dedi. Birileri ekmek bulamaz, birileri lüks içinde yaşar, televizyonlar da tevelole programlarıyla bunu sergilerse, bu tip alış verişler devam eder.
     <br/>
     <br/>
     Aydoğan sözlerini bağlarken cep telefonuna yeni bir mesaj düşüyor. Merak ediyoruz, bizi kırmıyor. Mesajı okuyor: "Başta 460 kahraman kurultay delegesi olmak üzere, kurultayımıza göstermiş olduğunuz ilgi ve desteğe teşekkür ederim... Musfata Sarıgül." Kaderin cilvesi bu olsa gerek...
