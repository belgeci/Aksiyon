# "Ölü çocuk resimleri"

*Adem Yavuz Arslan 1999.08.28*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   Çocuk, yani kutsal emaneti Allah'ın. Bizi göğün katlarına çeken ses, incecik çığlık...
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   KENT...
   <br/>
   <br/>
   Emanetini yitiren adamların, kadınların yaşadığı yer.
   <br/>
   <br/>
   Çocuklarsa, daha çok merdiven boşluklarında unutulmuş. Kapı aralıklarında, soğuk odaların beton duvarları arasında... Kuş sesleri uzak geliyor çocuklara. Denizler, gökyüzü, annelerin öpücükleri çok uzak... Arabaların ve apartmanların hüküm sürdüğü kentlerde, okula gitmek ve yaramazlık yapmamak için yaşıyor çocuklar... Erkenden büyümek için...
   <br/>
   <br/>
   ANNE...
   <br/>
   <br/>
   Elinden tutuyor bütün çocukların. Sevgisi gökyüzünden daha mavi, denizlerden daha engin. Apartman devlerini sevimli kılan peri. Elinden tutuyor bütün çocuklar, alınlarından öpüyor anneleri...
   <br/>
   <br/>
   MERDİVEN...
   <br/>
   <br/>
   Dalgacı müteahhitlerin "aritmetik öğretir çocuklara merdivenler" sözlerini hatırlayarak çıkıyoruz gökdelenlere. Asansörlerin sustuğu zamanlarda basamaklar çarpıyor ayaklarımıza. Işıkların bedenimizden canımızı çekip alır gibi kesilivermesi, yanıltıyor bizi. Basamaklarından birer birer düşüyoruz merdivenlerin. Ya çocuklar, evlerin melek yüzlü nineleri... Onlar da öldüler mi? Merdiven boşluğuna düşerek geceleri...
   <br/>
   <br/>
   MİMAR...
   <br/>
   <br/>
   Toprakla gökyüzü arasında kuşları, ihtiyarları ve çocukları barındıran sanatçı. Ya müteahhitler. Kent soytarıları... Apartmanların mucidleri... Ölü çocuk resimlerinin katilleri... Merdiven boşluklarında gölgeleri Azrail'in nefesi gibi dolaşan betonperest kentliler...
   <br/>
   <br/>
   ANASIR-I ERBAA...
   <br/>
   <br/>
   Toprak, hava, su, ateş. Çocuklar ateş elbiselerini giyer gibi sokuluyorlar annelerine. Toprak, kuru bir mevsimi andırır gibi sarsılıyor geceleri... Su, yaşlanmış insanların ölüm uykusu. Hava, temizliyor yıkık dökük evleri... Deniz kabarıyor, kabarıyor deniz... Dermandan kesiliyor; sulara, toprağa yenik düşüyor ellerimiz.
   <br/>
   <br/>
   DEPREM...
   <br/>
   <br/>
   Yıkılınca her şeyi anlıyoruz yeniden. Apartmanların ezip geçtiğini çiçekleri. Kesip attığımız gül ağaçlarını da. Alınlarından öpmeyi unuttuğumuz çocukları. Merdiven boşluklarında bıraktığımız haminneler, masalcı dedeler geliyor gözlerimize. Gökyüzünün kurşun gibi ağırlığı, apartmanların beşikler gibi sallanırken, kentin üzerimize çöküvermesi... Kibirle dolaştığımız sokaklardan ölümden kaçar gibi kaçarak Allah'a sığınmamız... Kentlerde deprem, kalbimizin en gizli köşelerinde deprem. Binlerce kez yıkılıyoruz gecenin kıyısında, denizlerin, toprağın çağrısıyla...
   <br/>
   <br/>
   ENKAZ...
   <br/>
   <br/>
   Ölü çocuk resimleri topluyor anneler. Yerlebir olmuş apartman enkazlarının arasında... Çocuklar kayıp, merdiven boşluklarına düşmüş çoğu. Duvarların arasına sıkışıp kalmış. Uykularında geçmişler, Azrail'in açtığı pencerelerden...
   <br/>
   <br/>
   Ölü çocuk resimleri topluyor anneler. Ağlıyorlar, dövüyorlar dizlerini... Kayıp çocuklar, eski şarkılar, içli dualar sökün ediyor anılardan. Ağlıyor anneler, yitirilmiş emanetler, ölüm tacı giymiş çocuklar için...
   <br/>
   <br/>
   VE DUA...
   <br/>
   <br/>
   Allahım, kentlerin başımıza yıkıldığı şu günde sen bizi koru. Çocuklarımızın uykularını çalan kent soytarılarının maskelerini düşür. Yaşarken unuttuğumuz ve uzaklaştığımız toprak, ateş, hava ve suya bizi yaklaştır. Dağlarından, denizlerinden emin kıl bizi. Gökten gelen de yerden gelen de senin icadın. Çocuklar da... Sana söz veriyoruz, evlerimizin bahçelerinden çiçekler eksik olmayacak. Sana söz veriyoruz, sulara, gökyüzüne ve toprağa dokunmayacak zehirlerimiz. Gözlerimizin içine bakar gibi bakacağız ateşe... Sana söz veriyoruz, sen bizi bağışla. Çocuklarımızı koru...
   <br/>
