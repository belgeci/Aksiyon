# Amiga'nın dönüşü

*Necati Yılmaz 1998.09.12*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   PC tarafından tahttan indirilmeden önce bilgisayar piyasasında prensesliğini sürdüren Amiga, eski günlerine dönme çabasında.
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   Bu yıl bitmeden yenilikleriyle birlikte "merhaba" demeye hazırlanıyor.
   <br/>
   <br/>
   Bilgisayarla yeni tanışanların ismini bile duymadığı, eskiden bilenlerin ise PC'deki gelişmeleri görünce bıraktığı Amiga aslında piyasadan çekilmedi; sürekli imal edilip satıldı. Hatta otomobil sektöründeki Wolkswagen'ların tutkunları gibi bağımlıları oluştu. Onu hiçbir zaman terketmediler, üzerine and dahi içtiler. Web sayfaları ve tartışma guruplarıyla geliştirmeye, yaygınlaştırmaya çalıştılar.
   <br/>
   <br/>
   Alan firma battı
   <br/>
   <br/>
   Amiga'nın gelişmesinde bugüne kadar ilginç anekdotlar yaşandı. Tarihçesinde görüleceği gibi Amiga ilk üreticisi HiToro Şirketi mâli krize girince Mayıs 84'te Commodore'a satılır. Nisan 94'de Commodore iflas eder, Amiga'yı ESCOM satın alır. Haziran 96'da ESCOM da iflas eder Amiga adeta ortada kalır. Mayıs 97'de Gateway 2000 sahiplenir ve bu yıl firma Amiga ile ilgili projelerini açıklar. Amiga Şirketi'ni kurduğunu açıklayan Gateway 2000'in öncekiler gibi iflas edip etmeyeceğini ise zaman gösterecek.
   <br/>
   <br/>
   Yeri apayrı bir bilgisayar
   <br/>
   <br/>
   Amiga'nın tutkunlarına hak vermemek elde değil. Daha sonra geniş şekilde özelliklerini anlatmaya çalışacağımız Amiga halen PC'den üstün yönleriyle dikkatleri üzerine çekmekte. Ancak elde ediş maliyetinin yüksekliği en büyük dezavantajı.
   <br/>
   <br/>
   Amiga'nın hedefleri
   <br/>
   <br/>
   Gateway 2000 firması Amiga ile ilgili hedeflerini şu ana başlıklarla açıkladı:
   <br/>
   <br/>
   Yeni bir işletim sistemi,
   <br/>
   <br/>
   Muhtemelen intel tabanlı yeni bir bilgisayar,
   <br/>
   <br/>
   Daha sonra, intel tabanlı olmayan yeni bir işlemci,
   <br/>
   <br/>
   Tüketici uygulamaları,
   <br/>
   <br/>
   Donanımın üretimini üçüncü şahıslara açmak.
   <br/>
   <br/>
   Gateway'in kurduğu Amiga Firması'nın sözcüsü Bill McEwan, "Şirketimiz yonga ve bilgisayar üretmeyecek. Bizler bilgisayar mimarisi üzerine odaklanacağız, üçüncü şahıslar bunu üretecek" şeklindeki ifadesiyle stratejilerini özetledi.
   <br/>
   <br/>
   Microsoft ve İntel'e rakip olabilecek mi?
   <br/>
   <br/>
   İspanyolca 'Kız arkadaş' anlamına gelen Amiga için firması Kasım 98'de işletim sistemi AmigaOS'nin 4.0 sürümünü çıkarmayı planlıyor. 93'deki 3.1 sürümünün terfisi olacak işletim sistemi İntel veya PC uyumlu bir platform üzerinde çalışacak. Ayrıca bir emulatör veya isteğe bağlı kart ilavesiyle, "Amiga Klasiği" denilen platformda yazılmış eski programları da çalıştırabilecek. Ancak piyasaya sürülecek makine ve işletim sistemi ticari sürümden öte "developer's system" olacak ve AmigaOS 5.0 için bir basamak teşkil edecek.
   <br/>
   <br/>
   Silicon Prairie Dergisi'nden Osman Köroğlu arkadaşımızın çevirdiği makaleye göre online yayınlanan Amiga Report Magazine'in editörlerinden Jason Campton, Gateway'in yeni atılımı için rekabete dikkatleri çekiyor. Campton, "Sadece PC üreticisiyseniz pazarda zayıfsınız demektir. Bu yüzden Gateway teknoloji portfolyosunu genişletmek istiyor. Büyük ihtimalle Amiga'yı kimseye bağımlı olmadan özellikle İntel ve Microsoft'a, kendi platformunu kurabilme yolu olarak görüyor" şeklinde yorum getiriyor. AmigaOS 5.0 platformu hakkında firma sözcüsü McEwan ise birşey söylemiyor. Fakat diğer kaynaklar 'G4' ismi üzerinde duruyor. Bilindiği gibi G4, yakın zamanda çıkması beklenen Motorola'nın yeni nesil PowerPC işlemcisinin kod adı.
   <br/>
   <br/>
   Kız arkadaşımız Amiga'nın kendini yenilmesi, rekabet piyasasında ayakta duracak yeni değerlere sahip olması, en çok tutkunlarını belki de çok sayıda yeni kullanıcıları sevindirecek.
   <br/>
