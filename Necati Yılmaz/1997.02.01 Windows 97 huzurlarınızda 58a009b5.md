 <p>
  <font class="content">
   Microsoft firması Windows 97'yi piyasaya sürme hazırlıklarına başladı. Geçtiğimiz haftalarda piyasaya sunduğu Office 97 ile yazılımda atak yapan Microsoft firmasının bu yıl içerisinde Windows 97 başta olmak üzere daha pek çok yeni ürünü piyasaya sürmesi bekleniyor. Microsoft'un Başkanı Bill Gates bir süre önce yaptığı açıklamada 1997 yılında hem internet alanında, hem de diğer alanlarda piyasaya çok sayıda yazılım süreceklerini belirtti.
   <br>
    <br>
     İsmi henüz kararlaştırılmadı
     <br/>
     <br/>
     İsmi konusunda henüz net bir karar verilmeyen yeni yazılımın şu andaki kod adı  Memphis. Microsoft Windows Ürün Müdürü Phil Holden, firma olarak Windows 97 ismi üzerinde durduklarını söylüyor. Windows 95’ten stratejik açıdan pek farklı olmayan yeni işletim sistemi, PC işletim sistemlerine pek çok yenilik getiriyor.
     <br/>
     <br/>
     Ağustos’ta piyasaya çıkıyor
     <br/>
     <br/>
     Microsoft Windows Ürün Müdürü Phil Holden ürünün beta sürümünün mart ayı içerisinde hazır olacağını, kullanıcıların ise yeni işletim sistemiyle ağustos ayında buluşacağını söylüyor.
     <br/>
     <br/>
     Getirdiği yenilikler
     <br/>
     <br/>
     Windows 97 ana hatlarıyla şu yenilikleri içeriyor: Windows 97’de ev kullanıcıları dikkate alınıyor ve maalesef ülkemizde henüz bulunmayan ISDN hatlarının kullanımında kolaylıklar sağlanarak internete anında bağlanıp çalışma imkanı getiriliyor. ISDN 1.1 desteği, sisteme araç olarak yerleştirilen ISDN kurma sihirbazı ile gerçekleştiriliyor. Internet Connection Wizard (ICW) ile kullanıcı birkaç adımda kendi servis sağlayıcısının bilgilerini sistemine kaydedebiliyor veya mevcut internet servis sağlayıcı listesinden seçebiliyor. Böylece kullanıcı internet üzerinde anında çalışmaya başlayabiliyor.
     <br/>
     <br/>
     Internet Explorer 3.0 yerine Explorer 4.0 arabirimiyle çalışılıyor.
     <br/>
     <br/>
     Sistemin IDE bölümü geniş olarak IDE teyp yedekleme ünitelerine destek veriyor.
     <br/>
     <br/>
     Windows 95'in OEM sürümünden itibaren bilinen FAT—32 (File Allocation Table) Windows 97'ye de eklenmiş. DOS ve Windows 95 işletim sistemlerinde FAT—16 sistemi kullanılıyordu. Yeni sistem, sabit sürücülerin daha çok parçalara ayrılmasını ve sabit diski yüzde 20 daha etkin bir biçimde kullanarak daha çok veri depolamaya imkan veriyor.
     <br/>
     <br/>
     Windows 95’in OEM’ler için geliştirdiği OSR—1 ve OSR—2 sürümlerinde bulunan fare ve klavyeler için Universal Serial Bus (USB) desteği ve yeni FAT—32 disk formatları eklenmiş durumda. Oyun kurulumları için geliştirilmiş özellikler sağlayan DirectX ve Activemovie ise standart özellikler olarak sağlanmış.
     <br/>
     <br/>
     USB yeni işletim sisteminde video ve audio transferini de destekliyor.
     <br/>
     <br/>
     Mevcut VGA kartının yanına opsiyonel olarak ikinci, hatta üçüncü video adaptörünün takılabilmesine imkan veriyor. İlave video ekranlarının bilgisayarlara eklenebilme özelliği ise kullanıcının araç pencerelerinin bir ekranda bulunmasını, diğer bir ekranda ise uygulamaların çalıştırılabilmesini sağlıyor. Windows her iki ekran arasındaki bağlantıyı bir bağlantı yazılımı ile sağlıyor. İlave ekranlar oyun oynamakta kullanılabildiği gibi ekrandaki görüntüyü genişletmek için de kullanılabiliyor.
     <br/>
     <br/>
     Sistemi tekrar çalıştırmaya gerek kalmadan değiştirilebilen, ekran renklerine kazandırılmış derinlik ve çözünürlük özelliği çok güzel bir yenilik. Buna ek olarak font upgrade’ine ve animasyonlu ekran görüntülerine imkan veren “Plus!” paketi ise standart bir özellik olarak konmuş.
     <br/>
     <br/>
     Windows 97’de donanım desteğine de büyük önem veriliyor. Sistemin Digital Video Disk ve IEEE—1394 Fire—Wire özellikleri ile uyumuna büyük özen gösterilmiş. Video görüntülerini görme ve değiştirme özellikleri, USB’nin kısa zamanda video donanımlarına da geleceği düşünülerek, daha basit hale getirilmiş. Windows 97, Novell Netware 4.1 için ayrı bir araç olarak mevcut olan NDIS desteğini de kendi içinde bütünleştirmiş. Sistem dosyaları Windows 97’de Pentium işlemcinin kullanılması ile optimize edilebiliyor.
     <br/>
     <br/>
     Windows 97’deki eksiklikler
     <br/>
     <br/>
     Hangi tür MMX uzantılarının sisteme ekleneceğine henüz karar verilmemiş.         Mevcut disk kapasitesini hesaplayan programlarla uyum sağlanmadığı taktirde sistemde hesaplama problemleri ortaya çıkabilir. Microsoft bu problemin çözümünü, FAT—12 ve FAT—16 yapılarını FAT—32 yapısına çevirebilecek bir aracı Windows 97’ye eklemekte bulmuş. Ayrıca yeni yapıya DOS ve Windows 95 işletim sistemlerinden erişmenin imkansız olması Microsoft’un sistem disketinin önemini de artırıyor. Windows 97’yi çalıştırma problemi ortaya çıktığında, sürücülerdeki bilgilere sadece bu disket sayesinde erişilebiliyor.
     <br/>
     <br/>
     Öte yandan Windows NT kullanıcılarının böyle bir yapıya NTFS’den alışık olmalarına karşılık Windows 95 kullanıcılarının bu yeni yapıyı kullanmakta zorluk çekebilecekleri
