# Risale-i Nurlar CD'de

*Necati Yılmaz 1998.04.11*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   Kur'anı Kerim'in asrımızdaki en güzel tefsiri Risalei Nur eserleri sanal ortamda yerini alıyor.
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   6 bin sayfadan oluşan toplam 48 eser bütünüyle tek CD'de toplandı. Kütüphanelerinizin raflarında ciltlerce yer tutan Risalei Nur Külliyatı şimdi 1 milimetre kalınlığında, 12 santimetre çapındaki CD'yle kullanıma hazır. Hem de kitaplarda bulamayacağınız kolaylıklarla birlikte.
   <br/>
   <br/>
   'Nur 1.0' ismiyle piyasaya sunulan CD, Envar Neşriyat tarafından Redoks Bilgisayar Şirketi'ne hazırlatıldı. Titiz bir çalışmanın ürünü olan CD'de yer alan yazıların sayfa numaraları ile Envar Neşriyat tarafından en son baskısı yapılan Risalei Nur Külliyatı'nın sayfa numaraları aynı. CD, 44 sayfalık bir kitapçıkla birlikte veriliyor.
   <br/>
   <br/>
   Şimdiye kadar kitap sayfalarından okunan eserler bundan böyle bilgisayar ekranlarından ve istenilen yerlere anında ulaşıp daha birçok imkandan faydalanarak izlenebilecek.
   <br/>
   <br/>
   CD'yle gelen kolaylıklardan bazıları
   <br/>
   <br/>
    İstediğiniz kitabı seçip sayfa sayfa okuyabilirsiniz.
   <br/>
   <br/>
    İstediğiniz kitaba ve sayfasına anında gidebilirsiniz.
   <br/>
   <br/>
    İstediğiniz eserde veya kitapların bütününde kelime araması yapabilirsiniz.
   <br/>
   <br/>
    Anlamını bilmediğiniz kelimenin açıklamasını anında (kelimeyi seçip sağ tuşa basarak) öğrenebilirsiniz.
   <br/>
   <br/>
    75 bin kelime hazinesiyle Türkiye'nin en büyük OsmanlıcaTürkçe Sözlüğü'nü kullanabilirsiniz.
   <br/>
   <br/>
    200 binden fazla kelime indeksi,
   <br/>
   <br/>
    12 bin kelimelik Arapça ibareler indeksi kullanımınıza hazır.
   <br/>
   <br/>
    Eserler üzerinde her türlü araştırma ve özel aramalar yapabilirsiniz.
   <br/>
   <br/>
    İstediğiniz eserde veya bütününde seçtiğiniz bir konuyla ilgili bölümleri bir araya getirip mevzu dosyası oluşturabilirsiniz.
   <br/>
   <br/>
    Tarihçeyi Hayatı daha fazla resim ve belgelerle okuyabilirsiniz.
   <br/>
   <br/>
    Eserlerden seçilmiş güzel sözlerin resimlerle dillendirildiği 'Kainatın Dili' bölümünü müzik eşliğinde izleyebilirsiniz.
   <br/>
   <br/>
    Eserler üzerinde çalışma imkanı sağlayan yazı editörüyle çok amaçlı çalışabilirsiniz.
   <br/>
   <br/>
    Külliyat'ta geçen ayetlerin sure ve ayet numaralarını görebilirsiniz.
   <br/>
   <br/>
    Arapça kelime arayabilirsiniz.
   <br/>
   <br/>
    Kullanımı basit Arapça editörüyle kolayca çalışma imkanına sahipsiniz.
   <br/>
   <br/>
    Arapça ibareleri, Osmanlıca karakterleri de destekleyecek biçimde görebilir ve üzerlerinde çalışma yapabilirsiniz.
   <br/>
   <br/>
   Gerekli donanım
   <br/>
   <br/>
    486 veya daha üstü işlemci (Pentium tavsiye olunur), Microsoft Windows 95 işletim sistemi.
   <br/>
   <br/>
    8 MB Bellek (16 MB ve üstü tavsiye olunur)
   <br/>
   <br/>
    50 MB Harddisk'te boş alan. (Hızlı kurulum için gereklidir)
   <br/>
   <br/>
    CD ROM Sürücü (16X tavsiye olunur)
   <br/>
   <br/>
    16 Bit Ses Kartı (Seçime Bağlı)
   <br/>
   <br/>
    640x480x256 renk çözünürlük. (16 veya 24 bit renk tavsiye edilir)
   <br/>
   <br/>
    Microsoft uyumlu Mouse.
   <br/>
   <br/>
   Tavsiye olunan donanım Risalei Nur CD'sinin daha hızlı ve rahat çalışabilmesi için, standart donanıma sahip bir bilgisayarda Program'ın tüm özellikleri çalışmakta.
   <br/>
   <br/>
   İlgili adresler
   <br/>
   <br/>
   http://www.redoks.com/nur.html
   <br/>
   <br/>
   EMail: nur@redoks.com
   <br/>
   <br/>
   Envar Neşriyat ve diğer kitabevlerinden CD'yi elde edebileceğiniz gibi internet kanalıyla yukarıdaki adreslerden sipariş verilebilirsiniz.
   <br/>
   <br/>
   Her güzelin kusuru
   <br/>
   <br/>
   Büyük emeklerle hazırlanmış CD'nin, güzel eserlerin sanal dünyaya aktarılmasında önemli yer tutacağı muhakkak. Ancak hazırlayanların da geliştirmeyi düşündüğü ve görüş sorduğu bu ilk versiyon için bazı noktaları vurgulamakta fayda var:
   <br/>
   <br/>
    Kurulumundaki gereksiz bürokrasi/şifre kullanıcıyı baştan itiyor. (Herhalde ben yanlış yaptım) CD kutusunu jelatininden çıkardım, jelatini çöpe attım, CD'yi bilgisayarıma taktım. Bir sürü formalite karşıma çıktı. Neyse boşlukları dolduruyorum, şifre sordu. Şifreyi aradım, nice gayretlerden sonra çöpte buldum. Meğer şifre, çöpe attığım jelatinin üzerine yapıştırılmış etiketin üzerindeki 15 haneli sayı ve harf topluluğuymuş.
   <br/>
   <br/>
    Çalışma hızı bildiğimiz CD'lere göre yavaş.
   <br/>
   <br/>
    Risalelerde çokça yer alan eserlerin diğer bölümlerine atıflar elektronik ortamda kolayca gerçekleştirilebilir.
   <br/>
   <br/>
    Ayetlerde olduğu gibi hadislerde de kaynak gösterimi sağlanabilir.
   <br/>
   <br/>
    Ayet ve hadislerin, hatta diğer Arapça ve Farsça ifadelerin anlamlarına yer verilebilir.
