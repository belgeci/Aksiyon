# Dayak şiddeti Richter'le bile ölçülemez

*Melda Bekcan 2015.02.23*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Bazı zamanlarda…
  <br>
   Deprem haberleri geliyor kulağımıza.
   <br>
    Önce derin bir üzüntü duyuyor
    <br>
     Ardından kendimizi toplayıp
     <br>
      Kaç Richter ölçeğinde olduğunu soruyoruz.
      <br/>
      Malum!
      <br/>
      Depremin şiddeti, 7 Richter ölçeğinden fazlaysa,
      <br/>
      Tehlike arz ediyor;
      <br/>
      7 Richter’in altındaysa şayet
      <br/>
      Hafif düzeyde zararla atlatılıyor.
      <br/>
      …
      <br/>
      Hatırlar mısınız?
      <br/>
      Yıllar öncesinde, Gölcük’te deprem meydana gelmişti,
      <br/>
      7.4 Richter ölçeğindeydi,
      <br/>
      İstanbul’da bile şiddeti hissedilmiş,
      <br/>
      Apartmanlar yıkılmış,
      <br/>
      Ağaçlar devrilmişti.
      <br/>
      Can ve mal kaybıyla ocaklar yok olmuştu,
      <br/>
      İnsanlar göç etmiş,
      <br/>
      Ve kapanması çok zor olan yaralar açılmıştı.
      <br/>
      …
      <br/>
      Tıpkı deprem şiddetinin,
      <br/>
      Richter ölçeği ile belirlenmesi gibi
      <br/>
      Hayatımızdaki ölçü, önemli bir kriterdir.
      <br/>
      Kaderi bile ölçü ile tanımlar, âlimler!
      <br/>
      Kimin, daha fazla yaşayacağı,
      <br/>
      Ne kadar nefes alacağı,
      <br/>
      Kaç çocuğunun olacağı,
      <br/>
      Hangi koordinatlarda bulunacağı,
      <br/>
      Hep ölçü ile belirlenmiştir.
      <br/>
      Ve ölçü şaşmaz,
      <br/>
      Vakti zamanı geldiğinde, olan olur,
      <br/>
      Yaşanacaklar, vuku bulur.
      <br/>
      …
      <br/>
      Biliyorum tezat oluşturuyor
      <br/>
      Amma velâkin!
      <br/>
      Hayatta bazı değerler ölçülemiyor, sevgili okur.
      <br/>
      Örneğin ‘an’lar!
      <br/>
      Geçmişe dönüp ‘an’larımızı satın almaya kalksak
      <br/>
      Kıymeti tespit edilebilir mi?
      <br/>
      Nakit para versek,
      <br/>
      Satın alınabilir mi?
      <br/>
      Ya da bir yakınımızı kaybetmenin acısı,
      <br/>
      Nasıl izah edilir?
      <br/>
      Nasıl resmedilir?
      <br/>
      Ölçüsü nedir?
      <br/>
      …
      <br/>
      Şiddet de ölçülemeyen başka bir kriterdir!
      <br/>
      Erkekten şiddet gören bir kadının yaşadıkları,
      <br/>
      Yeryüzünde hiçbir ölçü birimince tarif edilemez.
      <br/>
      Öyle bir tahribat meydana gelir ki bünyede,
      <br/>
      Depremin şiddeti gibi
      <br/>
      Richter’le ölçmeye kalksan,
      <br/>
      Beceremezsin!
      <br/>
      O kadar geniş bir sahaya yayılır ki mutsuzluğu,
      <br/>
      Alan ölçüsü misali,
      <br/>
      Santimetrelere ayırsan,
      <br/>
      Güç yetiremezsin.
      <br/>
      Ya da kalbe akan gözyaşlarını,
      <br/>
      Terazide tartmaya kalksan,
      <br/>
      Çekilen çilenin karşılığını ölçmeyi
      <br/>
      İcat edemezsin.
      <br/>
      …
      <br/>
      Kelimeler de kifayetsiz kalır anlatmaya!
      <br/>
      Sanki kar yağarken,
      <br/>
      Herkes dışarıda neşeyle kartopu oynar da
      <br/>
      Şiddet gören kadın içine kapanmış,
      <br/>
      Tüm zevklerden mahrum gibidir...
      <br/>
      Sanki güneş varken gökyüzünde,
      <br/>
      Şiddet gören kadın,
      <br/>
      Güneş ışınlarından mahrum kalmış,
      <br/>
      Her daim üşümeye mahkûm gibidir.
      <br/>
      …
      <br/>
      Şiddet gören kadının gülmesi,
      <br/>
      Sadece yüzdeki mimikten ibarettir,
      <br/>
      Yürekten hissedilmez,
      <br/>
      Dayak zamanı çoktan geçse de
      <br/>
      Yaşananlar, hafızadan silinmez!
      <br/>
      Şiddet gören kadının konuşması,
      <br/>
      Sadece hecelerden ibarettir,
      <br/>
      O denli kırılmış ve incinmiştir ki
      <br/>
      Kendini toparlayıp,
      <br/>
      Hissettiklerini ifade edemez.
      <br/>
      …
      <br/>
      Gelgelelim şiddet uygulayan erkeğe!
      <br/>
      Onun zalimliğinin de ölçüsü olamaz,
      <br/>
      Kıyası yoktur,
      <br/>
      Karşılığında kıstas yapılamaz.
      <br/>
      Ve ellerimi semaya açıp dua ediyorum
      <br/>
      ‘Ya Rab, bizi, onların zulmünden koru.’
      <br/>
      Âmin.
     </br>
    </br>
   </br>
  </br>
 </p>
