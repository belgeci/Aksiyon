# Fotoğrafçı olmanın mutsuzluğunu bilir misin?

*Melda Bekcan 2015.05.26*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Düşün ki bir düğün fotoğrafçısısın…
  <br>
   Her gün bir düğüne gidiyorsun
   <br>
    Hatta öyle günler oluyor ki
    <br>
     Birden fazla düğüne katılıyorsun,
     <br>
      Onlarca gelin ve damadın fotoğrafını çekiyorsun,
      <br/>
      Her birinde farklı bir heyecan var
      <br/>
      Onların mutluluğunu kare kare resmediyorsun
      <br/>
      Tıpkı bir palyaço gibi!
      <br/>
      Sen de gülüyorsun,
      <br/>
      Onları da güldürüyorsun
      <br/>
      Ya da nasıl gülmeleri gerektiğini söylüyorsun!
      <br/>
      Sonra mesai bitiyor
      <br/>
      Ama senin işin bitmiyor
      <br/>
      Bu sefer eve gelip
      <br/>
      Bilgisayarı açıp
      <br/>
      Çekilen fotoğraflar arasından eleme yapıyorsun
      <br/>
      Böylelikle fotoğraf çekimi esnasında yaşananları,
      <br/>
      Tekrar yaşıyorsun…
      <br/>
      ‘Eee! Bunda ne var canım?’
      <br/>
      Sorusunu yöneltiyorsan sevgili okur!
      <br/>
      Cevabım hazır;
      <br/>
      Eğer evlenmek niyetindeysen
      <br/>
      Ama mevcut koşullar el vermiyorsa
      <br/>
      O zaman çektiğin görüntüler,
      <br/>
      Sana yalnızlığını hatırlatır,
      <br/>
      Ve de insanın fıtratı gereği
      <br/>
      Canı acır!
      <br/>
      …
      <br/>
      Düşün ki bir bebek fotoğrafçısısın…
      <br/>
      Her gün bir doğuma gidiyor
      <br/>
      Ve yeni doğmuş bebekle,
      <br/>
      Ona sımsıkı sarılan ailesinin fotoğrafını çekiyorsun.
      <br/>
      Hatta öyle günler oluyor ki
      <br/>
      Birden fazla organizasyona katılıyorsun
      <br/>
      Bir anne ve babanın,
      <br/>
      Yeni doğmuş bebeğe olan şefkatini
      <br/>
      Anbean görüntülüyorsun.
      <br/>
      Tıpkı bir palyaço gibi!
      <br/>
      Sen de gülüyorsun,
      <br/>
      Onları da güldürüyorsun
      <br/>
      Ya da nasıl gülmeleri gerektiğini söylüyorsun
      <br/>
      İnanılmaz bir enerji aktarılıyor objektife
      <br/>
      Ayakların yerden kesiliyor,
      <br/>
      Sanki bir ebeveyn olmuş gibi hissediyorsun…
      <br/>
      ‘Eee! Bunda ne var canım?’
      <br/>
      Sorusunu yöneltiyorsan şayet sevgili okur!
      <br/>
      Cevabım hazır;
      <br/>
      Eğer çocuk sahibi olma hayalindeysen
      <br/>
      Ama mevcut koşullar el vermiyorsa,
      <br/>
      İmkânsız gibi görünüyorsa
      <br/>
      O zaman çektiğin görüntüler,
      <br/>
      Sana çocuk özlemini hatırlatır,
      <br/>
      Ve de insanın fıtratı gereği
      <br/>
      Canı acır!
      <br/>
      …
      <br/>
      Düşün ki bir araba fotoğrafçısısın!
      <br/>
      Her gün bir çekime gidiyor
      <br/>
      Galerideki gıcır gıcır arabaları çekiyorsun
      <br/>
      Sıfır kilometre!
      <br/>
      O kadar güzel görünüyorlar ki
      <br/>
      Direksiyona oturmaya kıyamazsın!
      <br/>
      Bir arabadan diğerine geçiyorsun
      <br/>
      Arabaların özelliklerinden bahsedildiğinde
      <br/>
      Hızını alamıyorsun
      <br/>
      ‘Bastı mı gaza!’
      <br/>
      ‘Gider mi gider’
      <br/>
      Şarkısını içinden mırıldanıyorsun.
      <br/>
      O denli yakınsın ki arabalara!
      <br/>
      Sanki anahtarı alıp yola çıkacaksın
      <br/>
      Zaten ehliyetin de var
      <br/>
      Artık rüyalarının gerçekleşme zamanı!
      <br/>
      Tam o sırada fiyatı geliyor aklına!
      <br/>
      Galeri sahibine çıtlatıyorsun,
      <br/>
      Aldığın cevap karşısında
      <br/>
      Dizlerinin bağı çözülüyor,
      <br/>
      Fotoğraf makinesi elinden düşüyor,
      <br/>
      Kendini yola veriyorsun.
      <br/>
      Yürümekten bitap düştüğünde
      <br/>
      İnadından vazgeçip
      <br/>
      Toplu taşıta biniyorsun
      <br/>
      Fotoğraf makinesini koluna takıp
      <br/>
      Sımsıkı sarılıp
      <br/>
      İçine kaydettiğin son model arabaları düşünüp
      <br/>
      Kendini onların içinde hayal ediyorsun.
      <br/>
      Aniden frene basıyor şoför,
      <br/>
      Gerisini artık sen düşün!
     </br>
    </br>
   </br>
  </br>
 </p>
