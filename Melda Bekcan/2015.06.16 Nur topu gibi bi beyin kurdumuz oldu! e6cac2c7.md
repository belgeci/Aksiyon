# Nur topu gibi bi beyin kurdumuz oldu!

*Melda Bekcan 2015.06.16*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Bir seçim sürecini daha geride bıraktık ya!
  <br>
   Oh be! Şükürler olsun.
   <br>
    Neydi o tantana
    <br>
     Yani inanılır gibi değil…
     <br>
      Caddenin bir yanında parti sloganları atılıyor,
      <br/>
      Bayraklar sallanıyor,
      <br/>
      Müziğin sesi sonuna kadar açılmış,
      <br/>
      Bangır bangır çalıyor.
      <br/>
      Haa! Durum sadece bundan ibaret olsa
      <br/>
      Ona da eyvallah!
      <br/>
      Ama caddenin tam karşısında
      <br/>
      Sanki ayna yansıması gibi
      <br/>
      Başka bir parti,
      <br/>
      Aynı performansı sergiliyor.
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  …
 </p>
 <p>
  Bu durumda ne oluyor?
  <br/>
  Olan sana, bana, yani halka oluyor.
  <br/>
  Yapılanlar o kadar komik ki!
  <br/>
  Sanki ben tercihimi yaparken…
  <br/>
  Sokağın iki tarafındaki elektrik direklerine bağlanmış olan
  <br/>
  İpten sarkan afişlere bakıp
  <br/>
  Hangi adayın daha iyi göründüğünü tartıp
  <br/>
  Ona göre mi karar vereceğim
  <br/>
  Öyle mi?
  <br/>
  Resmen şaka gibi!
  <br/>
  Ya da sesi sonuna kadar açılmış müzik setindeki
  <br/>
  Şarkının kıvraklığına,
  <br/>
  Şarkı sözlerinin anlamına mı bakacağım?
  <br/>
  Geleceğimi belirleyen kararı
  <br/>
  Böyle mi oylayacağım?
 </p>
 <p>
  …
 </p>
 <p>
  Ve nihayet seçim süreci bitti,
  <br/>
  Propaganda makinelerinin fişi çekildi,
  <br/>
  Şarkıların türkülerin sesi soluğu kesildi
  <br/>
  Derken!!!
  <br/>
  Şimdi daha büyük bir sorun çıktı karşımıza.
  <br/>
  Meğer yaşanan onca debdebe,
  <br/>
  Yapılan onca şov,
  <br/>
  Kaldırılan toz,
  <br/>
  Atılan çamur
  <br/>
  Ve tüm bunların neticesinde ortaya çıkan manzara
  <br/>
  Fırtına öncesi sessizlikmiş.
  <br/>
  Rüyamda dahi görsem inanmazdım,
  <br/>
  Bir kâhin kulağıma fısıldasa, aldırmazdım.
  <br/>
  Bizzat başıma gelince kabullenmek mecburiyetinde kaldım
  <br/>
  Birileri bana çimdik atsın diye bekliyorum
  <br/>
  Meğer bu gelinen sonuç hayal değil
  <br/>
  Gerçekmiş!
 </p>
 <p>
  …
 </p>
 <p>
  Müjdeler olsun!
  <br/>
  Artık nur topu gibi bi beyin kurdumuz oldu.
  <br/>
  Şimdi ne yapacağız diye kara kara düşünüyoruz.
  <br/>
  Sahi! Bundan sonra ne olacak?
  <br/>
  Devlet büyüklerimiz nasıl bir karara varacak?
  <br/>
  Koalisyon olacak mı?
  <br/>
  Olmayacak mı?
  <br/>
  Yoksa kapımız ‘tık tık’ çalıp
  <br/>
  ‘Kim o?’ diye sorduğumuzda
  <br/>
  Kapının ardından gelen ses
  <br/>
  Benim adım ‘erken seçim’ mi olacak!
  <br/>
  Ve bu durumda biz ne yapacağız?
 </p>
 <p>
  …
 </p>
 <p>
  Müjdeler olsun!
  <br/>
  Artık nur topu gibi bi beyin kurdumuz oldu
  <br/>
  Şimdi kara kara düşünüyoruz.
  <br/>
  Dolar yükseldi,
  <br/>
  Borsa magma seviyesine kadar indi,
  <br/>
  Yaptığımız programlar altüst oldu,
  <br/>
  Yarın bize ne getirecek?
  <br/>
  Bilmiyoruz,
  <br/>
  Bekliyoruz,
  <br/>
  Haberleri merakla seyrediyor,
  <br/>
  İşin nereye varacağını sorguluyoruz.
 </p>
 <p>
  …
 </p>
 <p>
  Müjdeler olsun!
  <br/>
  Artık nur topu gibi bi beyin kurdumuz oldu.
  <br/>
  Sorular, beynimizi kemiriyor,
  <br/>
  Kurtların sayısı artıyor,
  <br/>
  Birilerinin bu gidişata ‘dur’ demesi gerekiyor.
  <br/>
  Geceleri başımı yastığa koyduğumda
  <br/>
  Bir sonraki güne dair kaygılanmak istemiyorum,
  <br/>
  Kurtlar bedenimi terk etsin.
  <br/>
  Geleceğimiz nasıl şekillenecek?
  <br/>
  Ben bileyim,
  <br/>
  Sen bilesin,
  <br/>
  Herkes bilsin.
 </p>
</div>
