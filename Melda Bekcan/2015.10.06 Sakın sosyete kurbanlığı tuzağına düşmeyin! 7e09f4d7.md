# Sakın sosyete kurbanlığı tuzağına düşmeyin!

*Melda Bekcan 2015.10.06*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Herkesin bir anısı vardır,
  <br/>
  Kurbana dair!
  <br/>
  Kurban Bayramı denince,
  <br/>
  Farklı görüntüler belirir,
  <br/>
  Zihinlerde…
  <br/>
  Kimisinin hatırına, haberlere yansıyan
  <br/>
  Kaçan kurban görüntüsü gelir,
  <br/>
  Kimisi kavurma yemeği hatırlar,
  <br/>
  Ağzı lezzetleniverir.
  <br/>
  Kimisi yapılan bayram temizliğini anımsar,
  <br/>
  Burnuna arap sabunu kokusu gelir,
  <br/>
  Kimisi taa çocukluğuna gider
  <br/>
  Kurban kesilirken akıttığı gözyaşları
  <br/>
  Gözbebeğinde belirir,
  <br/>
  Kiminin aklına gıcır gıcır alınan
  <br/>
  Bayramlık ayakkabıları gelir,
  <br/>
  Kimine göreyse Kurban Bayramı,
  <br/>
  Muhtaç olanlara yardım ulaştırma vesilesidir.
  <br/>
  …
 </p>
 <p>
  Sahi!
  <br/>
  Kurban Bayramı ne demektir, size göre?
  <br/>
  a) Kurban satışındaki pazarlık mı?
  <br/>
  b) Kurban kesenin helallik istemesi mi?
  <br/>
  c) Kanlı kurban kesimi mi?
  <br/>
  d) Eve gelen etlerin parçalanması mı?
  <br/>
  e) Telaş içinde yapılan kavurma
  <br/>
  f) Ev halkının tencere başına toplanması
  <br/>
  g) Ve içine bandırılan ekmek mi?
  <br/>
  h) Etlerin, verilecek kişilere dağıtılması mı?
  <br/>
  ı) Dolabın etle dolması mı?
  <br/>
  i) Etrafın kirlenmesi
  <br/>
  j) Evde temizlik yapılması mı?
  <br/>
  k) Eve sinen et kokusu mu?
  <br/>
  l) Fakir oldukları bilinen insanların,
  <br/>
  Kapılarına kadar gidip
  <br/>
  Ellerinizle et ulaştırmak mı?
  <br/>
  m) Hepsi mi?
  <br/>
  …
 </p>
 <p>
  Bu sorunun cevabı,
  <br/>
  Hangi şıktır zihninizde?
  <br/>
  a mı, b mi?
  <br/>
  Sizi bilmiyorum ama
  <br/>
  Benim cevabım,
  <br/>
  Tüm bu şıklardan, daha da fazlası…
  <br/>
  Bilirim!
  <br/>
  Kurban Bayramı herkes için özeldir elbet
  <br/>
  İbadet boyutuyla halistir,
  <br/>
  Muhtaç olanların yüzünü güldürür,
  <br/>
  Ama bizim ailede, dededen gelen bir ananedir,
  <br/>
  Rahmetli dedem ve rahmetli babam
  <br/>
  Hayatta olduklarında et ticareti ile uğraştıkları için
  <br/>
  Daha kapsamlı yaşarız, bu bayramı
  <br/>
  Kılcallara kadar ineriz…
  <br/>
  Böyle bir ailede yetişmenin şansını ise
  <br/>
  Yıllar sonrasında fark ettim.
  <br/>
  Bir kere kurbanın caiz olan her organını yerim!
  <br/>
  Beyin, dil, böbrek hiç fark etmez.
  <br/>
  Ayrıca bir bayan olarak asla çekinmem,
  <br/>
  Kurban kesiminin her safhasını seyredebilirim.
  <br/>
  Ölüm meselesini,
  <br/>
  Kan tutma sakıncasını,
  <br/>
  Soğukkanlı olmayı,
  <br/>
  Her şeyden de önemlisi
  <br/>
  Teslimiyeti,
  <br/>
  Allah rızası için kurban kesmeyi,
  <br/>
  Ben bu bayramda öğrendim.
  <br/>
  …
 </p>
 <p>
  Bazı hadiseler vardır ki
  <br/>
  Sadece okunur!
  <br/>
  Bazı hadiseler vardır ki
  <br/>
  Sadece seyredilir!
  <br/>
  Bazı hadiseler vardır ki
  <br/>
  Sadece dinlenir!
  <br/>
  Bazı hadiseler vardır ki
  <br/>
  Beş duyuyla algılanmaz
  <br/>
  Sadece hissedilir!
  <br/>
  Bazı hadiseler vardır ki
  <br/>
  Meseleye müdahil olmak,
  <br/>
  Zahmete girmek
  <br/>
  Bizzat icraat gerektirir.
  <br/>
  İşte kurban ibadeti de
  <br/>
  Bu kapsamda yapılmasını icap ettirir.
  <br/>
  …
 </p>
 <p>
  Şimdi gelelim attığım başlığa!
  <br/>
  Neden mi sosyete kurbanlığından bahsettim?
  <br/>
  Kurban ibadetini yalnızca hesaba para yatırmaktan,
  <br/>
  Fakir olan insanlara et ulaştırmaktan,
  <br/>
  İbaret görmek,
  <br/>
  Sosyete usulü,
  <br/>
  Elini kana bulaştırmadan
  <br/>
  O zahmete girmeden
  <br/>
  Bedeli ödeyip gerçekleştirmek,
  <br/>
  Yeterli değil!
  <br/>
  1)Kurban pazarlığı yapmak,
  <br/>
  Sünnettir.
  <br/>
  2)Kurban helalliği almak,
  <br/>
  Fıkıh ilmidir.
  <br/>
  3)Kurban kesimine şahit olmak,
  <br/>
  Teslimiyettir.
  <br/>
  4)Bayram sabahında kurban eti yemek
  <br/>
  Sünnettir.
  <br/>
  Ve yeni neslin
  <br/>
  Bu ibadetlerin canlı olarak yaşanmasından
  <br/>
  Mahrum edilmesi onlara zarar verebilir,
  <br/>
  Elbette!
  <br/>
  Bunlar benim âcizane düşüncelerim.
 </p>
