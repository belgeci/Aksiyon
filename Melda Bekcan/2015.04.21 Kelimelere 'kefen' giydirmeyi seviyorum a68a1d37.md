# Kelimelere 'kefen' giydirmeyi seviyorum

*Melda Bekcan 2015.04.21*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Kelimelere ‘kefen’ giydirmeyi seviyorum
  <br>
   Ama!
   <br>
    Her zaman değil elbette,
    <br>
     Nadiren karşılaştığım bir durum bu...
     <br>
      Karşımdaki insanla iletişim kuramadığımda,
      <br/>
      Artık söylenecek bir şey kalmadığında,
      <br/>
      Önce kabuğuma çekilip
      <br/>
      Uzun bir sessizliğe bürünüyorum.
      <br/>
      Bu esnada yavaşça kefene sarılıyor sözlerim,
      <br/>
      O insana dair.
      <br/>
      Ve kelimeler, bir kez ağzımdan çıktı mı
      <br/>
      Muhatabına ulaştı mı
      <br/>
      Yaşananları kalbimin derinliklerine gömüyorum
      <br/>
      Ruhuna ‘El Fatiha!’
      <br/>
      Başına taş dahi koymuyorum.
      <br/>
      …
      <br/>
      Kabul! Belki size garip gelebilir.
      <br/>
      Ama!
      <br/>
      Kelimelere gelinlik giydirmeyi de seviyorum...
      <br/>
      Baştan aşağıya bembeyaz tüllere bürüyüp
      <br/>
      Üstünü beyaz duvakla örtüp
      <br/>
      Tertemiz bir niyetle,
      <br/>
      Saf duygularla cümleler kurup
      <br/>
      Karşımdaki kişiye ilettiğimde!
      <br/>
      Üstüne üstlük aynı şekilde karşılık verilirse,
      <br/>
      İşte o zaman değmeyin keyfime,
      <br/>
      Dünyalar benimmiş gibi olur.
      <br/>
      ...
      <br/>
      Kelimelerden mermi yapmayı seviyorum
      <br/>
      Bu da en son tercih ettiğim bir durum…
      <br/>
      Ne zaman ki tasallut söz konusu olursa,
      <br/>
      Sabretmeme rağmen dozaj artarsa,
      <br/>
      Kendimi müdafaa etmezsem
      <br/>
      Yine de taarruz devam ederse,
      <br/>
      O zaman nokta atışı yapmak gerekir!
      <br/>
      Önce hedefe kilitlenilir.
      <br/>
      Kelimeler yan yana dizilir
      <br/>
      Mermi hâline getirilir,
      <br/>
      Çok dikkatli hareket etmeli!
      <br/>
      Zira tahrip gücü yüksek bir cümle,
      <br/>
      Ortalığın altını üstüne getirir.
      <br/>
      …
      <br/>
      Kelimelere kilo aldırmayı seviyorum.
      <br/>
      Ne kadar kilo
      <br/>
      O kadar ağırlık demektir!
      <br/>
      Hele bir ağızdan çıkmayagörsün!
      <br/>
      Sözü duyanın üstüne bir oturdu mu
      <br/>
      Bir daha kalkmaz!
      <br/>
      Olduğu yerde çakılıverir,
      <br/>
      Bir yere kımıldayamaz.
      <br/>
      Yani bir kelime,
      <br/>
      Gülleden bile daha tesirli olabilir.
      <br/>
      …
      <br/>
      Bazen kelimelere diyet yaptırmak gerekir!
      <br/>
      Hayatı her zaman ciddiye almak doğru değil.
      <br/>
      Dünyada gezmeye, eğlenmeye de yer var.
      <br/>
      İşte diyete girmiş kelimeler,
      <br/>
      Tam da böyle zamanlarda kıymetlidir.
      <br/>
      Hem fazla yer kaplamaz
      <br/>
      Hem de taşıması kolaydır.
      <br/>
      Dünyanın öbür ucuna gitsen
      <br/>
      Hiç zahmet vermeden
      <br/>
      Yanı başında gelir.
      <br/>
      …
      <br/>
      Kelimelere çengel takmayı seviyorum
      <br/>
      Bir kere zihne takılmayagörsün!
      <br/>
      Gece gündüz peşini bırakmaz
      <br/>
      Tıpkı kuyruk gibi arkandan gelir,
      <br/>
      Çekip çıkarmaya güç getiremezsin...
      <br/>
      Hep bir kan sızıntısı olur çevresinde,
      <br/>
      Yara kapanmaz,
      <br/>
      Daha da derinleşir.
      <br/>
      Kancalı kelimelerin en faziletlisi,
      <br/>
      Kulağa küpe gibi takılan,
      <br/>
      İnsana tecrübe verendir.
      <br/>
      …
      <br/>
      Peki, tüm bunları neden mi yapıyorum?
      <br/>
      Offf! İnsanlarla geçinmek çok zor,
      <br/>
      Ne yapıp etsem de olmuyor.
      <br/>
      Artık elden bir şey gelmediğinde,
      <br/>
      Çaresiz kaldığımda,
      <br/>
      Kelimeler yetişiyor feryadıma!
      <br/>
      Ağızdan çıkan
      <br/>
      Ya da bir yerlere not alınan kelimeleri
      <br/>
      Sakın küçümsemeyin,
      <br/>
      Bazen bir padişah fermanından bile kuvvetli olabilir,
      <br/>
      Hükümler katidir,
      <br/>
      Kelimeler, istikametimizi belirler,
      <br/>
      Kader çizgimize yön verir.
     </br>
    </br>
   </br>
  </br>
 </p>
