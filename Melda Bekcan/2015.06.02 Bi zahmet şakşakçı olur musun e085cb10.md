# Bi zahmet şakşakçı olur musun?

*Melda Bekcan 2015.06.02*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Son günlerde yaşadığım en büyük sıkıntı
  <br>
   İnsanların bana yönelttiği bu soru;
   <br>
    Bi zahmet şakşakçı olur musun?
    <br>
     Haa!
     <br>
      Zannetmeyin ki aleni karşıma çıkıp
      <br/>
      Kelimelerle sual yöneltiyorlar.
      <br/>
      Asla!
      <br/>
      Hatta mümkün olduğunca gizleniyor, bu duygu.
      <br/>
      Ama!
      <br/>
      Eninde sonunda benden yapmam beklenen…
      <br/>
      Ayağa kalkmam
      <br/>
      Islık falan çalmam
      <br/>
      Ve mümkün olabilecek en yüksek sesle
      <br/>
      Alkış yapmam!
      <br/>
      …
      <br/>
      Zira!
      <br/>
      İnsanlar, kendi zihinlerinde, insan olmayı aşmışlar
      <br/>
      Kendilerini her şey zannediyorlar!
      <br/>
      Yani hem senarist
      <br/>
      Hem yönetmen
      <br/>
      Hem yapımcı
      <br/>
      Hem oyuncu
      <br/>
      Hem kameraman
      <br/>
      Hem ışık şefi
      <br/>
      Hem sanat yönetmeni
      <br/>
      Kısa ve öz olarak
      <br/>
      Her şey!
      <br/>
      …
      <br/>
      Üstelik!
      <br/>
      Ne sinemadan bahsediyorum
      <br/>
      Ne televizyondan
      <br/>
      Ne de tiyatrodan!
      <br/>
      Benim tüm rahatsızlığım,
      <br/>
      Gerçek hayata dair!
      <br/>
      İnsanlardaki, hastalık boyutundaki ‘sahip olma’ duygusu.
      <br/>
      İşte tüm mağduriyetim bu.
      <br/>
      …
      <br/>
      Bazı insanlar var ki…
      <br/>
      Oturdukları yerden senaryo yazıyor,
      <br/>
      Yazdıkları senaryodaki tek başrolü kendine veriyor
      <br/>
      Sonra etrafından figürasyon seçip
      <br/>
      En uygun zamanı
      <br/>
      Ve mekânı seçip
      <br/>
      ‘Bir, iki, üç’ diye start verip
      <br/>
      Başlıyorlar kendi filmlerini çekme aşamasına!
      <br/>
      …
      <br/>
      Yalnız!
      <br/>
      Burada en önemli unsur
      <br/>
      Ortaya çıkan performansı beğenecek,
      <br/>
      Duygularını açıkça ifade edip
      <br/>
      Herkesin içinde onu övecek kişi!
      <br/>
      Zaten!
      <br/>
      Başka bir seçenek yok ki, azizim.
      <br/>
      Yapılan iş ya övülecek
      <br/>
      Ya da övülecek.
      <br/>
      …
      <br/>
      Tam da bu noktada
      <br/>
      Yazının başlığında ifade ettiğim
      <br/>
      Durum çıkıyor ortaya,
      <br/>
      Çaktırmadan sual iletiliyor;
      <br/>
      Bi zahmet şakşakçı olur musun?
      <br/>
      Önce telefonla arz ediliyor, malum istek.
      <br/>
      Kibarca geri çeviriyorum,
      <br/>
      Yok, anlamıyor,
      <br/>
      Üstelik!
      <br/>
      Telefon trafiği devam ediyor...
      <br/>
      Ardından açıkça reddediyorum
      <br/>
      Yine kabul görmüyor.
      <br/>
      …
      <br/>
      Bakıyorum ki!
      <br/>
      Ne yapsam da karşımdaki vazgeçmeyecek,
      <br/>
      Yeryüzündeki tüm mazeretleri sıralasam da
      <br/>
      Asla pes etmeyecek!
      <br/>
      Yenilgiyi kabullenip
      <br/>
      Bayrağı yarıya indirip
      <br/>
      İçime derin bir nefes çekip
      <br/>
      Başlıyorum vazifemi yerine getirmek üzere
      <br/>
      Şak şak yapmaya!
      <br/>
      …
      <br/>
      Hadi!
      <br/>
      Bir kere taviz veriyorum.
      <br/>
      Belli bir süre sonra tekrar talep geliyor.
      <br/>
      İkinci kez tüm enerjimi toplayıp
      <br/>
      Yine şak şak yapmaya başlıyorum.
      <br/>
      Ama!
      <br/>
      İsteklerin ardı arkası kesilmiyor.
      <br/>
      Ve bu aşamanın ardından…
      <br/>
      Sinir erozyonu devreye giriyor.
      <br/>
      Aman kimse kırılmasın gücenmesin kaygısıyla
      <br/>
      Kendi kendimi kemirmeye başlıyorum.
      <br/>
      İşin kötü yanı da…
      <br/>
      Ben bu atakları hep yaşıyorum.
     </br>
    </br>
   </br>
  </br>
 </p>
