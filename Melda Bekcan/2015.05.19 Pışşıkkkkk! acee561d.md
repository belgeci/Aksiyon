# Pışşıkkkkk!

*Melda Bekcan 2015.05.19*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Takvimlerden koparılan her sahife,
  <br>
   Doğum günü pastasına eklenen her yeni mum,
   <br>
    Sahip olunan diplomalar,
    <br>
     Binbir zorlukla ulaşılan mevki makam,
     <br>
      Bizlere neler getirdi?
      <br/>
      Ve bizden neler alıp götürdü?
      <br/>
      İşte bunları enine boyuna tartışıp
      <br/>
      İrdelememiz gerekiyor.
      <br/>
      …
      <br/>
      Bazen pişman oluyorum büyüdüğüme!
      <br/>
      Bir çocuk gibi davranıp
      <br/>
      Bir çocuk gibi duygularımı ifade etmek istiyorum.
      <br/>
      Örneğin…
      <br/>
      Beni aldatmak isteyen insanlara,
      <br/>
      Geçmişte canımı yaktığı yetmiyormuş gibi
      <br/>
      Yeniden yanıma yaklaşanlara,
      <br/>
      Karşıma dikilip laf salatası yapanlara karşı,
      <br/>
      Çocuksu bir dürtü geliyor içimden,
      <br/>
      Kendimi zor tutuyorum,
      <br/>
      Parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak
      <br/>
      Ve haykırmak istiyorum;
      <br/>
      Bir daha seninle oynamayacağım!
      <br/>
      ...
      <br/>
      Piyasada öyle bir kategori var ki
      <br/>
      Kendini uyanık sanıyor,
      <br/>
      Onlarla en çok alışveriş esnasında karşılaşıyorum.
      <br/>
      Yahu, gözünün içine baka baka yalan söylüyor.
      <br/>
      Yok efendim!
      <br/>
      Sattıkları organik gıda imiş
      <br/>
      Tamamen doğal malzemelerden üretilmiş!
      <br/>
      Üstüne konulan zam,
      <br/>
      Bu ürünün hakkı imiş!
      <br/>
      Çocuksu bir dürtü geliyor içimden,
      <br/>
      Kendimi zor tutuyorum,
      <br/>
      Parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak
      <br/>
      Ve haykırmak istiyorum;
      <br/>
      Yalan söylüyorsun!
      <br/>
      …
      <br/>
      Bir de fitneciler var etrafta
      <br/>
      İşleri güçleri ortalığı karıştırmak olan.
      <br/>
      Ne yaparsan yap!
      <br/>
      Onları asla mutlu edemezsin
      <br/>
      Mutlak suretle bir yerlerden kusur bulur,
      <br/>
      Şayet bulamazsa da uydurur,
      <br/>
      Ortalığı birbirine katar,
      <br/>
      Makineli tüfek gibi saymaya başlar…
      <br/>
      İçinde bulunduğum ortamda,
      <br/>
      Onlardan biri dedikodu yapma girişiminde bulunsa,
      <br/>
      Çocuksu bir dürtü geliyor içimden
      <br/>
      Kendimi zor tutuyorum
      <br/>
      Parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak
      <br/>
      Ve haykırmak istiyorum;
      <br/>
      Seni sevmiyorum.
      <br/>
      …
      <br/>
      Haa! Bir de yüklü insanlar vardır!
      <br/>
      Bunlar kenarda durup etrafı seyreder
      <br/>
      Güzelliklere dair yaşananları çekemez,
      <br/>
      İçine atar,
      <br/>
      Hiç ses çıkarmadan zihnine eleştiriler yükler
      <br/>
      Çok da iyi sabreder.
      <br/>
      Zira ne zaman ayağın tökezlese,
      <br/>
      Ya da başına bir bela musibet gelse
      <br/>
      Ağırlığından kurtulmaya niyetlenir,
      <br/>
      Yükünü boşaltmak için gelir…
      <br/>
      İşte böyle zamanlarda,
      <br/>
      Çocuksu bir dürtü geliyor içimden
      <br/>
      Kendimi zor tutuyorum,
      <br/>
      Parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak istiyorum.
      <br/>
      Ve haykırmak istiyorum;
      <br/>
      Benden uzak dur.
      <br/>
      …
      <br/>
      En önemli kategori ise omuz gözetleyendir!
      <br/>
      Bunların işi gücü yoktur,
      <br/>
      Başkalarından nemalanmak, hayat düsturudur.
      <br/>
      Kendileri çalışmayı sevmez,
      <br/>
      Emek sarf etmez,
      <br/>
      Parmağını bile kaldırmak istemez…
      <br/>
      Üstüne üstlük gözü hep yükseklerdedir,
      <br/>
      Başkalarının omzuna çıkmayı hedeflenir,
      <br/>
      Taktikleri de bellidir!
      <br/>
      Önce dost olarak göstermeye çalışır kendini,
      <br/>
      Ayrıca gösteriş yapmaya meyillidir…
      <br/>
      İşte böyle birisiyle karşılaştığımda,
      <br/>
      Çocuksu bir dürtü geliyor içimden
      <br/>
      Kendimi zor tutuyorum,
      <br/>
      Parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak
      <br/>
      Ve haykırmak istiyorum;
      <br/>
      Senin gerçek yüzünü biliyorum.
      <br/>
      …
      <br/>
      Gelin görün ki!
      <br/>
      Yaş ilerliyor,
      <br/>
      Boğaza atılan düğümlerin sayısı çoğalıyor,
      <br/>
      Yüze takılan maskelerin sayısı
      <br/>
      Ve çeşitliliği artıyor,
      <br/>
      ‘El âlem ne der?’ kaygısı güdülüyor.
      <br/>
      Hâl böyleyken!
      <br/>
      Çocuksu dürtüler gelse de içimden,
      <br/>
      Ve parmağımı gözüme iliştirip
      <br/>
      ‘Pışşıkkkkk!’ yapmak istesem de
      <br/>
      Kendimi cidden tutuyorum,
      <br/>
      Kendimi ‘cidden’ zor tutuyorum.
     </br>
    </br>
   </br>
  </br>
 </p>
