# Aramak ve giz perdesini aralamak

*Melda Bekcan 2015.03.02*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  İş yerinde bilgisayarı açıyorum
  <br>
   Bir konu araştırmak için
   <br>
    Hayda!
    <br>
     Karşıma derhal ‘ara’ emri çıkıveriyor…
     <br>
      Eve geldiğimde bir yemek yapmak istiyorum
      <br/>
      Bu sefer de yemek tariflerini ‘ara’maya başlıyorum!
      <br/>
      Bir hastalık nedeniyle doktora gideceğim
      <br/>
      Yine iyi doktor ‘ara’ma peşindeyim!
      <br/>
      Arabaya biniyorum, istikametime ulaşmak için
      <br/>
      Ne yapıyorum dersiniz?
      <br/>
      Radyo frekanslarını ‘ara’mak görevim.
      <br/>
      …
      <br/>
      ‘Ara’yan Mevla’sını da bulur;
      <br/>
      Belasını da!’ demiş atalarımız.
      <br/>
      Demek ki arayan kişi,
      <br/>
      Eninde sonunda amacına ulaşır.
      <br/>
      Önemli olan unsur;
      <br/>
      Arama niyetidir,
      <br/>
      Yola çıkış enerjisidir!
      <br/>
      Arama eylemi, kişiden kişiye değişir.
      <br/>
      Kimisi inşa etmek için arar,
      <br/>
      Kimisi de yıkım için.
      <br/>
      Ameller, niyetlere göre belirlenir.
      <br/>
      …
      <br/>
      Aramaktan aramaya da fark vardır elbet;
      <br/>
      Kişinin, sahip olduğunu kaybetmesi,
      <br/>
      Çekilen onca hasretin ardından
      <br/>
      Kaybedileni ‘ara’ması,
      <br/>
      Tekrar kavuşacağı anı hayal etmesi, başkadır!
      <br/>
      Hiç sahip olmadığını kurgulaması,
      <br/>
      Hiç bilmediğini bilmeye çalışması
      <br/>
      Onu hayal etmesi,
      <br/>
      Ve hayallerinin peşinden gitmesi başka.
      <br/>
      Gün gelir!
      <br/>
      Yıllardır aramakta olduğun tam karşında dikilir,
      <br/>
      Ama farkında bile değilsindir.
      <br/>
      …
      <br/>
      ‘Ara’ma eyleminin yönünü bilemezsin;
      <br/>
      Dış dünyada ‘ara’ma olabilir,
      <br/>
      Ya da içine doğru yönelebilirsin.
      <br/>
      Belki bir ömür sürebilir
      <br/>
      Belki kısa sürede muradına erersin.
      <br/>
      ‘Ara’mayı belli bir formata sokamazsın!
      <br/>
      Bazen aradıkça kafan karışır,
      <br/>
      Arapsaçına döner düşüncelerin.
      <br/>
      İpin bir ucundan tutarsın,
      <br/>
      Takip edersin,
      <br/>
      Ama sonunu bir türlü getiremezsin.
      <br/>
      …
      <br/>
      Bazen de aradıkça huzura erersin,
      <br/>
      Ararken kendinden uzaklaşırsın,
      <br/>
      Sıkıntılarını çıkarmaya başladıkça hatırından,
      <br/>
      Ötekilerin derdi, sana ait olur.
      <br/>
      ‘Ara’dığının aslında başkasında olduğunu anlarsın!
      <br/>
      ‘Hayır yapmak’ amacıyla arayan vardır,
      <br/>
      Derdi yardımdır.
      <br/>
      Üşenmez,
      <br/>
      Faydalı şeylerin peşinden koşturur
      <br/>
      Ve uğraştıkça kapılar ardına kadar açılır!
      <br/>
      Kimileri de şer peşindedir,
      <br/>
      Niyeti kötülüktür,
      <br/>
      İşgüzarlık, iliklerine değin işlemiştir.
      <br/>
      …
      <br/>
      Küçükken çok yaramaz değildim
      <br/>
      Öyle pencere camı kırmazdım,
      <br/>
      Sakin sakin oturur
      <br/>
      Ama merak dolu sorular sorardım.
      <br/>
      Yaşım ilerledikçe
      <br/>
      ‘Ne aradığımı’ aramaya başladım!
      <br/>
      Gönlümde taşımak istediğim duygunun peşinde,
      <br/>
      Samimi,
      <br/>
      Sahici,
      <br/>
      Sıcak hissiyatların izindeyim şimdilerde.
      <br/>
      …
      <br/>
      Arıyorum!
      <br/>
      Ve aradıkça aralıyorum, hayatın giz perdesini.
      <br/>
      Çocukluğumdan beri yılmadım,
      <br/>
      Problemlerle boğuşmaktan bıkmadım,
      <br/>
      Hep bir sebep sonuç ilişkisi kurma hedefinde,
      <br/>
      Anahtar kilit ilişkisinde!
      <br/>
      Asla pes etmemeli,
      <br/>
      Her ‘an’ı değerlendirmeli,
      <br/>
      Zaten bu dünyada en çok korktuğum kişi,
      <br/>
      Hiç ‘ara’mamış,
      <br/>
      Hiç tasalanmamış,
      <br/>
      Ne istediğini bilmeyendir.
     </br>
    </br>
   </br>
  </br>
 </p>
