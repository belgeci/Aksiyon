# Birbirimizi 'deli' gibi sevmeliyiz

*Melda Bekcan 2015.03.17*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Akılla kafayı bozmuş,
  <br>
   Her daim mantık peşinde,
   <br>
    Hesap kitap derdindekilerle,
    <br>
     Hiç işim olmaz benim
     <br>
      Hatta itiraf etmeliyim ki
      <br/>
      Onlardan uzak durur,
      <br/>
      Arama mesafe koyarım, Sevgili Aksiyon okuru.
      <br/>
      Çünkü benim indimde, akıllılık değil,
      <br/>
      Delilik daha makbuldür.
      <br/>
      Kabul! Çoğu insan akıllı olabilir.
      <br/>
      Gelgelelim!
      <br/>
      Deliliğin dozunu ayarlamak daha zordur.
      <br/>
      …
      <br/>
      Daha önce defalarca bu konuya değindim,
      <br/>
      Şimdi sırası gelmişken tekrar altını çizeyim,
      <br/>
      Dozunda deli olmak faydalıdır,
      <br/>
      Hiç kimseye zararı yoktur.
      <br/>
      Emin olun ki!
      <br/>
      Bu durum toplumun refah düzeyini bile artırır.
      <br/>
      Ne demiş ünlü kimyacı Paracelsus;
      <br/>
      Aslında her madde zehirdir,
      <br/>
      Zehir ile ilacı birbirinden ayıran
      <br/>
      Dozudur!
      <br/>
      …
      <br/>
      Varsın bir miktar deli olayım!
      <br/>
      Aksi takdirde sabahları nasıl kalkacağım?
      <br/>
      Kendimde o gücü nereden bulacağım?
      <br/>
      Evde keyif yapmak varken…
      <br/>
      Ben gidip işte mesai yapmayı seçiyorum!
      <br/>
      Evde kahvaltı sofrasındaki lezzetleri terk edip…
      <br/>
      İş yerinde ayaküstü bir şeyler atıştırıyorum!
      <br/>
      Gün boyunca heva ve heveslerime uymak varken…
      <br/>
      Akşama kadar durmadan çalışıyorum!
      <br/>
      …
      <br/>
      Şu üç günlük dünyada!
      <br/>
      Ne kadar nefes alacağımı bile bilmeden
      <br/>
      Yarına çıkacak mıyım,
      <br/>
      Yoksa çıkmayacak mıyım
      <br/>
      Kestiremeden!
      <br/>
      Bunları yapmak, akıl kârı mı?
      <br/>
      Bunca emeğin, koşturmanın devam etmesi,
      <br/>
      Mücadele edebilmem,
      <br/>
      Kendimi motive edebilmem için
      <br/>
      Birazcık deli olmam lazım.
      <br/>
      …
      <br/>
      ‘Sahabelerin yaşadıkları dönemi görseydiniz,
      <br/>
      Onları, deli zannederdiniz’ demiş, Efendimiz.
      <br/>
      Demek ki!
      <br/>
      Günümüz şartlarında, fazla akıllı olmakla
      <br/>
      Pek doğru bir yolda değiliz.
      <br/>
      Sadece maddi değil
      <br/>
      Manevi değerler için de
      <br/>
      Aklı bir kenara alıp
      <br/>
      Belli saatlerde onu uykuya yatırıp
      <br/>
      Üstüne de kalın bir yorgan çekip
      <br/>
      Deliliğe dört elle sarılıp
      <br/>
      Birlikte vakit geçirmeliyiz.
      <br/>
      …
      <br/>
      Evet, dozunda delilik güzeldir!
      <br/>
      Zira birbirimizi ‘deli’ gibi sevmeliyiz.
      <br/>
      Çıkar gözetmeden,
      <br/>
      Kaz gelen yerden,
      <br/>
      Tavuğu esirgemeden!
      <br/>
      Kötülüklerden el ayak çekip
      <br/>
      Sadece ve sadece temiz bir niyetle,
      <br/>
      Sahip olduklarımızı paylaşıp
      <br/>
      Hayır işleriyle uğraşmalıyız.
      <br/>
      …
      <br/>
      Evet, dozunda delilik güzeldir!
      <br/>
      Zira ‘deli gücüyle’ mücadele etmeliyiz.
      <br/>
      Biz, bir koşmaya başladık mı!
      <br/>
      Hiçbir kötülük yıldırmamalı.
      <br/>
      Şeytanın elimizi kolumuzu bağlamaya,
      <br/>
      Yolumuza set çekmeye,
      <br/>
      Gücü yetmemeli.
      <br/>
      Deliler gibi gözümüz kararmalı!
      <br/>
      Etraftaki engelleri görmemeli.
      <br/>
      …
      <br/>
      Evet, dozunda delilik güzeldir.
      <br/>
      Böylelikle önyargılardan kurtuluruz.
      <br/>
      Kim zenginmiş,
      <br/>
      Kimin şöhreti varmış,
      <br/>
      Kim makam sahibiymiş,
      <br/>
      Bilmeden,
      <br/>
      Araştırmadan,
      <br/>
      Sormadan,
      <br/>
      Herkese eşit olarak yaklaşırız…
      <br/>
      Kulağımızı sağdan soldan gelen seslere tıkayıp
      <br/>
      Deliler gibi inandığımız yolda ilerleyip
      <br/>
      Israr etmeliyiz.
     </br>
    </br>
   </br>
  </br>
 </p>
