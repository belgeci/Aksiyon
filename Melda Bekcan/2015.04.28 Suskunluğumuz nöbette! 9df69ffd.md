# Suskunluğumuz nöbette!

*Melda Bekcan 2015.04.28*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Enfekte düşüncelerimiz var!
  <br>
   Örneğin suskunluğa dair…
   <br>
    Suskunluğumuzun nedeni bilmemek değil
    <br>
     Hatta tam tersi
     <br>
      Yalanları değil
      <br/>
      Gerçekleri tercih ediyoruz,
      <br/>
      Bildiğimiz için susuyoruz,
      <br/>
      En büyük sesi
      <br/>
      En sessiz çığlıklar çıkarırmış!
      <br/>
      Susuyoruz…
      <br/>
      Çünkü suskunluğumuz nöbette.
      <br/>
      Bir sen,
      <br/>
      Bir ben,
      <br/>
      Bir de diğerleri tutuyor nöbeti
      <br/>
      Artık sıra kime geldiyse
      <br/>
      Sırayla susuyoruz
      <br/>
      Aslında sessizce konuşuyoruz,
      <br/>
      Sessizce işitiyoruz.
      <br/>
      …
      <br/>
      Enfekte düşüncelerimiz var!
      <br/>
      Örneğin tatile dair…
      <br/>
      Püff! Tatilmiş!
      <br/>
      Bana tatilden bahsettiklerinde,
      <br/>
      İnsanlar, tatilden dolayı sevindiklerinde
      <br/>
      Gülüp geçiyorum!
      <br/>
      Tatil olsa ne olur?
      <br/>
      Tatil olmasa ne olur ki?
      <br/>
      İnsanın tatili kalbindedir,
      <br/>
      İnsanın tatili zihnindedir.
      <br/>
      Kalbin ve zihnin tatile kapandıysa şayet
      <br/>
      Yılın 365 günü tatil ilan edilse dahi
      <br/>
      Nafiledir.
      <br/>
      …
      <br/>
      Enfekte düşüncelerimiz var!
      <br/>
      Örneğin narkoza dair…
      <br/>
      Sanki narkozu sadece ameliyat esnasında alıyoruz.
      <br/>
      Kocaman bir yalan!
      <br/>
      Bedenimiz sağlıklıyken,
      <br/>
      İstediğimiz her yere hareket edebiliyorken
      <br/>
      Hiçbir şey yapmıyoruz.
      <br/>
      Ya hareket edemeseydik?
      <br/>
      Kim bilir neler yapmaya heveslenirdik
      <br/>
      Neler yapmayı hayal ederdik?
      <br/>
      Öyle değil mi?
      <br/>
      Yapmaya gücümüz varken,
      <br/>
      Yapmamak!
      <br/>
      Konuşmaya gücümüz varken,
      <br/>
      Konuşmamak!
      <br/>
      Duyduklarımızı, algılayamamak.
      <br/>
      Bunların hepsi narkoz etkisi değil mi?
      <br/>
      Ne anestezi uzmanına gerek var
      <br/>
      Ne de ameliyathaneye,
      <br/>
      Aslında hepimiz narkoz tesirindeyiz.
      <br/>
      …
      <br/>
      Enfekte düşüncelerimiz var!
      <br/>
      Örneğin paraya dair…
      <br/>
      Çocukluk yıllarından itibaren didin dur.
      <br/>
      Çoktan seçmeli sorular,
      <br/>
      Sınavlar,
      <br/>
      Ödevler,
      <br/>
      Üniversite,
      <br/>
      Diploma,
      <br/>
      Kariyer derken,
      <br/>
      Tam para kazanma hedefini tutturmuş,
      <br/>
      Mal mülk yerini bulmuşken
      <br/>
      Bir de bakıyorsun
      <br/>
      Hastalık gelip çatıyor.
      <br/>
      Sonrası havuz problemlerini andırıyor.
      <br/>
      Bir yandan musluk dolarken
      <br/>
      Diğer yandan boşaltıyor!
      <br/>
      Paralar, sağlık sektörüne harcanıyor.
      <br/>
      …
      <br/>
      Enfekte düşüncelerimiz var!
      <br/>
      Örneğin evliliğe dair…
      <br/>
      Ortalama bir yaş kotası konmuş.
      <br/>
      Yok efendim, 35’ten önce olmazmış,
      <br/>
      Kariyer yapmak lazımmış!
      <br/>
      İlla bir ev,
      <br/>
      Bir de araba şartmış!
      <br/>
      Ne alakası var?
      <br/>
      O imkânlara sahip olduktan sonra neden evleneyim ki?
      <br/>
      Zaten yolu yarılamış
      <br/>
      İstediklerimi elde etmişim!
      <br/>
      Mesele hayat arkadaşı bulmakta,
      <br/>
      Zorlukları birlikte göğüslemekte,
      <br/>
      Baş başa verip ağlamakta,
      <br/>
      El ele verip ilerlemekte.
      <br/>
      Haa! Bu gerçeği ben de biraz geç kavradım
      <br/>
      Hepinizin huzurunda itiraf ediyorum.
     </br>
    </br>
   </br>
  </br>
 </p>
