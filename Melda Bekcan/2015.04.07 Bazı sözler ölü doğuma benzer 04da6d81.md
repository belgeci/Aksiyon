# Bazı sözler ölü doğuma benzer

*Melda Bekcan 2015.04.07*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Hepimiz yeryüzüne ayak basıyoruz
  <br>
   Peki, hiç düşündün mü?
   <br>
    Acaba yeryüzünün neresine?
    <br>
     Hayal dünyasında yaşıyorsak şayet
     <br>
      Bastığımız zemin pek kaygan demektir,
      <br/>
      Her an vıjjjtttt diye kayabilir,
      <br/>
      Başımızda yıldızlar üşüşebilir!
      <br/>
      Yok, basiretle hareket ediyorsak
      <br/>
      Yaptığımız işleri düşünüp taşınıyor,
      <br/>
      Kötülüklerden sakınıyor,
      <br/>
      Faydalı işlerle uğraşmaya özen gösteriyorsak,
      <br/>
      Sırtımız kolay kolay yere gelmez!
      <br/>
      Kaplumbağa gibi yavaş
      <br/>
      Ama sağlam toprak üstünde ilerleriz.
      <br/>
      …
      <br/>
      Risklerle dolu bir hayat,
      <br/>
      Mayın tarlasında yürümeye benzer!
      <br/>
      Hedeflediğin istikamette gidersin
      <br/>
      Ama attığın bir adımın sonrasını göremezsin.
      <br/>
      Yüzde elli ihtimal vardır…
      <br/>
      Ya yaptığın işe devam edeceksin
      <br/>
      Ya da bomba patlayacak
      <br/>
      Sahip olduklarını bir bir kaybedeceksin!
      <br/>
      Art niyetli insanlarla beraberlik ise
      <br/>
      Dikenlerin üstünde dans etmekle eşdeğerdir.
      <br/>
      Birlikte hareket edersin etmesine de!
      <br/>
      Canın acır,
      <br/>
      Yaptığın işten keyif alabilmek, düşük ihtimaldir.
      <br/>
      …
      <br/>
      Hiç düşündün mü?
      <br/>
      Neden insanlar geri geri gitmez?
      <br/>
      Çünkü hayat hep ileriye yönelik kodlanmıştır.
      <br/>
      Zaman akar
      <br/>
      Asla durmaz,
      <br/>
      Geri dönüş yoktur
      <br/>
      Zira pek çok adım attık bugüne değin.
      <br/>
      Bazıları küçüktü,
      <br/>
      Bazıları hızlı ve büyük atıldı.
      <br/>
      Peki, farkında mısın?
      <br/>
      Adımların seni nereye götürüyor?
      <br/>
      Adımların, seni amacına kaç karış ulaştırıyor?
      <br/>
      Kim bilir!
      <br/>
      Belki de ilerlediğini zannederken
      <br/>
      Yerinde sayıyorsundur!
      <br/>
      …
      <br/>
      Hani ‘ağzı olan konuşuyor’ derler ya’
      <br/>
      Hepimiz konuşuyoruz
      <br/>
      Peki, hiç düşündün mü?
      <br/>
      Bu fiili nasıl uyguluyoruz?
      <br/>
      ‘Susmak gereken yerde konuşmak
      <br/>
      Ve konuşmak gereken yerde susmak
      <br/>
      İnsanı felakete götürür!’ demiş, ünlü düşünür.
      <br/>
      Hakikaten de iyi tartmalı,
      <br/>
      Nerede ve nasıl konuşmalı?
      <br/>
      Bazen ağzımızdan çıkan bir kelime,
      <br/>
      Nokta atışıyla isabet eder,
      <br/>
      Kurşun gibi saplanır,
      <br/>
      Hedefi tam on ikiden yere serer!
      <br/>
      …
      <br/>
      Bazen de sözcükler ölü yapılan doğuma benzer.
      <br/>
      İçinde aylarca saklarsın,
      <br/>
      Beslersin,
      <br/>
      Yaşatırsın,
      <br/>
      Büyütmeyi hedeflersin,
      <br/>
      Odalar hazırlarsın,
      <br/>
      İçini süsleyip püslersin,
      <br/>
      En sonunda kucağın boş kalır
      <br/>
      Ve daha da kötüsü!
      <br/>
      Hevesin kursağına yapışır
      <br/>
      Su bile içemezsin,
      <br/>
      Nefes alamazsın.
      <br/>
      …
      <br/>
      Hepimiz düşünüyoruz…
      <br/>
      Peki, hiç aklına takıldı mı?
      <br/>
      Bu eylemi hangi alanlarda kullanıyoruz?
      <br/>
      Örneğin Hz. Ömer kadar feraset sahibi miyiz?
      <br/>
      Başımıza gelecekleri, önceden hissedebilir miyiz?
      <br/>
      Bilir misin…
      <br/>
      Hz. Ömer kendisini şehit eden kişiyi önceden sezinlemiş!
      <br/>
      Bir gün çarşıda dolaşırken
      <br/>
      Mecusi bir köle yanına gelip
      <br/>
      Efendisinin aldığı yevmiyeyi şikâyet etmiş.
      <br/>
      Köleye yönelttiği soruların ardından
      <br/>
      Durumun normal olduğu hükmünü veren Hz. Ömer,
      <br/>
      Köleye ücretle yel değirmeni yapmasını teklif etmiş.
      <br/>
      ‘Sana bir yel değirmeni yaparım ki
      <br/>
      Doğuda ve batıda dillere destan olursun’ yanıtını alınca
      <br/>
      Tehdit edildiğini hemen algılamış
      <br/>
      Etrafındaki insanlara hissiyatını ifade etmiş,
      <br/>
      Ve hemen ertesi gün sabah namazında
      <br/>
      Mihraba çıktığı anda
      <br/>
      O köle tarafından hançerlenmiş.
     </br>
    </br>
   </br>
  </br>
 </p>
