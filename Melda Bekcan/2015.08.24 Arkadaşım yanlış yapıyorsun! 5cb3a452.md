# Arkadaşım yanlış yapıyorsun!

*Melda Bekcan 2015.08.24*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Hey, sen!
  <br>
   Sağa sola bakma.
   <br>
    Evet, sana sesleniyorum:
    <br>
     Arkadaşım, yanlış yapıyorsun.
     <br>
      Meslek seçmek kolay değildir...
      <br/>
      Öyle internete girip
      <br/>
      Üzerine en uygun kıyafeti seçip
      <br/>
      Siparişle kapına getirmeye benzemez.
      <br/>
      Üstelik!
      <br/>
      Beğenmediğin zaman
      <br/>
      Telefona sarılıp
      <br/>
      Şikâyetlerini sıralayıp
      <br/>
      Gerisin geri de gönderemezsin!
      <br/>
      …
      <br/>
      Meslek seçimi zordur,
      <br/>
      Matematiğe falan da benzemez.
      <br/>
      Önce aritmetik kurallarını öğrenip
      <br/>
      Sonra sayıları yan yana dizip
      <br/>
      Yanına eşitlik işareti koyup
      <br/>
      Sonuca ulaşman mümkün değildir!
      <br/>
      Bilimselliğin yanı sıra
      <br/>
      Görünmeyen, dokunulmayan,
      <br/>
      Sadece varlığı hissedilen
      <br/>
      Ruhsal bir yanı vardır.
      <br/>
      …
      <br/>
      Evet, hangi mesleği seçersen seç
      <br/>
      O duyguya bürünürsün.
      <br/>
      Örneğin bir film yönetmeni olacaksan,
      <br/>
      Herkesi yönetmeye başlarsın,
      <br/>
      Bu duygudan kaçamazsın!
      <br/>
      Sahneler çekilir,
      <br/>
      Ekip dağılır,
      <br/>
      Set biter,
      <br/>
      Ama sen kendini hâlâ ‘yöneten’ zannedersin
      <br/>
      Yönetmen koltuğundan kalkmak istemezsin!
      <br/>
      …
      <br/>
      Meslek seçimi gurmelik de değildir ki!
      <br/>
      Önüne pasta konulduğunda,
      <br/>
      Ağzına tatlı lezzetler gelmeyebilir.
      <br/>
      Hiç ummadığın bir anda
      <br/>
      Meksika acısında yanabilirsin!
      <br/>
      Lezzetler karmakarışıktır
      <br/>
      Belki daha önce hiç tatmamışsındır,
      <br/>
      Ayrım yapamazsın.
      <br/>
      Sakın ola burun kıvırma,
      <br/>
      Razı olmasını bilmelisin.
      <br/>
      …
      <br/>
      Bir kere hayaller kurmamalısın
      <br/>
      Haa!
      <br/>
      ‘İlle de hayal kuracağım’ diye ısrar ediyorsan
      <br/>
      O zaman başka!
      <br/>
      Ben bunca yıl okula gittim,
      <br/>
      Sabahın erken saatinde kalktım,
      <br/>
      Servise en önce bindim,
      <br/>
      Sokak sokak dolaştım,
      <br/>
      Senelerce mürekkep yaladım,
      <br/>
      Tatil yapmadım,
      <br/>
      Bu yaşıma değin hep test çözdüm!
      <br/>
      Diye ısrar ediyorsan
      <br/>
      Elbette!
      <br/>
      Yerden göğe kadar haklısın.
      <br/>
      Gelgelelim…
      <br/>
      Ben de sana doğruları söyleyeceğim;
      <br/>
      Yaşayacağın hayal kırıklığına
      <br/>
      Şimdiden hazır ve nazır olmalısın.
      <br/>
      …
      <br/>
      Ayrıca!
      <br/>
      Her şey bir kenara,
      <br/>
      Senin ailen de hata yaptı, arkadaşım.
      <br/>
      Neden mi?
      <br/>
      Çünkü onlar seni hayatın gerçeklerinden uzaklaştırdı.
      <br/>
      Tabii ki kötü niyetleri yoktu
      <br/>
      Ne yazık ki…
      <br/>
      Seni bir sınav makinesinden ibaret gördüler,
      <br/>
      Sen hayatın içine girmek istedin,
      <br/>
      Onlar izin vermedi.
      <br/>
      Sen, dışarıya dokunmak istedin,
      <br/>
      Reddedildin,
      <br/>
      Cam bir fanusun içine kapatıldın,
      <br/>
      Yiyeceğin yemek bile önüne getirildi.
      <br/>
      …
      <br/>
      Üniversite sınavı, senin tek hedefin,
      <br/>
      Hayatının merkezi oldu.
      <br/>
      Lâkin şunu bil ki;
      <br/>
      Kariyer sahibi olmak
      <br/>
      Her şey demek değil,
      <br/>
      Daha çok şey var öğreneceğin…
      <br/>
      Öğrendikçe,
      <br/>
      Bilmediğini fark edeceğin!
      <br/>
      Anladıkça,
      <br/>
      Anlaşılmazlar tarafından kuşatılacağın!
      <br/>
      İfade etmek istedikçe,
      <br/>
      Dilinin lâl kesileceği!
      <br/>
      …
      <br/>
      Daha ne diyeyim?
      <br/>
      Yapacağın üniversite tercihi,
      <br/>
      Vatana,
      <br/>
      Millete,
      <br/>
      Devlete,
      <br/>
      Ailene,
      <br/>
      Ve her şeyin de ötesinde
      <br/>
      Kendine!
      <br/>
      Hayırlı olsun arkadaşım.
     </br>
    </br>
   </br>
  </br>
 </p>
