# Sözlerine 'söz' geçiremeyenler!

*Melda Bekcan 2015.05.05*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Adımlarıma ‘söz’ geçiremiyorum
  <br>
   Çünkü beni dinlemez,
   <br>
    İstikametini kendi belirler.
    <br>
     Hep ileri gider,
     <br>
      Gerisin geriye gitmeyi hiç bilmez
      <br/>
      Hatta koşar adımlarla ilerler...
      <br/>
      İstiyorum ki biraz akıllı olsun,
      <br/>
      Başkaları tarafından ezilmesin,
      <br/>
      Kendine dikkat etsin.
      <br/>
      Ne yapsam da nafile!
      <br/>
      Bir kere yola çıkmayagörsün
      <br/>
      Çamurda yürür,
      <br/>
      Dikenlerden geçer,
      <br/>
      Ayakkabısı kirlenir,
      <br/>
      Yalınayak kalır,
      <br/>
      Tabanları parçalanır,
      <br/>
      Kan revan içinde kalır,
      <br/>
      Yen kırılır,
      <br/>
      Yine içinde kalır!
      <br/>
      Asla davasından vazgeçmez.
      <br/>
      …
      <br/>
      Gözlerime ‘söz’ geçiremiyorum,
      <br/>
      Usul usul yaşlar dökülüyor, kenarlarından,
      <br/>
      Hiç durmadan, duraksamadan!
      <br/>
      Oysa belli etmemeliler, düşüncelerimi,
      <br/>
      Anlatmamalılar, ne denli üzüldüğümü,
      <br/>
      Gözlerim, şahidi olmamalı hislerimin…
      <br/>
      Ne yapsam da nafile!
      <br/>
      Kirpiklerimdeki nemliliğe.
      <br/>
      Mâni olabilmem mümkün değil.
      <br/>
      …
      <br/>
      Sözlerime ‘söz’ geçiremiyorum!
      <br/>
      Kırık bir ses tonuyla çıkıveriyor heceler,
      <br/>
      Ne denli incindiğimi belli ediyor kelimeler.
      <br/>
      Bu durumdan rahatsızlık duyuyorum,
      <br/>
      Biz bize halledelim,
      <br/>
      Yabancılar dertlerimi anlamasınlar, istiyorum.
      <br/>
      Ne yapsam da nafile!
      <br/>
      Dudaklarımdan dökülüveren cümlelere,
      <br/>
      Mâni olabilmem mümkün değil.
      <br/>
      …
      <br/>
      Hislerime ‘söz’ geçiremiyorum!
      <br/>
      Tüm bedenimi sarıveriyor,
      <br/>
      Canları ne zaman isterse,
      <br/>
      Kılcallarıma değin işliyor.
      <br/>
      Duygularım, kulaklarını tıkayıp
      <br/>
      Mantığımı işitmiyor,
      <br/>
      Aldanıp ardından gidiyor insanların…
      <br/>
      İstiyorum ki biraz frene bassın,
      <br/>
      Temkinli olsun,
      <br/>
      Hemen hız yapmasın.
      <br/>
      Ama ne yapsam da nafile!
      <br/>
      Onları tahtından vazgeçiremiyorum.
      <br/>
      …
      <br/>
      Kalemime ‘söz’ geçiremiyorum!
      <br/>
      Bir kere kâğıda dokunmayagörsün
      <br/>
      Kanat takıp uçmaya başlıyor,
      <br/>
      Kendine laf söyletmiyor,
      <br/>
      Kimseyi kafasına takmıyor...
      <br/>
      İstiyorum ki daha temkinli olsun,
      <br/>
      Yavaş yavaş ilerlesin
      <br/>
      Yaptıkları dümdüz bir yazı olsun.
      <br/>
      Ne yapsam da nafile!
      <br/>
      Satırları sevmiyor,
      <br/>
      Şiir gibi yaşamayı,
      <br/>
      Daldan dala konmayı tercih ediyor.
      <br/>
      …
      <br/>
      Kaderime ‘söz’ geçiremiyorum!
      <br/>
      Bir milim kıpırdayamıyorum
      <br/>
      Adım dahi dışına çıkamıyorum, yazdıklarından
      <br/>
      Hep onun bildikleri gerçekleşiyor...
      <br/>
      Bazen istemeden şeytana uyuyorum!
      <br/>
      İstiyorum ki hayallerim gerçekleşsin,
      <br/>
      Yaptığım planlar bir bir yerine gelsin.
      <br/>
      Ne yapsam da nafile!
      <br/>
      Sadece ve sadece
      <br/>
      Levh-i mahfuzda yazılanları yaşıyorum.
      <br/>
      …
      <br/>
      Önyargılarıma ‘söz’ geçiremiyorum!
      <br/>
      İnsanlara pat diye yaftayı yapıştırıveriyor,
      <br/>
      Tanıştığım her insana bir kulp takıyor.
      <br/>
      Yok efendim!
      <br/>
      Kiminin görünümünü beğenmemiş,
      <br/>
      Dikkat etmek gerekirmiş!
      <br/>
      Sürekli beni cimdikliyor…
      <br/>
      İstiyorum ki biraz rahat bıraksın,
      <br/>
      Olayların akışına göre karar vereyim.
      <br/>
      Ne yapsam da nafile!
      <br/>
      Tıpkı Einstein’ın söylediği gibi!
      <br/>
      Atom çekirdeğini parçalamak,
      <br/>
      Önyargıları yok etmekten daha kolay, inanın…
      <br/>
      Ve derin bir sessizliğe bürünüyorum.
      <br/>
      Kendimle ilgili bu denli âcizken,
      <br/>
      Kendime dahi söz geçiremezken,
      <br/>
      Başkalarının yaptıklarını nasıl eleştirebilirim?
      <br/>
      Bu hakkı kendimde nasıl bulabilirim?
      <br/>
      Neden böyle bir cahillik yapıyorum?
      <br/>
      El cevap: Bilmiyorum.
     </br>
    </br>
   </br>
  </br>
 </p>
