# Kelimeler acayip yol kesiyor

*Melda Bekcan 2015.03.31*

<div class="pNewsDetailMainContent" itemprop="articleBody">
 <p>
  Güya kelam ediyoruz!
  <br>
   Aslı yok,
   <br>
    Astarı yok,
    <br>
     Kelimeler, uçurtma gibi uçuşuyor gökyüzüne,
     <br>
      Dalgalanıyor,
      <br/>
      İncecik bir iplikle bağlı yere.
      <br/>
      Hafiften bir meltem esse,
      <br/>
      Ya da ipi tutan kişinin ayağı tökezlese,
      <br/>
      İnceldiği yerden kopup
      <br/>
      Ufukta kaybolacak,
      <br/>
      Kimse yakalayamayacak.
      <br/>
      Zaten kimin derdinde?
      <br/>
      Peşinden giden de olmayacak.
      <br/>
      …
      <br/>
      Yazık!
      <br/>
      Gerçekten yazık ediyoruz, kelimelere.
      <br/>
      O an rüzgâr nereden eserse,
      <br/>
      Duruma göre cephe alınıyor!
      <br/>
      O dönem kim iktidar sahibiyse,
      <br/>
      Kimin elinde imkân varsa,
      <br/>
      Nabza göre şerbet dökülüyor!
      <br/>
      Ağzımızdan çıkan sözlerin hükmü yok,
      <br/>
      Arkasında duran yok.
      <br/>
      Anlayacağınız…
      <br/>
      Gerçek kimliğini yitirmiş kelimeler,
      <br/>
      Sahibinin yazdığı senaryonun gereği!
      <br/>
      Acayip rol kesiyor.
      <br/>
      …
      <br/>
      Keskin bir koku alıyor burnum.
      <br/>
      Rahatsız oluyorum
      <br/>
      Üstelik nereden geldiğini kestiremiyorum.
      <br/>
      Sanki çürümeye terk edilmiş
      <br/>
      Etrafına kötü kokular yayan beden gibi!
      <br/>
      Ama bu sefer bedenler yaşarken,
      <br/>
      Ruhlar, ölüme mahkûm edilmiş.
      <br/>
      Bir kenara yığılmış,
      <br/>
      Üst üste istiflenmiş…
      <br/>
      Düşün düşün çıldıracağım.
      <br/>
      Neden bana geliyor bu koku?
      <br/>
      Neden kimseler şikâyetçi değil?
      <br/>
      …
      <br/>
      Yazık!
      <br/>
      Gerçekten yazık ediyoruz benliğimize…
      <br/>
      Sadece dış görünüme odaklanan,
      <br/>
      ‘Kim ne demiş’ kaygısıyla hareket eden,
      <br/>
      Özünü araştırmayan,
      <br/>
      O gün geçerli olan karakter yapısını palto gibi
      <br/>
      Bir giyinip
      <br/>
      Sıkılınca çıkarıp atan!
      <br/>
      Bedenlerin mezara dönüştüğü,
      <br/>
      Ruhlara ‘el Fatiha’ çekilen,
      <br/>
      Sanki ruhu ölü hâlde yaşayan,
      <br/>
      Çoğunluğu oluşturan bir kesim var.
      <br/>
      …
      <br/>
      Çığlıklar duyuyor kulaklarım,
      <br/>
      Avazı çıktığı kadar bağıran var!
      <br/>
      Sanki bir bedene işkence yapılıyor
      <br/>
      Ama bu sefer bedenler sağlamken,
      <br/>
      Ruhlar acı çekiyor,
      <br/>
      Son nefesi vermek üzere feryat ediyor.
      <br/>
      Ruhlar sarsılıyor,
      <br/>
      Ruhlar nefes alamıyor,
      <br/>
      Sahip bedenlerce bir köşeye itilmiş
      <br/>
      Hazin sona doğru yavaşça yaklaşılıyor.
      <br/>
      …
      <br/>
      Yazık!
      <br/>
      Yazık ediyoruz kendimize.
      <br/>
      Neyin peşinde olduğumuzu bilmeden
      <br/>
      Kaptırıp gidiyoruz!
      <br/>
      Bir madde âlemi sarmış ki!
      <br/>
      Çevremizde ateşten bir çember var...
      <br/>
      Örneğin ‘paha’sı biçilemeyen evler!
      <br/>
      Onlar uğruna bir ömür çalışsan,
      <br/>
      Yemesen, içmesen, harcamasan,
      <br/>
      Yine de almaya güç yetiremezsin!
      <br/>
      Bir de ‘paha’sı biçilemeyen arabalar var ki…
      <br/>
      Hediye edip anahtarını verseler,
      <br/>
      Kontağı çevirmeye cesaret edemezsin!
      <br/>
      Dört lastik üstündeki metal yığını,
      <br/>
      Alçı, tuğla, demir parçaları,
      <br/>
      Bu denli pahalıyken,
      <br/>
      İnsan haysiyeti denen kavram
      <br/>
      Neden yerlerde çiğneniyor,
      <br/>
      Neden paspastan bile daha ucuz?
      <br/>
      …
      <br/>
      Simalarda bir perde görüyorum.
      <br/>
      Gerçeklerin önüne inmiş.
      <br/>
      Öyle ki!
      <br/>
      Gerçekleşmeyen hayaller kuran,
      <br/>
      Kabul olmayan dualar eden,
      <br/>
      Sonu olmayan yola giden,
      <br/>
      Dibi görülmeyen suya giren.
      <br/>
      Yani hakikatten uzak
      <br/>
      Hayatın hakkını veremeyen,
      <br/>
      Gerçek kimliğinden uzak kimlikler…
      <br/>
      Asıl yolunda yürümeyen,
      <br/>
      Tali yollara sapmış,
      <br/>
      Hedefe doğru gitmek isterken,
      <br/>
      Kafası daha da karışmış
      <br/>
      Çıkmaz yollara ulaşmış.
     </br>
    </br>
   </br>
  </br>
 </p>
