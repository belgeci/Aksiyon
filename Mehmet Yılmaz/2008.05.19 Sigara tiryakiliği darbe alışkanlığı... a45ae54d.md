   <font class="content">
    Tabii, içilmesine müsaade etmek de…
    <br>
     <br>
      19 Mayıs’tan itibaren özellikle kahvehanelerde, işyerlerinde, kamu binalarında sigara içilmeyecek artık.
      <br>
       <br>
        Hatta ticarî taksilerde bile…
        <br>
         <br/>
         Bir zamanlar şehirlerarası yolcu otobüslerinde duman altı olduğumuz günleri hatırlayınca, nereden nereye diyesi geliyor insanın…
         <br/>
         <br/>
         ***
         <br/>
         <br/>
         Sigara tiryakiliği, üzerinde önemle durulması gereken bir vakıa. Onca telkine, tavsiyeye rağmen insanlar bir türlü bu alışkanlıklarından vazgeçemiyorlar.
         <br/>
         <br/>
         Neden acaba?
         <br/>
         <br/>
         Bir irade zafiyeti mi söz konusu?
         <br/>
         <br/>
         Yoksa…
         <br/>
         <br/>
         Alışkanlıkları terk etmek sanıldığı kadar kolay değil mi?
         <br/>
         <br/>
         Hâlbuki tiryakileri sigaradan uzaklaştıracak birçok veri var ortada...
         <br/>
         <br/>
         En başta sağlıkla ilgili olanlar…
         <br/>
         <br/>
         ***
         <br/>
         <br/>
         Herkes biliyor ki bu ürün sağlığa zararlı.
         <br/>
         <br/>
         Sigaradaki 4 bin kimyasal maddeden 83’ü kanserojen etkisine sahip. Akciğer kanserlerinin yüzde 85-90’ı, kronik bronşitin yüzde 75’i, kalp hastalıklarının yüzde 25’i sigara ve tütün alışkanlığından kaynaklanıyor.
         <br/>
         <br/>
         Her sene 100 bin kişi, sigaraya bağlı hastalıklardan hayatını kaybediyor. Bu hastalıkların tedavisi için bir yılda harcanan para ise yaklaşık 50 milyar dolar.
         <br/>
         <br/>
         Dünya Sağlık Örgütü, 2020 yılında dünyada 10 milyon kişinin sigaradan öleceğini tahmin ediyor.
         <br/>
         <br/>
         Meselenin diğer boyutları da oldukça vahim.
         <br/>
         <br/>
         ***
         <br/>
         <br/>
         Türkiye’de son 20 yılda sigara tüketimi yüzde 80 oranında artmış durumda. Toplumun üçte biri sigara içiyor maalesef.
         <br/>
         <br/>
         Erkeklerin yarısı, kadınların da yüzde 20’si bu alışkanlığa müptela. 18 yaş üzeri sigara içme oranı ise yüzde 33,4…
         <br/>
         <br/>
         Halkımız her gün ortalama 15 milyon paket sigara tüketiyor. Yani kişi başına yılda ortalama 76,1 paket sigara içiliyor ülkemizde.
         <br/>
         <br/>
         Ya duman olup giden paralar? O da ayrı bir ıstırap aslında…
         <br/>
         <br/>
         Türk milletinin cebinden sigara için yılda ne kadar para çıkıyor biliyor musunuz?
         <br/>
         <br/>
         Tam 16,5 milyar YTL.
         <br/>
         <br/>
         Sırf sigara için kişi başına ortalama 233,8 YTL harcama yapılıyor Türkiye’de…
         <br/>
         <br/>
         Bu rakamın ne anlama geldiğini kavramak için devletin sağlığa bir günde ne kadar para harcadığına bakmak yeterli.
         <br/>
         <br/>
         18,1 milyon YTL…
         <br/>
         <br/>
         ***
         <br/>
         <br/>
         Sigara meselesinin dinî, ahlakî, insanî ve hukukî açıdan daha pek çok yönleri de var. Bu yazıda onlara girmek niyetinde değiliz; çünkü yerimiz kısıtlı.
         <br/>
         <br/>
         Fakat…
         <br/>
         <br/>
         Sigara alışkanlığı vesilesiyle dikkatlerinizi çekmek istediğimiz bir husus var. O da şu:
         <br/>
         <br/>
         Kaba saba yasaklarla, rencide edici yaftalarla, aşağılayıcı ithamlarla insanların alışkanlıklarını değiştirmek o kadar da kolay değil.
         <br/>
         <br/>
         Hele hakikat belledikleri alışkanlıklarından vazgeçirmek için insanların korku damarını işletmek ise hiç mümkün değil.
         <br/>
         <br/>
         Velev ki bu alışkanlık basit bir sigara tiryakiliği olsa da…
         <br/>
         <br/>
         ***
         <br/>
         <br/>
         İşte bu gerçekten yola çıkan Sağlık Bakanlığı, sigara ile etkili mücadele edebilmek için konsept değişikliğine gitti bu yıl. Kampanya vesilesiyle konuşan Sağlık Bakanı Recep Akdağ, “Biz bu meselenin cezalandırarak çözüleceğini pek düşünmüyoruz. Onun için kampanyada özellikle ‘yasak’ kelimesini kullanmadık.” diyor.
         <br/>
         <br/>
         Yani…
         <br/>
         <br/>
         Devlet, yasaklarla insanları ikna etmenin yol olmadığını anlamış görünüyor.
         <br/>
         <br/>
         Peki ya darbe yapmayı alışkanlık hâline getirenler…
         <br/>
         <br/>
         Toplumun değerleriyle alay etmeyi hayatının bir parçası hâline getirenler…
         <br/>
         <br/>
         Halkı küçümsemeyi aydın olmanın bir ön şartı kabul edenler…
         <br/>
         <br/>
         İnsanları, kurumları kategorilere göre tasnif etmeyi marifet sayanlar…
         <br/>
         <br/>
         Onlar şu gerçeği anlamışlar mıdır peki?
         <br/>
         <br/>
         Korkutarak, yıldırarak, tehdit ederek, söverek, döverek, yasaklayarak, özgürlüklerini elinden alarak insanları ve toplumları kalıptan kalıba sokmak mümkün müdür?
         <br/>
         <br/>
         Öyle olmadığını tarih söylüyor bize. Ve bundan sonra da söylemeye devam edecek.
         <br/>
         <br/>
         Ta ki toplum mühendisliğine soyunanlar, zihinleri ve kalpleri kazanmanın, alışkanlıkları değiştirmenin yolunun korkudan değil sevgiden geçtiğini anlayana kadar.
         <br/>
         <br/>
         İyi haftalar.
         <br/>
        </br>
