# 700 milyar nerede Perinçek?

*Bilinmeyen-yazar Bilinmeyen-tarih*

<div>
 <p>
  <font>
   Sermaye Piyasası Kurulu (SPK), Aydınlık Dergisi'nde 'Ulusal Kanal Direniyor Var mısın?' çağrısıyla banka hesabı açtırarak
   <br/>
   <br/>
   halktan para toplanmasını yasaya aykırı bularak söz konusu kanalı ikaz etti. Banka hesapları araştırıldığında ise İşçi Partisi Genel Başkanı Doğu Perinçek'in toplandı dediği 700 milyarın izine rastlanamadı. Para ortaklık payı adı altında toplanıyordu. Perinçek, 8 Ağustos 2002 tarihli Aydınlık'ta ortaklardan 700 milyar lira toplandığını açıklamıştı.
   <br/>
   <br/>
  </font>
 </p>
 <p>
  <font>
   (Bu yazı 1143 kez okunmuştur)
  </font>
 </p>
