   <br/>
   (İşçi Partisi’nde ayaklanma olunca)
   <br/>

   <br/>
   <br/>
   <br/>
   <b>
    SEZAİ OKAY
   </b>
   (Şehit Asteğmen Zeki Burak Okay’ın babası)
   <br/>
   Çocuk benim çocuğum, çocuğumu bu vatana helal etmiyorum. Benim evladım şehit değil. Çünkü savaşa gitmedi. Ne olduğu belirsiz bir savaşın içinde. Bir tek kurşun atamayan çocuk, sınırda bu ülkeyi nasıl korur?
   <br/>
   <br/>
   <br/>
   <b>
    BAŞBAKAN ERDOĞAN
   </b>
   (Şehit cenazesi görmek istemiyoruz diyen vatandaşa)
   <br/>
   Bakınız askerlik herhalde yan gelip yatma yeri değil. Hepimiz askerlik yapıyoruz, terör bir beladır, her ülkede bu var, buna karşı bu mücadeleyi uzun soluklu olarak yapıyoruz, yapacağız.
   <br/>
   <br/>
   <br/>
   <b>
    GEORGE W. BUSH
   </b>
   (ABD Başkanı)
   <br/>
   İran rejimi El Kaide kadar tehlikeli. Hizbullah örgütünü destekleyen İranlı yöneticiler, Amerika'ya mutlak düşmanlıklarını ilan etti. Amerika tiranların karşısında eğilmez...
   <br/>
   <br/>
   <br/>
   <b>
    FUAT SİNYORA
   </b>
   (Lübnan Başbakanı)
   <br/>
   (Tezkere oylamasının ardından Başbakan Erdoğan’a)
   <br/>
   Misafirimiz olacaksınız, Lübnan halkı yaptıklarınızı unutmayacaktır.
   <br/>
   <br/>
   <br/>
   <b>
    SELÇUK DERELİ
   </b>
   (Almanya’nın San Marino’yu 13-0 yendiği maçın hakemi)
   <br/>
   Müthiş bir olaydı, hayatımda böyle bir müsabaka yaşamadım. Sanırım bundan sonra da yaşamayacağım.
