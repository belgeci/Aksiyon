   ŞADİ ERGÜVENÇ (Emekli Tuğgeneral)
   <br/>
   (La Repubblica gazetesine verdiği demeçte)
   <br/>
   Gül, cumhurbaşkanı seçilince toplum sokaklara döküldü, bayram etti. Demokrasi yolunda seyreden askerden darbe beklenemez.
   <br/>
   <br/>
   ATİLLA KOÇ (Eski Kültür ve Turizm Bakanı)
   <br/>
   (Görevini devrettikten sonra gazetecilerin, ‘bakanlıktan sivil araçla ayrılıyorsunuz’ sözleri üzerine)
   <br/>
   Tabutun kırmızı plakası olmaz.
   <br/>
   <br/>
   DENİZ BAYKAL (CHP Genel Başkanı)
   <br/>
   Ben hiç dans etmedim. Aslında çok da teklif aldım ama sevmem ben.
   <br/>
   <br/>
   RAMAZAN KERİM ÖZKAN (CHP Burdur Milletvekili)
   <br/>
   Gül, resmen Türkiye Cumhuriyeti’nin 11. Cumhurbaşkanıdır. Tepkiler kişilere verilir, makama karşı olmaz.
   <br/>
   <br/>
   YAŞAR BÜYÜKANIT (Genelkurmay Başkanı)
   <br/>
   (30 Ağustos resepsiyonunda Cumhurbaşkanı Gül’ü selamlamadan yolcu edip, Meclis Başkanı Köksal Toptan’ı ise esas duruşta selamlamasının nedeni sorulduğunda)
   <br/>
   Yani böyle davranmayı uygun buldum. Eşiyle gelmiş olan benim Meclis Başkanımı baş selamıyla uğurladım. Gerekirse amuda bile kalkarım. O kadar...
   <br/>
   <br/>
   MECNUN OTYAKMAZ (Sivasspor Kulübü Başkanı)
   <br/>
   (TFF’nin kararına tepki olarak)
   <br/>
   Taraftarları tahrik edip, sahaya indirip kendimizi tokatlattırdığımız için ve 2-3 haftadır futbol kamuoyunu yaptığımız bu sıkıntılı hareketten dolayı da meşgul ettiğimiz için Türk futbol kamuoyundan özür diliyoruz.
   <br/>
   <br/>
   MESROB II (Türkiye Ermenileri Patriği)
   <br/>
   (TTK Başkanı Halaçoğlu’na cevaben)
   <br/>
   İnsanların soyunun sopunun irdelendiği, dinî aidiyetin sorgulandığı ortamlardan ürkmemek mümkün mü?
