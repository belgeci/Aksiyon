   SEMRA ÖZAL (8’inci Cumhurbaşkanı Turgut Özal’ın eşi)
   <br>
    (Özal’ın saçına) kökünden alınamadığı için maalesef otopsi yapılamadı. O ukde benim hep içimde.
    <br/>
    <br/>
    KADRİ YAMAÇ (Prof. Dr, Gazi Üniversitesi Rektörü)
    <br/>
    (8 Haziran 2006 tarihinde Teknik Eğitim Fakültesi Konferans Salonu’nda Yıl Sonu Akademik Genel Kurulu’nda 100 civarındaki öğretim üyesine hitap ederken)
    <br/>
    Kırmızı plaka sahibi kişi, devlet istihbaratının tahmin edersiniz çok önemli bölümleri paylaştığı kişilerdir. Ben bu fakültede isim isim maalesef, hangi hocanın hangi reislerle hangi görüşmeleri yaptığını sesli olarak biliyorum.
    <br/>
    <br/>
    YAŞAR KEMAL (Ünlü romancı, 1973’ten beri Nobel adayı)
    <br/>
    (Kadıköy Kitap Günleri’nin açılışında yöneltilen ‘Bir daha dünyaya gelseniz yine yazar mısınız?’ sorusu üzerine.)
    <br/>
    Yalan söylemeyeceğim. Bir daha dünyaya gelsem traktör şoförü olurdum. Çünkü ben gençliğimde traktör şoförlüğü yaptım. Hangi işe girsem hükümet beni attırıyordu.
    <br/>
    <br/>
    HAKAN ŞÜKÜR (Galatasaraylı millî futbolcu)
    <br/>
    Ben şu anda sporcu kimliğimle devam etmek istiyorum. Gerekirse siyasetin içinde de olmak isterim.
    <br/>
    <br/>
    ŞENOL GÜNEŞ (Teknik Direktör)
    <br/>
    (Üç yıllığına FC Seul ile anlaşması üzerine)
    <br/>
    Türkiye’de 40 yıllık birikimimi hayata geçirme şansı olmadığını gördüğüm için ülke dışında çalışacağım.
    <br/>
    <br/>
    ÖMER YÜNGÜL (Vestel İcra Kurulu Başkanı)
    <br/>
    Bora dayalı enerji yatırımları konusunda dünya çapında ses getirecek başka projelerimiz de var. Şu an açıklayamıyoruz. Şubat ayını bekleyin. Erke dönergecini  boş görmeyin, gelişmeleri izleyin.
