# Arnavut öğrenci Erdoğan’ı duygulandırdı

*Bilinmeyen-yazar 2005.02.21*

<div>
 <p>
  <font>
   Başbakan Recep Tayyip Erdoğan, Balkan gezisi sırasında Arnavutluk’ta
   <br/>
   <br/>
   Türk işadamları tarafından açılan Özel Turgut Özal Koleji’ni de ziyaret etti. Barona Tartari isimli Arnavut kız öğrenci, Necip Fazıl Kısakürek’in Kaldırımlar şiirini Türkçe okurken Erdoğan duygulandı.Başbakan Recep
   <br/>
   <br/>
  </font>
 </p>
 <p>
  <font>
   (Bu yazı 882 kez okunmuştur)
  </font>
 </p>
