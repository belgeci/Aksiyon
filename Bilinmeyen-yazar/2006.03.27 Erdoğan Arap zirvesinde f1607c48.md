# Erdoğan, Arap zirvesinde

*Bilinmeyen-yazar 2006.03.27*

<div>
 <p>
  <font>
   Başbakan  Tayyip Erdoğan, 28-29 Mart’ta Sudan’ın başşehri Hartum’da yapılacak Arap Birliği zirvesinin açılışında konuşacak. Konuşmasında Türkiye’nin ‘medeniyetler arası diyalog’ konusuna nasıl baktığını anlatacak.
   <br/>
   <br/>
   Irak’taki son gelişmelerin de masaya yatırılacağı zirvede Türkiye’nin statüsü ‘gözlemci üyelik’ ile aynı anlama gelen ‘daimi misafirlik’. Birliğin geçen yılki zirvesi, İspanya Başbakanı Jose Luis Rodriguez Zapatero’nun konuşmasıyla açılmıştı.
   <br/>
   <br/>
  </font>
