 <p>
  <font class="content">
   Amerikan karşıtı gösteri ve fikirlerin sert savunucusu Ulusalcı Hareket, internette kendini Amerika’ya teslim etti. Hareketin bayrağını kimseye kaptırmayan Ulusal Kanal, Aydınlık Dergisi ile ‘kahrolsun Amerika’ parolalarını dillerden düşürmeyen Türkiye Komünist Partisi de internet sitelerini California ve Dallas’taki bilgisayarlardan yayınlıyor.
   <br>
    <br>
     Avrasya Hareketi lideri Aleksandr Dugin’in fikirlerini esas alan, temel karakteri Amerikan karşıtlığı olan Ulusalcı Hareket’in bayraktarlığını yapan yayın kuruluşları, internet sitelerini Amerika’ya emanet etti. İşçi Partisi Genel Başkanı Doğu Perinçek’in yönetimindeki Aydınlık Dergisi (www.aydinlik.com.tr) ve Ulusal Kanal’ın (www.ulusalkanal.com.tr) web siteleri ABD’den yayın yapıyor. Ulusalcı yayın grubu, Türkiye’de bulunan 70’den fazla İnternet Servis Sağlayıcı ve 567 Web Hosting şirketini yok sayıp sitelerini Amerika’ya taşıdı.
     <br/>
     <br/>
     Avrupa Birliği ve Amerika’ya muhalif yayınlarıyla tanınan bu iki yayın kuruluşunun siteleri İtalya’nın Francavilla al Mare şehrinde bulunan 7Host SRL şirketine ait web sunucular üzerinden yayın yapıyor. 7Host SRL şirketinin web sunucuları ise ABD’nin California eyaletinde bulunan Electronic Communications Unlimited şirketinin Data Center’ında bulunuyor.
     <br/>
     <br/>
     Nisan ayından beri ‘ortaklık payı’ adı altında halktan 2,3 trilyon lira toplayan Ulusal Kanal, Sermaye Piyasası Kurulu (SPK) tarafından yapılan denetim sonucunda “Halktan usulsüz olarak para toplanamaz.” diye uyarılmıştı. Ulusal Kanal’ın, “ulusalcı değerlerin halka aktarılabilmesi” misyonunu lanse ederek halktan para toplamasına karşılık ODTÜ’den tescil ettirdiği sitesini AB merkezli bir şirketin ABD’deki web sunucusundan yayınlaması ABD karşıtı Perinçek’in Chevrolet marka araca binmesi gibi manidar bulunuyor.
     <br/>
     <br/>
     İşçi Partisi’nin web sitesi (www.ip.org.tr) ile ilgili durum gruptaki diğer sitelerden biraz farklı. İP’nin sitesi Türkiye’den yayın yapıyor. İP’nin web sitesi, Türk Silahlı Kuvvetlerini Güçlendirme Vakfı’nın bir kuruluşu olan Havelsan’da görevli Barış Dinç’in yönettiği web sunucuda barındırılıyor.
     <br/>
     <br/>
     Geçtiğimiz yıllarda Interpro’nun organize ettiği ilk 500 Bilişim Şirketi araştırmasında sektörel yazılımlar alanında yüzde 36,6’lık pazar payıyla birinci olan Havelsan, Ankara Ticaret Odası üyeleri arasında yapılan kurumlar vergi sıralamasında üçüncü durumda.
     <br/>
     <br/>
     Adalet Bakanlığı; Ulusal Yargı Ağı Otomasyonu (UYAP), Tapu ve Kadastro Genel Müdürlüğü Bilgi Sistemi (TAKBİS) ve Kara Kuvvetleri Komutanlığı gibi stratejik kamu kurumlarına ait bilişim projelerini yürüten Havelsan’ın İşçi Partisi’ne web sitesi yayınlama haricinde ne gibi destekler verdiği merak ediliyor.
     <br/>
     <br/>
     Ulusalcı politikalara destek veren, eski adıyla Sosyalist İktidar Partisi (SİP), yeni adıyla Türkiye Komünist Partisi (TKP) de ABD’ye yönelik eleştirileri ile tanınıyor. Aydınlık ve Ulusal Kanal ile aynı çelişkiyi gösteren TKP’nin web sitesi (www.tkp.org.tr) de ABD’nin Dallas şehrinde bulunan web sunucular üzerinden yayın yapıyor.
