   <b>
    DENİZ BAYKAL
   </b>
   (CHP Genel Başkanı)
   <br>
    Kabine, Tayyip Erdoğan kabinesi değil, Unakıtan kabinesi. Görülmeyen güç merkezi, Unakıtan’dır.
    <br/>
    <br/>
    <b>
     TUNCAY ÖZKAN
    </b>
    (Kanal Türk Televizyonu sahibi)
    <br/>
    Çukurova Grubu başına geldiğimde ilk işim Reha Muhtar’ı göndermek oldu. Çünkü etkisizleşme operasyonunun en önemli oyuncusu idi.
    <br/>
    <br/>
    <b>
     NECATİ ŞAŞMAZ
    </b>
    (Kurtlar Vadisi Irak filminin başrol oyuncusu)
    <br/>
    Ehl-i tasavvufum. Peygamber soyundan geliyorum. Dedem Kadirî şeyhlerinden.
    <br/>
    <br/>
    <b>
     TURGUT OKYAY
    </b>
    (Abdullah Öcalan’ı yargılayan mahkeme heyetinin başkanı)
    <br/>
    CHP’den aday olmak istedim. Apo’yu yargılayan hâkimi partiye alırsak, Kürt oylarını böleriz diye beni istemediler. Çok kırıldım.
    <br/>
    <br/>
    <b>
     METİN TEKÇE
    </b>
    (Hakkâri Belediye Başkanı)
    <br/>
    PKK terör örgütü değildir. İçişleri Bakanı Abdülkadir Aksu’nun Kürtlüğü benim için beş para etmez.
    <br/>
    <br/>
    <b>
     TAYYAR ALTIKULAÇ
    </b>
    (AK Parti İstanbul Milletvekili) - (Kaya Çilingiroğlu’nun Feraye Tanyolaç ile olan ilişkisi üzerine)
    <br/>
    Evlilik dışı ilişkiler meşru değildir. Bu tür ilişkiden doğan çocuklara da terminolojide ‘veled-i zina’ denir.
