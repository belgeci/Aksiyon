# Türkiye-İsrail Askeri Eğitim Anlaşması

*Bilinmeyen-yazar 1996.05.18*

  <font class="detail-spot">
   Elimize ulaşan bilgiler ve belge suretleri çerçevesiftde, Meclis ve kamuoyunca merak edilen, tartışmalara yolaçan anlaşmanın bazı maddelerinin tercümesini sunuyoruz:

 </div>
 <div id="newsText">
  <font class="detail-text">
   TURKIYE CUMHURIYETI GENELKURMAYI.LLE ISRAIL MILLI SAVUNMA BAKANUCI ARASINDA ASKERI EGITIM LŞALRUGI ANLAŞMASI TARAFLAR
   <br/>
   <br/>
   İki ülke arasındaki mevcut dostluk ve iş birliği ilişkilerinin daha da geliştirilmesinin önemini vurgulayarak,
   <br/>
   <br/>
   Askeri eğitim işbirliğinin her iki tarafında yararına olduğuna inanarak,
   <br/>
   <br/>
   Bu tip işbirliklerinin taraflar arasındaki ilişkilerin daha gelişmesine katkıda bulunacağını kabul ederek,
   <br/>
   <br/>
   Aşağıdaki hususlar üzerinde anlaşmaya vardılar;
   <br/>
   <br/>
   AMAÇ
   <br/>
   <br/>
   Bu anlaşmanın amacı, iki ülke arasında askeri eğitim alanında işbirliği gerçekleştirmektir.
   <br/>
   <br/>
   KAPSAM
   <br/>
   <br/>
   Bu anlaşma askeri eğitim kurumlarını, birlikleri, bu kurumlarda eğitim alacak personeli, bu birimlerdeki eğitimi, tarafların anlaşmaya konu olan elemanlarını, tarafların sorumluluklarını, işbirliğini ilgilendiren prensip ve prosedürleri kapsar.
   <br/>
   <br/>
   TANAMLAR
   <br/>
   <br/>
   "Gönderen ülke", değişim personelini gönderen; "Kabul eden ülke", değişim personelini kabul eden ülkedir.
   <br/>
   <br/>
   "Yetkili otorite"
   <br/>
   <br/>
   a. Türkiye'de Türk tarafının yetkili otoritesi Türk Genelkurmayı, İsrail tarafının yetkili otoritesi İsrail'in Ankara Büyükelçiliği'dir.
   <br/>
   <br/>
   b. İsrail'de, Türk tarafının yetkili otoritesi Telaviv'deki Türk Büyükelçiliği, İsrail tarafının yetkili otoritesi İsrail Savunma Bakanlığı'dır.
   <br/>
   <br/>
   "Grup ya da takım lideri", değişim personelinin gönderen ülke kanunlarına göre atanan en kıdemli elemanı; "Ev sahibi birlik", kabul eden ülkenin değişim personeline ev sahipliği yapan birlik, "Değişim birimi" gönderen ülkenin bu anlaşma maksadıyla kabul eden ülkeye gönderdiği birim, yerleştirilen birimdir.
   <br/>
   <br/>
   EGİTİMİN İÇERİGİ
   <br/>
   <br/>
   1. Eğif'imde işbirligi .aşağıdakilerini .ihtiva eder:
   <br/>
   <br/>
   a. Profesyonel uzmanlık alanlarında mütekabi1iyet esasınca, farklı seviyelerde işbir1iğini sağlamak.
   <br/>
   <br/>
   b. Askeri akademiler, birlikler ve karargahlar arasında karşılıklı ziyaret.
   <br/>
   <br/>
   c. Eğitim ve tatbikat uygulamaları.
   <br/>
   <br/>
   d. Birbirinin askeri tatbikatlarını izlemek amacıyla gözlemciler gönderilmesi.
   <br/>
   <br/>
   e. Bilgi, tecrübe sağlamaya yönelik gerçekleştirecek personel değişimi, özellikle askeri tarih, askeri müze, askeri arşivleri de
   <br/>
   <br/>
   kapsayan sosyal ve kültürel alanlarda.
   <br/>
   <br/>
   f. Donanına gemilerinin karşılıklı liman ziyaretleri.
   <br/>
   <br/>
   g. Askeri spor takımları ile sanat gruplarının tur ve karşılıklı ziyaretleri.
   <br/>
   <br/>
   h. Askeri film ve fotoğraf stüdyoları arasında işbirliği.
   <br/>
   <br/>
   2. Yukarıda zikredilen değişim etkinlikleri eğitim faaliyetlerinin gerçekleştirilmesi ve  aşağıdaki detaylarla ilgili hazırlıklar üzerinde taraflar anlaşacak ve koordinasyon sağlayacak:
   <br/>
   <br/>
   a. Eğitimin konusu.
   <br/>
   <br/>
   b. Eğitimin prensipleri.
   <br/>
   <br/>
   c. Eğitim süresi, tamamlanış tarihi,
   <br/>
   <br/>
   d. Gönderilecek personelin seçiminde uygulanacak kriterler.
   <br/>
   <br/>
   e. Kullanılacak diL.
   <br/>
   <br/>
   f. Eğitim ve diğer aktivitelerle ilgili olarak gönderen devlet tarafından yapılacak ödemelerin miktarı ve metodu.
   <br/>
   <br/>
   g. Gerekli görülen diğer noktalar.
   <br/>
   <br/>
   h. Sosyal ve kültürel aktivitelerin organizasyon şekli.
   <br/>
   <br/>
   i. Aktivitelerin sona ermesi ile ilgili metod.
   <br/>
   <br/>
   3. Değişim birliği, değişim personeli ve eğitim ile ilgili bilgi ve detaylar en az 4S gün önceden kabul eden ülkeye bildirilecektir.
   <br/>
   <br/>
   4. Taraflar hava kuvvetlerine bağlı uçak..llIIl! ların ve donanına gemilerinin karşılıklı ziyaretlerini artırmaya gayret gösterecektir.
   <br/>
   <br/>
   S. Taraflar, Silahlı Kuvvetler arasında sportif faaliyetlerin artırılması için gayret edecektir.
   <br/>
   <br/>
   6. Eğitim faaliyetleri dönüşümlü olarak İsrail ve Türkiye'de gerçekleşecektir.
   <br/>
   <br/>
   ASKERİ EGİTİM İŞBİRLİGİNİN GEREKLERİ
   <br/>
   <br/>
   ı. Eğitim, kabul eden ülkenin dilinde yapılacaktır. Bir yıldan az süren eğitim programlarında gönderen ülkenin isteği üzerine mütercimler kullanılabilir. Bu durumda gönderen devlet yeterli sayıda askeri/sivil mütercim gönderebilir. Kabul eden ülke de bu bakımdan istek üzerine yardımcı olabilir. Tercüme masrafları gönderen ülke tarafından karşılanacaktır.
   <br/>
   <br/>
   2. Taraflar, teknik ve daha düşük düzey anlaşmalarla işbirliği yapılacak uygulama alanlarının detaylarını kararlaştıracaktır. B. anlamda, tarafların temsilcileri biraraya gelecek ve bu anlaşma çerçevesindeki aktiviteleri koordine edecektir. Bütün eğitim istekleri, lojistik destek ve güvenlik düzenlemeleri bu toplantılarda tartışılacak ve kararlaştırılacaktır.
   <br/>
   <br/>
   3. İşbirliği, bu anlaşmada belirlenen çerçeve ışığında sürdürülecek, spesifik durum ve şartlar ayrıca hadise hadise ele alınacaktır. Her eğitim faaliyetinin kapsamı ve teknik yönleri yukarıda bahsedildiği gibi Uygulama Toplantıları'nda tartışılacak, anlaşılacak ve bu anlaşmaya ilave olarak eklenecektir.
   <br/>
   <br/>
   4. İşbirliği, iki ülkenin ihtiyaçları ve Çıkarları gözönüne alınarak gerçekleştirilecektir.
   <br/>
   <br/>
   UYGULAMALAR VE İŞBİRLİGİNDEN DOGAN YUKUMLULUKLER
   <br/>
   <br/>
   ı. Eğitim süresince gerek duyulan hizmet, alet ve malzemeler kabul eden ülke tarafından değişim personeline verilebilir.
   <br/>
   <br/>
   2. Gerekli görüldüğünde eğitim faaliyetlerini düzenlemek için Uygulama Düzenlemesi yapılabilir. Uygulama Düzenlemesi aşağıdaki hususları kapsayabilir:
   <br/>
   <br/>
   a. Eğitimin konusu.
   <br/>
   <br/>
   b. Eğitimin prensipleri.
   <br/>
   <br/>
   c. Eğitimin süresi, başlaması ve bitişi. d. Eğitim için gönderilen değişim perso
   <br/>
   <br/>
   neli için ön koşullar.
   <br/>
   <br/>
   e. Eğitimde kullanılacak diL.
   <br/>
   <br/>
   f. Mali konular.
   <br/>
   <br/>
   g. Eğitim ile ilgili gerekli görülen diğer meseleler.
   <br/>
   <br/>
   3. Bu anlaşmanın uygulanmasıyla ilgili  bir yıllık plan yapılacaksa, her faaliyet ile ilgili aşağıdaki h ususlar tesbit edilecektir:
   <br/>
   <br/>
   a. Eğitimin niteliği, adı ve gerçekleştirilecek yer.
   <br/>
   <br/>
   b. Faaliyetin başlangıç ve bitiş tarihleri.
   <br/>
   <br/>
   c. Eğitimin düzeyi ve değişim personelinin taşıması gereken ön koşullar.
   <br/>
   <br/>
   d. Kaulacak personel sayısı.
   <br/>
   <br/>
   e. Mali meseleler;
   <br/>
   <br/>
   ı. Eğitim paralı mı, parasız mı olacak?
   <br/>
   <br/>
   Paralıysa ödeme şekli.
   <br/>
   <br/>
   2. Tarafların yükümlülükleri, masraflar...
   <br/>
   <br/>
   f. Yıllık planın hazırlanmasından sonra ek eğitim gerekirse, bu mesele tarafların karşılıklı mütabakau ile çözülecektir.
   <br/>
   <br/>
   4. Değişim personeli kapsamında gelen kursiyerler, gönderen devlet tarafından, kendi ihtiyaçlarına göre ve kabul eden ülkenin imkanları gözönünde tutularak belirlenir.
   <br/>
   <br/>
   5. Değişim personeli, gönderen ülke tarafkarından kabul eden ülke ile yapılacak koordinasyonla belirlenir.
   <br/>
   <br/>
   6. Eğitim programı ve konuların kapsamı Ev Sahibi Birim ve eğitimi verecek kurum tarafından tesbit edilecektir.
   <br/>
   <br/>
   7. Sınavlar, derecelendirmeler, eğitim sertifikası ve diğer dokümanlar kabul eden ülke tarafından tesbit edilecektir.
   <br/>
   <br/>
   GÜVENLİK VE SINIFLANDIRILMIŞ BİLGİNİN GİZLİLİGİ
   <br/>
   <br/>
   ı. Taraflar arasında bu anlaşmaya bağlı olarak aktarılan her türlü bilgi ve tecrübe, taraflar arasında 31 Mart 1994'te imzalanan Güvenlik/Gizlilik Anlaşması hükümlerine tabi olacaktır.
   <br/>
   <br/>
   2. Değişim personeli, kabul eden ülkenin güvenlikle ilgili düzenlemelerine ve eğitim sırasında verilen bilgilerin gizliliği ilkesine uyacaktır.
   <br/>
   <br/>
   3. Değişim personeli, kabul eden ülkenin spesifik olarak belirlediği güvenlik düzenlemelerine uyacakur.	"
   <br/>
   <br/>
   4. Deği§im personeli, kendilerine verilen sınıflandırılmış bilgilerin gizliliği esasına uymaya eğitimin tamamlanmasından sonra da devam edecektir.
   <br/>
   <br/>
   5. Değişim personeli, dolaylı ya da dolaysız olarak basınla temas kurmayacak.
   <br/>
   <br/>
   6. Ev Sahibi Birlik, değişim personelinin Ev Sahibi Birimi'ndeki güvenliğinden sorumlu olacakur.
   <br/>
   <br/>
   Anlaşma, tarafların diğer anlaşmalarla yüklendikleri yükümlülükleri etkilemeyecek.
   <br/>
   <br/>
   OLAGANÜSTÜ HAL VE MİSYON KISITLAMALARI
   <br/>
   <br/>
   ı. Genel Değişim personeli üçüncü bir ülke ile misafir ülke arasındaki çauşmalara katılmayacak ve kabul eden ülkede iken çalıştırılmayacakur.
   <br/>
   <br/>
   2. Hatırlatarak;
   <br/>
   <br/>
   a. Gönderen ülke gerekli bulduğu zaman personelini geri çağırabilir. Kabul eden ülke geri dönüşün gerçekleşmesi için gerekli ted birleri alacakur.
   <br/>
   <br/>
   b. Savaş, silahlı çauşma, halk ayaklanması, uluslararası kriz gibi durumlarda değişim personeli gönderen ülkenin isteğiyle ülkeden aynlabilecektir.
   <br/>
   <br/>
   MALİ MESELELER
   <br/>
   <br/>
   ı. Genel: Bu anlaşma ile ilgili karşılıklı yükümlülükler Uygulama Düzenlemeleri ile belirlenecektir.
   <br/>
   <br/>
   2. Ödemelerve masraflar:
   <br/>
   <br/>
   a. Mali durumlarla ilgili durum ve özel şartlar Uygulama Düzenlemeleri ile belirlenecektir. Prensipte, taraflar herkesin kendi masrafını karşılamasını benimserler.
   <br/>
   <br/>
   b. Kabul eden ülke kendi tesislerinde ücretsiz olarak kalacak yer ve yiyecek temin edecektir.
   <br/>
   <br/>
   3. Şahıslar kendi borçlarını kendileri ödeyecektir.
   <br/>
   <br/>
   4. Vergiler:
   <br/>
   <br/>
   a. Değişim personeli, kabul eden ülkenin her türlü vergi yasalarına uyacakur.
   <br/>
   <br/>
   b. Bu anlaşmanın kapsamına giren amaçlarla gümrüklerden geçirilecek teçhizat, mal. zeme, alet ve rasyonel miktarlardaki mallar, kabul eden ülke tarafından gümrüklerden muaf tutulacakur.
   <br/>
   <br/>
   c. Taraflar bu anlaşma çerçevesinde gerçekeşecek ithalat, ihracat, gümrükleme ve mali muafiyetlerin kötüye kullanılmaması için gerekli tedbirleri alacakur.
   <br/>
   <br/>
   5. Prensip olarak taraflar, maaş, sosyal haklar, ulaşım masrafları vb. ile değişim personeli tarafından evsahibi birliğe yüklenen masrafları kendileri ödemeyi kararlaştırmışlardır.
   <br/>
   <br/>
   ANLAŞMANIN DEGİŞTİRİLMESİ
   <br/>
   <br/>
   Gerekli görüldüğünde taraflar anlaşmanın değiştirilmesini ya da gözden geçirilmesini yazılı olarak teklif edecekler. Bu durumda taraflar 30 gün içinde değişmesi istenen maddeleri tartışmaya başlayacaklar. 45 gün içinde netice alınamaz ise, yazılı nota ile 30 gün önceden bildirmek suretiyle isteyen taraf anlaşmayı sona erdirebilir. Değişiklikler Madde 20'nin hükümlerine göre uygulamaya girecektir.
   <br/>
   <br/>
   SÜRE VE SONA ERME
   <br/>
   <br/>
   ı. Bu anlaşmanın süresi 5 yıldır. Taraflardan birinin sona erme süresinden 90 gün önce nota ile anlaşmanın sona erdiğini bil. dirmemesi halinde anlaşma bir yıllık süreler için devam eder.
   <br/>
   <br/>
   2. Taraflardan biri diğerinin bazı hükümlere uymadığına kani olursa, yazılı olarak görüşme isteyebilir. Bu görüşmeler notayı takip eden 30 gün içinde başlayacaktır. 45 gün içinde netice alınamaz ise, yazılı nota ile bildirmek koşulu ile tarafların anlaşmayı geçersiz kılma hakkı olacakur.
   <br/>
   <br/>
   SORUMLULUK
   <br/>
   <br/>
   ı. Taraflar birbirlerine karşı, birbirinin Silahlı Kuvvetleri mensup ve çalışanlarına karşı kendi silahlı kuvvetlerinin kullandığı eşyaların zarar görmesi, kaybolması gibi durumlarda, bu zarar aşağıdaki hükümlere uygun ise görmezden geleceklerdir:
   <br/>
   <br/>
   a. Şayet bu olay değişim personelinin bir çalışanı tarafından resmi bir görevin ifası sırasında meydana gelmiş ise.
   <br/>
   <br/>
   b. Bu zarar karşı tarafın yine resmi görev icabı kullandığı araç, gemi ya da uçağın kullanımından doğmuş ise.
   <br/>
   <br/>
   2. Taraflar resmi görevlerinde iken meydana gelen ölüm, yaralanma gibi olaylarda birbirlerine karşı iddiada bulunmayacaklardır.
   <br/>
   <br/>
   3. Taraflar, bu anlaşmada hükmü geçen aktiviteler sırasında üçüncü taraflara yönelik olaylarda değişim personelini savunacakur.
