# Dağıtıma dikkat

*Bilinmeyen-yazar Bilinmeyen-tarih*

<div>
 <p>
  <font>
   Öncelikle mükemmel bir dergi çıkardığınız için sizlere çok teşekkür ediyorum. Bizler pazartesiyi dört gözle bekliyoruz ama bazı haftalar dağıtımında problem oluyor. Dağıtım konusuna biraz daha eğileceğiniz ümidiyle iyi çalışmalar diliyorum.
   <br/>
   <br/>
   Kerim Singin
   <br/>
   <br/>
   <br/>
   <br/>
  </font>
 </p>
