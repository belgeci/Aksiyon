# Eminönü’ne cebir formülü

*Bilinmeyen-yazar 2006.06.26*

<div>
 <p>
  <font>
   İstanbul Eminönü’ndeki yüzlerce imalathanenin taşınmasıyla ilgili tartışmaya Başbakan Recep Tayyip Erdoğan son noktayı koydu.
   <br/>
   <br/>
   Geçtiğimiz hafta düzenlenen 1’inci Eminönü Sempozyumu’nda konuşan Erdoğan, “Gerekli adımları gerekirse cebirle atacağız.” dedi. Bölge turizm merkezi; boşaltılan tarihî mekânlar otel olacak.
   <br/>
   <br/>
  </font>
