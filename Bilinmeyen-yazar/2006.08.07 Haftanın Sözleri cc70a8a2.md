     <br/>
     Kesinlikle sonuca yaklaşıyoruz ve sanırım bu birkaç gün içinde olacak.
     <br/>
     <br/>
     <b>
      EHUD OLMERT
     </b>
     (İsrail Başbakanı)
     <br/>
     Onların üniforması yok. Kimin masum olduğunu kimin olmadığını nereden bilebiliriz… Tarih bilgime bakarak Avrupa’da İsrail’den daha temkinli davranacak bir ülke bilmiyorum. Avrupalılar 10 yıl önce Kosova’da ne yaptı? Kosova’da kaç sivil öldü? Karşılaştırın.
     <br/>
     <br/>
     <b>
      DONALD RUMSFELD
     </b>
     (ABD Savunma Bakan)
     <br/>
     Düşman, Irak’tan çıksak Afganistan’dan çekilmemizi isteyecek, oradan çekilsek tüm Ortadoğu’dan çıkmamızı isteyecek. Ortadoğu’dan ayrılırsak düşman, bu defa İspanya’dan Filipinler’e kadar uzanan tüm bölgeden çekilmemiz emrini verecek.
     <br/>
     <br/>
     <b>
      LOUAİ SAKA
     </b>
     (Terör örgütü El-Kaide yöneticisi) (Mahkemedeki savunmasında)
     <br/>

     <br/>
     <br/>
     <b>
      FATİH KISAPARMAK
     </b>
     (Türkücü) (İzmir Foça’daki konserinde)
     <br/>
