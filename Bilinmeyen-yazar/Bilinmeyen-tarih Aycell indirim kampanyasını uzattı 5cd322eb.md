# Aycell, indirim kampanyasını uzattı

*Bilinmeyen-yazar Bilinmeyen-tarih*

<div>
 <p>
  <font>
   Aycell, Ekim-Kasım aylarında kapsama alanına yeni girecek illerdeki abonelerin yararlanması için indirim kampanyasını 31 Ocak 2003 tarihine kadar uzattı. Daha önce 31 Ekim’de sona ereceği açıklanan indirim kampanyası nedeniyle diğer GSM operatörleri Rekabet Kuruluna şikayette bulunmuşlardı.
   <br/>
   <br/>
   Halen 7 ilde hizmet veren Aycell, kampanya sonuna kadar, Ericsson ile sözleşmesini imzaladığı yeni baz istasyonlarını kurarak tüm il merkezlerine ulaşmayı hedefliyor.
   <br>
    <br>
     İndirim kampanyası süresince abone sayısını 260 binin üzerine çıkaran Aycell’in kampanya süresince faturalı ve faturasız hatlarda uyguladığı tarifeler şöyle:
     <br>
      <br>
       Faturalı hatlarda;
       <br>
        - Aycell’den Aycell’e arama ücreti: 48 bin TL/dakika,
        <br>
         - Aycell’den sabit telefonları arama ücreti: 218 bin TL/dakika,
         <br/>
         - SMS ücreti: 48 bin TL/mesaj.
         <br/>
         <br/>
         Faturasız hatlarda;
         <br/>
         - Aycell’den Aycell’e yönelik aramalarda, dakikada 4 kontör yerine 2 kontör
         <br/>
         - Aycell’den sabit telefonlara yönelik aramalarda, dakikada 10 kontör yerine 5 kontör
         <br/>
         - SMS tarifesi ise 1 kontör olarak uygulanıyor.
         <br/>
         (Kontör ücreti 55 bin TL)
         <br/>
         <br/>
         <br/>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </font>
 </p>
