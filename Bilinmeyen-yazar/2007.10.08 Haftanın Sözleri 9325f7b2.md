  <font class="content">
   WALL STREET JOURNAL (Eleştiri ve Görünüm köşesinde yayımlanan sözde Ermeni soykırımı konulu makaleden.)
   <br>
    Siyasetçilere geleceği düşünmeleri için maaş ödeniyor, geçmiş için değil. Tarihi tarihçilere bırakalım.
    <br/>
    <br/>
    <br/>
    LÜTFÜ DOĞAN (CHP’li eski Devlet Bakanı, 11’inci Diyanet İşleri Başkanı)
    <br/>
    Müslümanlara yapılan dayatmalar ortadadır. İslam, baskı ve kavgadan yana değildir. Bugün mahalle baskısından söz edenleri, İslam’ı öğrenmeye, Müslümanları anlamaya davet ediyorum.
    <br/>
    <br/>
    <br/>
    RECEP TAYYİP ERDOĞAN  (Başbakan)
    <br/>
    Demokratik, laik, sosyal bir hukuk devleti olan Türkiye Cumhuriyeti’nin teminatı; bu hükümettir, bu Meclis’tir, bu millettir.
    <br/>
    <br/>
    <br/>
    ALİ BABACAN (Dışişleri Bakanı ve Başmüzakereci)
    <br/>
    <br/>
    Irak adeta terörist grupların antrenman sahası haline geldi. Belli ki kontroller Amerikalılar tarafından tam yapılmadı ki, bu silahlar (PKK’lı) teröristlerin elinde çıkabiliyor.
    <br/>
    <br/>
    <br/>
    RENE VAN DER LİNDEN (Avrupa Konseyi Parlamenter Meclisi Başkanı)
    <br/>
    Bazı Türklerin yüreklerindeki korkuyu anlıyorum. Türkiye, Avrupa’nın bir parçası olmak istiyorsa, devlet ve din ayrı olmalıdır.
    <br/>
    <br/>
    <br/>
    FATİH AKIN (Almanya’nın Oscar’a aday gösterdiği ‘Yaşamın Kıyısında’ filminin yönetmeni)
    <br/>
    Türkiye’de askerlik yapmak zorundayım. Gönlüm yok. Zorlama olursa vatandaşlıktan çıkarım.
    <br/>
    <br/>
    <br/>
    KEMAL UNAKITAN (Maliye Bakanı) (Ödedikleri vergiden yakınan kuyumculara hitaben)
    <br/>
    Beyannameleri verdiniz; ama Allah’a emanet. Sektör olarak kayıt dışı çalışıyorsunuz.
