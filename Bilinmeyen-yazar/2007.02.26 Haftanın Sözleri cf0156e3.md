   ABDULLAH GÜL (Başbakan Yardımcısı, Dışişleri Bakanı)
   <br>
    Irak’ın cumhurbaşkanı ile konuşmayalım, öbürüyle konuşmayalım. O zaman ne olacak? İnsanlar düşmanlarıyla bile konuşur. Yanlış yapıyorlarsa bunu yüzlerine söyleriz. Düzeltmezlerse maliyetine katlanırlar.
    <br/>
    <br/>
    İBRAHİM AYDOĞDU (DTP Diyarbakır İl Başkanı)
    <br/>
    Kerkük’e yapılan bir saldırıyı, Diyarbakır’a yapılmış kabul ederiz.
    <br/>
    <br/>
    ERTUĞRUL ÖZKÖK (Hürriyet Gazetesi Genel Yayın Yönetmeni)
    <br/>
    Türklük kavramı utanılacak değil, övünülecek bir kavramdır.
    <br/>
    <br/>
    ROBERT KOÇARYAN (Ermenistan Cumhurbaşkanı)
    <br/>
    1915 yılında olduğu gibi işlediği suçları kabul etmeyen aynı komşuya sahibiz. Bu yapılanların tekrarlanma riski var.
    <br/>
    <br/>
    RECEP TAYYİP ERDOĞAN (Başbakan, AK Parti Genel Başkanı)
    <br/>
    Hiçbir gencin kalem yerine silaha sarılmasına, yanlış rol modellerine özenmesine razı olamayız.
    <br/>
    <br/>
    DAVİD SİMMONS (Pfizer Orta-Güney Avrupa Başkanı)
    <br/>
    Gelişmiş ülkelerde ilaçta büyüme oranları oturdu. Ekonomik gelişmeye bağlı olarak Türkiye’de büyük potansiyel görüyoruz.
    <br/>
    <br/>
    ÖNDER HALİSDEMİR (Vakıfbank Genel Müdür Yardımcısı)
    <br/>
    Faizler sıfırlanıp vade 50 yıla çıksa da konut yapımında genişleme olmazsa kira öder gibi ev sahibi olmak hayal.
    <br/>
    <br/>
    İBRAHİM TATLISES (Ses sanatçısı)
    <br/>
    Korsana karşı korsancılık yapacağız. Korsanın satıldığı yerlerde biz kendi korsanımızı kendimiz çıkaracağız. Ama şerefli korsan çıkaracağız.
