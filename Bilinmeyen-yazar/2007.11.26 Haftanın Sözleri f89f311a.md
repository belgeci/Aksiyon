  <font class="content">
   MUHAMMED BARAKE (Hadaş Partisi Genel Başkanı. İsrail Parlamentosu (Knesset) üyesi)
   <br/>
   Öyle görünüyor ki, Livni henüz Arapların ve İsrailli Filistinlilerin burada doğup kendi yurtlarında yaşadıklarını özümseyememiş. Onlar herhangi bir ülkeden göç etmediler, kendi yurtlarında yaşıyorlar.
   <br/>
   <br/>
   <br/>
   HILLARY CLINTON (New York Senatörü, Demokrat Parti başkan aday adayı)
   <br/>
   Başkan olduğumda önceliklerimden biri de Türkiye ile bozulan ilişkileri düzeltmek olacak. Türkiye’yi yeniden kazanacağız.
   <br/>
   <br/>
   <br/>
   AGE HAREIDE (Norveç A Milli Futbol Takımı Teknik Direktörü)
   <br/>
   Sanki ringe çıkmış boksör gibiydim. Emre ve Nihat’ın golleri ile rakibim beni iplere serdi.
   <br/>
   <br/>
   <br/>
   MUSTAFA KOÇ (Koç Holding Yönetim Kurulu Başkanı)
   <br/>
   Yatırım ortamını öyle bir hale getirmek lazım ki yabancı yatırımcı Romanya, Polonya veya Çek Cumhuriyeti’ne gideceğine, Türkiye’ye gelip sıfırdan yatırım yapmalıdır.
   <br/>
   <br/>
   <br/>
   İBRAHİM ŞAHİN (TRT’nin yeni Genel Müdürü)
   <br/>
   Bu kadro ile 40 kanal bile yönetirim.
   <br/>
   <br/>
   <br/>
   MURAT KARAYALÇIN (SHP Genel Başkanı)
   <br/>
   Kültürel açılım kurdele keserek başlatılmaz. Bunu doğallıkla hayata geçireceksiniz. Bölgeye en iyi arkadaşlarımızın gitmesi gerek. Yani yeni Recep Yazıcıoğlular, Gaffar Okkanlar. Bölgenin seveceği insanlar.
   <br/>
   <br/>
   <br/>
   HALUK KOÇ (CHP Samsun Milletvekili, genel başkan adayı)
   <br/>
   Kurultayın kazanılması için 5-10 dakikada binlerce üye kaydedildi. Kasten adam yaralamadan; esrar, uyuşturucu satıcılığından mahkûm olmuşlar üye yapılıyor. CHP’yi getirdikleri durum budur. İbret vesikasıdır bunlar.
