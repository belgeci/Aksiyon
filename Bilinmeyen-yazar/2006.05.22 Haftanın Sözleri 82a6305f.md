   <b>
    DENİZ BAYKAL
   </b>
   (CHP Genel Başkanı)
   <br>
    Geçmişte Çanakkale’de, Sakarya’da kurtardık. Şimdi vatanı önümüzdeki seçimde sandıkta kurtarmak zorundayız. Artık olay siyasi parti olayı olmaktan çıkmıştır.
    <br/>
    <br/>
    <b>
     ÖZHAN CANAYDIN
    </b>
    (Galatasaray Başkanı)
    <br/>
    100'üncü yılımız Ekim ayında bitiyor. Kimse bize 100’üncü yılınızda şampiyon olamadınız diyemez.
    <br/>
    <br/>
    <b>
     KEMAL UNAKITAN
    </b>
    (Maliye Bakanı)
    <br/>
    Dalgalanma olur, duvara çarpar geri gelir. Sular biraz hareketlendikten sonra durulur.
    <br/>
    <br/>
    <b>
     SÜLEYMAN DEMİREL
    </b>
    (9'uncu Cumhurbaşkanı)
    <br/>
    Mart 1960, Temmuz 1980'de erken seçim yapılsaydı, Türkiye'de ihtilal olmazdı.
    <br/>
    <br/>
    <b>
     GÜLER SABANCI
    </b>
    (Sabancı Holding Yönetim Kurulu Başkanı)
    <br/>
    Doların yükselmesi normaldir. Dünya borsalarında, ABD'de faizlerde bir artış oldu. Her şey şeffaf. Orada oluyor, bizi de etkiliyor. Bir ufak düzeltme. Serbest piyasa ve serbest kur bunun için var.
    <br/>
    <br/>
    <b>
     ERSİN ÖZİNCE
    </b>
    (Türkiye İş Bankası Genel Müdürü)
    <br/>
    Biz artık günlük olaylardan etkilenmeyecek kadar istikrara ve kendimize güveniyoruz. Türkiye'den çıkan yabancı olursa onun yerini biz alırız.
    <br/>
    <br/>

     GÜNERİ CİVAOĞLU
    </b>
    (Gazeteci-Yazar)
    <br/>
    Gazeteciliğin en sevdiğim taraflarından biri geyiğidir.
