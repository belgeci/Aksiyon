   MEVLÜT VURAL (İSKİ Genel Müdürü)
   <br>
    İstanbul’un çeşitli yerlerine açtığımız kuyularla ek su kaynakları temin ettik. Havalar bu şekilde giderse ekim sonuna kadar su problemimiz olmaz.
    <br/>
    <br/>
    BÜLENT ARINÇ (TBMM Başkanı)
    <br/>
    (Şehit Yarbay Melih Gülova’nın, Manisa’daki cenaze törenindeki çirkin sözlere cevaben)
    <br/>
    50 yılını Manisa’da geçirmiş bir kişiye nasıl hain dersiniz? Türkiye 70 milyon, siz 70 kişisiniz, acıyorum sizlere.
    <br/>
    <br/>
    ABDULLAH GÜL (Dışişleri Bakanı, Başbakan Yardımcısı)
    <br/>
    Gazetelerden okuduğumuz doğruysa, Türk Silahlı Kuvvetleri mensuplarının böyle deli saçması şeylere gereken tepkiyi koyup, ağızlarının payını verip, salonu terk etmeleri gerekirdi. Bilemiyorum, belki de öyle yapmışlardır.
    <br/>
    <br/>
    ATA ÖZER (İstanbul Milli Eğitim Müdürü)
    <br/>
    Kimden para isteniyorsa bize müracaatını yapsın. Kayıtlarda para istenmeyecek. İsteyenler hakkında soruşturma açacağım.
    <br/>
    <br/>
    UFUK URAS (Özgürlük ve Dayanışma Partisi (ÖDP) eski Genel Başkanı, bağımsız milletvekili adayı)
    <br/>
    (CHP’yi kastederek) Nerede sağcı, nerede en kapitalist varsa, solcu diye pazarlamaya çalışıyorlar.
    <br/>
    <br/>
    DENİZ BAYKAL (CHP Genel Başkanı)
    <br/>
    Türkiye’nin tarihî bir noktaya yaklaşmakta olduğu kanısındayım ve birisine düşüncelerimi anlatmam, onu test etmem lazım. Böyle bir ilişki içine girebileceğim insanların önde gelenlerinden birisidir Sayın Demirel.
