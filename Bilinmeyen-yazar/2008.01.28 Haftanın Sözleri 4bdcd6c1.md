   YAŞAR ENGİN SELİMOĞLU (Yargıtay 15’inci Hukuk Dairesi emekli Üyesi)
   <br>
    Biz laik hukukun, laik hâkimleriyiz. Kimseye laikliği tartışma imkânı vermeyiniz. Laiklik giderse, sizlerin yerine kadılar gelir.
    <br/>
    <br/>
    <br/>
    ALİ BULAÇ  (Sosyolog, yazar)
    <br/>
    Gülen hareketini veya bir başka toplumsal olayı sadece sosyolojinin imkânlarıyla anlamak mümkün değildir.
    <br/>
    <br/>
    <br/>
    KÖKSAL TOPTAN  (TBMM Başkanı)
    <br/>
    Yasama yetkisini kısıtlayabilecek, kullanımı engelleyebilecek, bölecek veya gölgeleyebilecek hiçbir güç yoktur.
    <br/>
    <br/>
    <br/>
    OKTAY VURAL  (MHP Grup Başkan Vekili)
    <br/>
    Hiçbir kurum, TBMM’nin görevini yapmasını engelleyemez. Kim engelleyecekmiş bizi? Bizim irademiz durduğu sürece kimse bir şey yapamaz.
    <br/>
    <br/>
    <br/>
    MUHSİN YAZICIOĞLU  (BBP Genel Başkanı)
    <br/>
    Eğer kanunları Yargıtay çıkaracaksa buyursunlar ülkeyi de yönetsinler. Yok, eğer görevleri yürütme ve yasamanın çıkarmış olduğu kanunları uygulamaksa herkes görevini yapmalı.
    <br/>
    <br/>
    <br/>
    RICK WAGONER  (General Motors Başkanı)
    <br/>
    Yeni (Opel) Astra Sedan üretimi için Türkiye yerine Polonya’yı seçtiğimiz için pişmanız.
    <br/>
    <br/>
    <br/>
    BİNALİ YILDIRIM (Ulaştırma Bakanı)
    <br/>
    Türkiye, sadece cep telefonu cihazı için 50 milyar dolar harcadı. Ar-Ge faaliyetlerine önem verilseydi, bu paranın yarısı Türkiye’de kalsaydı iyi olmaz mıydı? Yazık değil mi?
