# "Hukuk devletinde gizli kanun olmaz"

*Bilinmeyen-yazar 2002.03.23*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   Meğerse bir de "gizli kanunumuz" varmış. Susurlukları yaşamış bir ülkede, 2425 Sayılı olmasından başka hakkında pek bir şey bilinmeyen bir kanunun ortaya çıkması pek kimseyi şaşırtmadı ama biz yine Anayasa Profesörü Burhan Kuzu'ya sorduk:
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   Bir hukukçu olarak böyle bir kanundan haberiniz var mıydı?
   <br/>
   <br/>
   O yeni bir hadise deşil. Yıllardan beri biz hukukçular bunu konuşuruz.
   <br/>
   <br/>
   Böyle bir kanun olabilir mi bir demokratik hukuk devletinde?
   <br/>
   <br/>
   Biz  hadisenin bizi ilgilendiren boyutuyla ilgileniyoruz. Devletin sırrıymış, bilmem neymiş, bizim dışımızda kalıyor o işler. Vatandaşa ne getirir, ne götürür, bu önemli.
   <br/>
   <br/>
   Bildişiniz başka gizli kanun var mı?
   <br/>
   <br/>
   Bildişim tek kanun bu. Varsa da bilmiyoruz. Ama 100'den fazla gizli kararname var. Benim bir kısım davalar vesilesiyle, bilirkişilik münasebetiyle bu konuda karşılaştışım bazı olaylar var. Mesela Avrupa Birlişi başlamında, elektrik enerjisi indirimi yok demişiz. Sübvansiyon yapmayacaşız demişiz. Devlet dışarıya karşı yapmayacaşız diyor ama içeride bu tür şeyler yapıyor.
   <br/>
   <br/>
   Kanunun içerişinden haberdar mısınız?
   <br/>
   <br/>
   Nereden haberim olsun. Kimse bilmiyor ki. devletin filanca kurumlarındadır.
   <br/>
   <br/>
   Bir demokratik hukuk devletinde bir yasa maddesi gizlenebilir mi? Açıklaması var mı bunun?
   <br/>
   <br/>
   Prensip olarak böyle şey olamaz. Anayasa gereşi hukuk devletinde gizli bir kanun veya mevzuat olmaz. Bu kanunun içerişini milletvekillerine vermeleri gerekir, kimin elindeyse.
   <br/>
   <br/>
   Bu kanuna dayanarak ceza veya bir hak verilebilir mi mahkeme tarafından?
   <br/>
   <br/>
   Resmi Gazete'de yayınlanmadışından dolayı vatandaşlara bir yükümlülük getirmiyor. Vatandaş bu kanunun içerişinden sorumlu olmaz. Gerçi yayınlandı diyenler de var.
   <br/>
   <br/>
   Batılı ülkelerde böyle kanun var mı?
   <br/>
   <br/>
   Bildişim kadarııyla yok. Ama savaş yıllarında olmuş olabilir.
   <br/>
   <br/>
   AB'nin kapısında olduşumuz sırada böyle kanunun var olmaya devam etmesi doşru mu?
   <br/>
   <br/>
   Varlışını devam ettirdişi de belli deşil. Ölü hüküm haline gelmiş olabilir. Hukuk devletinde böyle bir şeyi doşru bulmuyoruz.
   <br/>
  </font>
