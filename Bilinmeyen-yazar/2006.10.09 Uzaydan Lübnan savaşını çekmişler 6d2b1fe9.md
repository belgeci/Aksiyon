# Uzaydan Lübnan savaşını çekmişler

*Bilinmeyen-yazar 2006.10.09*

<div>
 <p>
  <font>
   İlk kadın uzay turisti Enuşe Ensari, astronot Jeft Williams ile beraber yeryüzüne dönen Rus kozmonot Pavel Vinogradov, Lübnan’daki savaşı, özellikle de yangınları uzaydan çıplak gözle görebildiklerini söyledi:
   <br/>
   <br/>
   “Bilirsiniz, insanoğlunun yaptığı olumsuz şeyler, özellikle de askerî operasyonlar uzaydan görülmektedir.” Uzay istasyonundan savaşı görüntülemişler.
   <br/>
   <br/>
