   <br/>
   <br/>
   PROF. DR. ERDOĞAN TEZİÇ (YÖK Başkanı)
   <br/>
   (Terör gündemli Rektörler Komitesi toplantısının ardından)
   <br/>
   Öfkemiz ne denli büyük olursa olsun, bunun iç barışımızı tehdit etmesine izin vermeyeceğiz.
   <br/>
   <br/>
   <br/>
   SÖNMEZ KÖKSAL (Milli İstihbarat Teşkilatı (MİT) eski Müsteşarı)
   <br/>
   Kamuoyu, sınır ötesi harekât yapılsa terör bitecekmiş gibi bir havaya sokuldu. Bu, psikolojik hata.
   <br/>
   <br/>
   <br/>
   HARUN ÖZTÜRK (Ülkü Ocakları Eğitim ve Kültür Vakfı Genel Başkanı)
   <br/>
   Gösteriler sırasında provokasyonlar olur, bir çatışma ortamı çıkarsa, ülkücüler, devletimizin milli birliği ve beraberliğini düşünerek, sokaklardan geri çekilir.
   <br/>
   <br/>
   <br/>
   DOÇ. DR. SEDAT LAÇİNER (Uluslararası Stratejik Araştırmalar Kurumu Başkanı)
   <br/>
   Suriye’deki (PKK) yapılanma(sın)da gruba liderlik yapanlar Ermenilerdir. Örgütün stratejik ve ideolojik yapılanmasında bu insanlar çok etkili.
   <br/>
   <br/>
   <br/>
   SERJ SARKİSYAN (Ermenistan Başbakanı)
   <br/>
   Ben Türklerin ne demek istediğini anlamıyorum. Şu anda Türkiye ile aramızda bir ilişki yok. Olmayan bir şeye de zarar veremezsiniz.
   <br/>
   <br/>
   <br/>
