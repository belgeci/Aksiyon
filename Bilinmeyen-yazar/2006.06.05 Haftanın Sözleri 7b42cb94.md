   <b>
    VECDİ GÖNÜL
   </b>
   (Milli Savunma Bakanı)
   <br/>
   Danıştay’ın basılması bence çok büyük bir olay. Tarihimizde, kamu makamlarına karşı yapılmış, Binbaşı Enver’in Yakup Cemil ile birlikte gerçekleştirdiği ve kendi komutanını vurduğu Babıâli baskınından bu tarafa yaşanmamış büyüklükte bir olaydır.
   <br/>
   <br/>
   <b>
    ENGİN ARDIÇ
   </b>
   (Gazeteci-yazar) [Danıştay’daki saldırının amacının sorulması üzerine]
   <br/>
   Görünürdeki amaçları hükümete gözdağı vermek, hükümeti erken seçime zorlamak, Çankaya’ya AK Parti’den birinin çıkmasını önlemek. Olayın dış boyutunu bilemiyoruz.
   <br/>
   <br/>
   <b>
    DENİZ BAYKAL
   </b>
   (CHP Genel Başkanı)
   <br/>
   Gel vatandaş gel, kim olursan ol gel, diyoruz. İster sağcı ol, ister muhafazakâr, ister liberal... Gün Atatürk Türkiye’sine sahip çıkma günüdür.
   <br/>
   <br/>
   <b>
    GEORGE W. BUSH
   </b>
   (ABD Başkanı)
   <br/>
   Türkiye ve Yunanistan’ı II. Dünya Savaşı’ndan sonra komünizmden biz kurtardık.
   <br/>
   <br/>
   <b>
    MAHMUD AHMEDİNECAD
   </b>
   (İran Cumhurbaşkanı)
   <br/>
   Dünya çapında faaliyet gösteren Siyonist şebekesi, Dünya Kupası’nı izlemek için Almanya’ya gitmemi engellemeye çalışıyor.
   <br/>
   <br/>

    MEHMET KISAKÜREK
   </b>

   <br/>
