   PROF. DR. MESUT PARLAK (İstanbul Üniversitesi Rektörü)
   <br>
    Bizim başörtüsü ile sorunumuz yoktur. Bu ülkenin de başörtüsü ile sorunu yok, türbanla sorunu var.
    <br/>
    <br/>
    <br/>
    TARHAN ERDEM (İsabetli anketlerle ün yapan Konda Araştırma Şirketi’nin sahibi)
    <br/>
    Yakın zamana dek ülkede örtünmenin, türban takanların sayısının azaldığını düşünüyordum. Ama şimdi türbanın arttığı izlenimine sahibim.
    <br/>
    <br/>
    <br/>
    HÜSAMETTİN CİNDORUK (TBMM eski Başkanı)
    <br/>
    Taslakta devletin laikliği gidiyor, kişinin laikliği çoğaltılıyor. Gül de aynı şeyi istiyor. Oysa devlet açısından laiklik, Türkiye’nin temel kuralıdır. Bu giderse cumhuriyet biter. Türkiye Malezya olur.
    <br/>
    <br/>
    <br/>
    CELAL TALABANİ (Irak Cumhurbaşkanı)
    <br/>
    (PKK’ya hitaben) Türkiye’deki saldırılarınız demokrasiyi ve Kürt halkının haklarını vuruyor.
    <br/>
    <br/>
    <br/>
    TASOS PAPADOPULOS (Kıbrıs Rum Kesimi Lideri)
    <br/>
    Siyasi arenada farklı fikirleri savunan rakiplerimiz olabilir. Rakiplerimiz bizim düşmanımız değil. Bizim bir tek düşmanımız var, o da Kıbrıs’taki Türk ‘işgal’ ordusudur.
    <br/>
    <br/>
    <br/>
    FATİH TERİM (Milli Futbol Takımları Teknik Sorumlusu)
    <br/>
    (2-2 sonuçlanan Malta-Türkiye maçı sonrasındaki sert eleştiriler üzerine.)
    <br/>
    Ben bundan sonra ders almam ders veririm. Prestijimi ve savaşarak, vuruşarak, emek vererek elde ettim. Tarih açık ve net ortada.
    <br/>
    <br/>
    <br/>
    ERMAN TOROĞLU (Futbol yorumcusu, eski hakem)
    <br/>
    Fatih Hoca’ma şunu sormak isterim. Yarın bir gün işler ters giderse, biz hesabı İçişleri Bakanlığı’ndan mı soracağız?
