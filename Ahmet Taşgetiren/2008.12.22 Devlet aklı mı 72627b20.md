   <font class="content">
    CHP’de başlayan “çarşafa meşruiyyet” açılımı acaba “devlet aklı”nın ürünü mü?
    <br>
     <br>
      Devletin bir yerlerinde, mesela askerî cenahta, köklü bir değerlendirme yapıldı ve devlet partisi CHP kanalıyla, devletin din ile ilişkisinde halkın beklentileri istikametinde bir esnemeye karar mı verildi?
      <br>
       <br/>
       Baykal’ın çıkışı, böyle bir devlet tavrının sonucu mu?
       <br/>
       <br/>
       Devlet aklını temsil eden Kırmızı Kitap’ta yazılanları mı hayata geçiriyor Baykal?
       <br/>
       <br/>
       Bütün bu sorular, Türkiye’de, Sovyetler’dekine benzer bir “Perestroyka-Yeniden Yapılanma” hamlesini akla getirdiği için, sistemden böyle bir açılım bekleyen sistem mağduru çevrelerde, özellikle dindar muhitlerde heyecan uyandırıyor.
       <br/>
       <br/>
       - 85 yıl sonra devlet, kendi sancılı alanlarını tedavi etmek niyetiyle geç kalmış bir hamle içine mi girdi?
       <br/>
       <br/>
       Ben, Cumhuriyet kurulurken belirlenen parametrelerde sistem-toplum ilişkileri sağlıklı tanzim edilemediği için, “yığınakta hata” yapıldığını ve bunun Türkiye’nin 85 yılını zora soktuğunu düşünenlerdenim.
       <br/>
       <br/>
       Osmanlı’nın yıkıldığı ve Cumhuriyet’in kurulduğu süreçte, bir büyük hamleye ihtiyaç vardı.
       <br/>
       <br/>
       Osmanlı’nın çözülüş süreci de, Cumhuriyet’e geliş süreci de, toplumun büyük yıkımlar yaşadığı, yorulduğu, hayattan bezdiği, diri güçlerini savaşta yitirdiği bir süreçti.
       <br/>
       <br/>
       Millî Mücadele, bu süreçte gerçekten ancak “can havliyle” diye nitelenebilecek bir ayağa kalkıştı.
       <br/>
       <br/>
       Bu yorgun toplumdan yepyeni bir hamle üretmek...
       <br/>
       <br/>
       Kolay değildi.
       <br/>
       <br/>
       Böyle bir toplumu, ona yeni yorgunluklar yükleyecek bir kargaşanın içine çekmek sağlıklı olmazdı.
       <br/>
       <br/>
       Belki bu yeni hamlenin tabanı, zemini, ateşleyici gücü, yine “Millî Mücadele” ruhuyla olmalıydı.
       <br/>
       <br/>
       Orada, din-devlet-toplum bütünleşmesi vardı.
       <br/>
       <br/>
       Devlet, canı boğazına geldiği bir noktada “Din ve Devlet için ayağa kalk” çağrısı yapmış, millet de bu çağrıya canıyla, kanıyla cevap vermişti.
       <br/>
       <br/>
       Sonra...
       <br/>
       <br/>
       Sistem yapılanırken...
       <br/>
       <br/>
       İşte orada, müthiş bir kırılma yaşandı.
       <br/>
       <br/>
       Toplumun en hassas olduğu alanda, içinden çıkılmaz bir tartışma içine girildi.
       <br/>
       <br/>
       Toplum-Din-Devlet ilişkileri, topluma kolay izah edilemeyecek bir ayrışmaya veya yeniden düzenlemeye tabi tutuldu.
       <br/>
       <br/>
       Bu millî mücadele çizgisiyle de bir ayrışmaydı.
       <br/>
       <br/>
       Ondan sonrası, devlet-toplum ilişkilerinde hep gerilimdir.
       <br/>
       <br/>
       Bu bir projeydi.
       <br/>
       <br/>
       Osmanlı’nın son çözülüş - arayış günlerinde “Batıcı tez” olarak gündeme gelen çizginin, devlet politikası hâline gelmesi idi.
       <br/>
       <br/>
       Osmanlı’da “batıcı tez” de çok tartışılmıştı.
       <br/>
       <br/>
       Batı’dan bir şeyler alınmalıydı, Osmanlı’nın yapmayı başaramadığı bazı olumlu şeyler vardı, ama bu, kendi kendimiz kalarak olmalıydı.
       <br/>
       <br/>
       Oysa üzerine “Cumhuriyet” damgası vurulan yeni proje, “İdare-i maslahatçılıkla esaslı devrim yapılamaz” diye yola çıkacak ve şok edici adımlar atacaktı.
       <br/>
       <br/>
       Doğrusu toplumda şok yaşandı.
       <br/>
       <br/>
       Şokun karşılığı ise, heyecan ve projeye sahipleniş değil, toplum küskünlüğü idi.
       <br/>
       <br/>
       Bu dönemde Cumhuriyet Halk Fırkası’na, projeyi hayata geçirecek devlet aygıtı rolü verildi.
       <br/>
       <br/>
       Devlet-toplum ilişkisi, bir noktada, CHP-toplum ilişkisi hâlinde somutlaştı.
       <br/>
       <br/>
       Küskünlük ise, CHP’ye yöneldi.
       <br/>
       <br/>
       Aslında, Tek Parti yönetimi hâlinde gerçekleşen bu dönemde, yönetim kadrosu, toplum küskünlüğünü görebilirdi.
       <br/>
       <br/>
       Ama yaptığı her işin “halk için” olduğunu düşünüyor, halkın kendi çıkarına olan şeyleri algılayamadığına inanıyor, ve devrimlerin gerekirse “halka rağmen” yapılacağı ön kabulüne göre hareket ediyordu.
       <br/>
       <br/>
       27 yıl bu gerilim içinde geçti.
       <br/>
       <br/>
       Bu dönemde, halkın sesinin duyulabildiği bütün oluşumlarda “Bu işte yanlış bir şeyler var” tepkisini algılamak mümkündü.
       <br/>
       <br/>
       Ama bu sesler, ağır baskılarla susturuldu.
       <br/>
       <br/>
       Baskıların mantığına, “Sen bilmezsin ben bilirim” yaklaşımı hâkimdi.
       <br/>
       <br/>
       Benim “Bir tür toplumsal sera ortamı” dediğim bu süreçte yeniden oluşturulan bir toplum kesimi vardı ve hâkim irade, bu toplum kesiminin varlığını, genel toplum içindeki yüzdesine bakmadan, “İşte bakın, nasıl oluyormuş” gibisinden bir yaklaşımla, genel toplum tepkilerini dışlamanın gerekçesi olarak kullandı.
       <br/>
       <br/>
       CHP ile toplum arasındaki gerilim, çok partili hayata geçtikten sonra daha da belirginleşti.
       <br/>
       <br/>
       CHP yola çıkarkenki proje mühendisliğini sürdürüyor, buna karşılık, toplumun geniş kesimlerinden de kopmuş olduğunu görüyordu.
       <br/>
       <br/>
       Yüzde 30’lardaydı önce.
       <br/>
       <br/>
       Sonra yüzde 15-20’lere kadar geriledi.
       <br/>
       <br/>
       Bunun sonucu olarak, 2000’li yıllarda bile Türkiye, 85 yıl öncesinin sancıları ile boğuştuğunu gördü.
       <br/>
       <br/>
       Türkiye’nin en büyük partisinin (yüzde 47 halk oyu) göğsüne “laiklik karşıtı eylemlerin odağı” yaftasının yapıştırıldığı tarih, 2008’dir.
       <br/>
       <br/>
       Türkiye’nin en büyük partisini laiklik karşıtı eylemlerin odağı suçlamasına muhatap kılan ve kendisini sistem koordinatlarını koruma misyonu içinde gören siyasi oluşum ise, yüzde 15-20 arası bir noktadadır.
       <br/>
       <br/>
       Bu görüntü, sistem-halk ilişkisinde hâlâ bir sancı bulunduğunun ispatıdır ve orada kilit rolde CHP vardır.
       <br/>
       <br/>
       İşte bu CHP, sistem-toplum kırılmasında en kritik noktada duran bir hadisede şerit değişikliği yapabileceği izlenimini veriyor.
       <br/>
       <br/>
       CHP bunu, başörtüsü yasağını perçinleyen Anayasa Mahkemesi kararına gelişte üstlendiği başvuru misyonundan sonra yapıyor.
       <br/>
       <br/>
       Ve CHP bunu yaparken, başörtüsü ile ilgili misyonunu terketmediğini vurgulayarak yapıyor.
       <br/>
       <br/>
       Bu ne oluyor o zaman?
       <br/>
       <br/>
       Şu andaki görüntü, henüz “yığınaktaki hata”nın gerçek anlamda tamiri niteliğinde değil.
       <br/>
       <br/>
       Henüz CHP; halkın kız çocuklarını başörtülü olarak okutabilme ve kamu görevinde istihdam talebini “Devlete türban giydirme” olarak niteliyor, dışlıyor.
       <br/>
       <br/>
       Bunu, kadın nüfusun yüzde 70’inin başını örttüğü bir ülkede yapıyor.
       <br/>
       <br/>
       Çarşaflıya sadece “Senin bu ülkede yaşama hakkını kabul ediyorum” diyor.
       <br/>
       <br/>
       CHP liderliğinin bu tavrı önemsiz değil.
       <br/>
       <br/>
       Bu tavrın altında, “Tek Parti uygulamaları”na yönelik bir eliştiri bulunduğu da, tartışmalar arasında ortaya çıkıyor.
       <br/>
       <br/>
       Bu da önemli. Bu da, CHP liderliğinin, zihninde, geçmişe yönelik bir özeleştiri birikimi olduğunu gösteriyor.
       <br/>
       <br/>
       Ama henüz köklü bir zihniyet dönüşümü gözlenmiyor, bir.
       <br/>
       <br/>
       Ve iki: CHP’nin 85 yılda oluşturduğu “Sera toplumu”, yeni duruş karşısında şok geçiriyor ve tepki koyuyor.
       <br/>
       <br/>
       - Alışacaksınız. Ezberleri bozacaksınız!
       <br/>
       <br/>
       - Alışmayacağız!
       <br/>
       <br/>
       Liderlik ve “bir tür taban” arasında bu restleşme yaşanıyor.
       <br/>
       <br/>
       Bu noktaya gelişte, CHP’ye yön veren bir “devlet aklı” mı var, yoksa halk terbiye mi ediyor.
       <br/>
       <br/>
       Ben ikincisini düşünüyorum.
       <br/>
       <br/>
       Türkiye’de, mono-blok bir “devlet aklı” var mı, diye sorulduğunda da buna şüphe ile yaklaşıyorum.
       <br/>
       <br/>
       Ama halk da terbiye eder.
       <br/>
       <br/>
       Hele çok daha siyasi-demokratik bilinç yüklenmiş, kendi değerlerine güvenen bir halk, çok daha terbiye edici olur.
       <br/>
       <br/>
       CHP’de olan, çok geç kalmış bir toplumsal uyum adımıdır?
       <br/>
       <br/>
       Bunun, CHP bünyesi tahammül edebilirse, daha köklü ve daha sür’atli olması Türkiye için kazanç olacaktır.
       <br/>
       <br/>
