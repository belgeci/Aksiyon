   <font class="content">
    Filistin’de yaşamak nasıl bir şey, yeterince biliyor muyuz?
    <br>
     <br>
      Sanmıyorum.
      <br/>
      <br/>
      Bilgi olarak bile bilmiyoruz, bir de yaşamak tarafı var oysa... Şartların bedeninize dokunması hadisesi...
      <br/>
      <br/>
      Gazze dramı yaşanırken bir harita dolaştı elden ele.
      <br/>
      <br/>
      1940’lardan bugüne o topraklardaki durumu anlatan bir harita.
      <br/>
      <br/>
      Başlangıçta küçücük Yahudi toplulukları yerleşmiş bu topraklara. Toprak Filistin toprağı...
      <br/>
      <br/>
      Sonra genişleme, genişleme ve bugün: Küçücük Gazze şeridi ve yerleşimcilerle kuşatılmış dağınık topluluklar hâlinde Filistinli barındıran Batı Şeria. Neredeyse Filistin yutulmuş ve İsrail toprağı hâline gelmiş. Bütün dünya, bu toprakların ‘İşgal edilmişliği’ni biliyor, ama toprağın, işgal altında kimlik değiştirmesine kimse ses çıkarmıyor.
      <br/>
      <br/>
      Düşünün ki, koca Filistin toprağından dışarıya, sadece Mısır tarafına açtığınız tünellerle geçebilir hâle gelmişsiniz.
      <br/>
      <br/>
      Tabii, bir de bu kuşatılmışlık içinde yaşanan hayat var. Hayat demeye bin şahit gerektiren.
      <br/>
      <br/>
      Türkiye ki, Filistin’i en çok bilen ülke sayılır.
      <br/>
      <br/>
      Şimdi aşağıda,  ‘Counterpunch’ adlı sitede ‘Life Under Prohibition in Palestine - Filistin’de yasaklar altında hayat’ başlıklı yazıdan notlar aktarmak istiyorum. Bakın bakalım, Filistin’le ilgili bu kahredici gerçeği, bu kuşatılmışlığı, bu savaş hâlinden önce ve sonra da yaşanan yokedici süreci yeterince biliyor musunuz? Biliyor muydunuz?
      <br/>
      <br/>
      Sabit yasaklar
      <br/>
      <br/>
      - Gazze Şeridi’ndeki Filistinlilerin Batı Şeria’da kalması yasak.
      <br/>
      <br/>
      - İsrail işgali altındaki Doğu Kudüs ve eski şehirde bulunan Haremmüşerif’in giriş-çıkışı İsrail askerlerinin kontrolü altında. İkinci İntifada’dan sonra 30 yaşın altındakilerin burada ibadet etmesi yasak. Bu yaş sınırı zaman zaman 40’a kadar ulaşır.
      <br/>
      <br/>
      - Batı Şerialı Filistinlilerin Erez sınır kapısından Gazze Şeridi’ne girmesi yasak.
      <br/>
      <br/>
      - Filistinlilerin köyler, araziler, kasabalar ve mahalleler boyunca çitler ve Yeşil Bölge’yi ayıran “bağlantı hattı”na girmeleri yasak (Batı Şeria’nın yaklaşık yüzde 10’una tekabül ediyor).
      <br/>
      <br/>
      - Filistinlilerin yerleşimcilerin bölgesine girişi (Toprakları yerleşimcilerin bölgesinde olsa bile) yasak.
      <br/>
      <br/>
      - Filistinlilerin Nablus’a bir araç ile girmeleri yasak (Batı Şeria’da Filistin Otoritesinin idaresinde olan bir şehir).
      <br/>
      <br/>
      - Kudüs’te mukim Filistinlilerin A bölgesine girmesi (Batı Şeria’daki Filistin kasabaları) yasak.
      <br/>
      <br/>
      - 16 yaşından küçük çocukların orijinal bir doğum sertifikası ve aile refakati olmaksızın Nablus’tan ayrılması yasak.
      <br/>
      <br/>
      - İzinli Filistinlilerin İsraillilerin ve turistlerin kullandığı kapıları kullanarak İsrail’e girmesi yasak.
      <br/>
      <br/>
      Dönem dönem uygulanan yasaklamalar
      <br/>
      <br/>
      - Batı Şeria’nın bazı kısımlarının sakinlerinin Batı Şeria’nın diğer kısmına seyahatleri yasak.
      <br/>
      <br/>
      - Bazı yaş gruplarındaki insanların, genel olarak 16’dan 30, 35 veya 40’a kadar olan erkeklerin, ikamet ettikleri bölgeleri terk etmeleri yasak. (Genellikle Batı Şeria’daki Nablus ve diğer şehirler…)
      <br/>
      <br/>
      Kontrol noktaları ve bariyerler
      <br/>
      <br/>
      - Şubat 2008 itibariyle Batı Şeria’da 580 kontrol noktası vardı.
      <br/>
      <br/>
      - Orada yollar ile köyler arasına yerleştirilmiş beton küpler, toprak siperler dâhil 446 engel, 88 demir kapı ve anayollar boyunca 74 kilometre çit var.
      <br/>
      <br/>
      - Sahiplerini topraklarından ayıran ayırma çiti boyunca 83 kapı var. Bu kapıların ancak 25 tanesi bazen açılıyor.
      <br/>
      <br/>
      - Musevi olmayan İsrail vatandaşları İsrail’de hiçbir şekilde toprak satın alamıyorlar
      <br/>
      <br/>
      - İsrail’de, Musevileri Musevi olmayanlardan ayırt edebilmek için Filistinlilerin araba plakaları farklı renklerdedir.
      <br/>
      <br/>
      - Batı ve Doğu, tüm Kudüs, ABD de dâhil olmak üzere tüm dünya toplumlarınca İsrail topraklarının bir parçası değil de, işgal edilmiş topraklar olarak kabul edilmiştir.
      <br/>
      <br/>
      - İsrail, su kaynaklarının % 85’ini Musevilere tahsis ederken kalan % 15’lik kaynak ise bölgede yaşayan tüm Filistinlilere veriliyor.
      <br/>
      <br/>
      - ABD, yardım adı altında her yıl İsrail’i 5 milyar dolar ile ödüllendiriyor.
      <br/>
      <br/>
      - İsrail’e yapılan yıllık Amerikan yardımı ABD’nin tüm Afrika kıtasındaki ülkelere vermeyi kabul ettiği miktarı aşıyor.
      <br/>
      <br/>
      - İsrail, Ortadoğu’da nükleer silahlara sahip olan tek ülkedir.
      <br/>
      <br/>
      - Ortadoğu’da nükleer silahsızlanma anlaşmasını reddeden tek ülke İsrail’dir ve İsrail uluslararası uzmanların bölgede yapmak istediği denetimi engellemektedir.
      <br/>
      <br/>
      - İsrail yıllardır siyasi düşmanlarını öldürmek amacıyla diğer ülkelere suikastçılar göndermektedir.
      <br/>
      <br/>
      - İsrail, düzenli olarak Filistinlilerin banka hesaplarına, işlerine ve topraklarına el koyabiliyor.
      <br/>
      <br/>
      - İsrailliler 1988’e kadar iş ilanlarını “Sadece Museviler” diyerek veriyorlardı.
      <br/>
      <br/>
      - Bugünün İsrail’i, şimdi yok edilmiş olan 400’den fazla eski Filistin köylerinin üzerine kurulmuştur ve eski izleri yok etmek için İsrailliler her bölgeyi yeniden isimlendirmişlerdir.
      <br/>
      <br/>
      - 1948 savaşı sonrası başlayan göçler neticesinde bugün 5 milyona yakın Filistinli evlerinden uzakta vatansız duruma düşürülmüşlerdir.
      <br/>
      <br/>
      Tecrid duvarı gerçekleri:
      <br/>
      <br/>
      - İsrail işgal devletinin Haziran 2002 tarihinde Batı Şeria’da başlattığı tecrit duvarı Çin Seddi’nden sonra dünyanın en uzun duvarı.
      <br/>
      <br/>
      - Duvarla Yeşil Hat arasında bulunan bölgeler “kapalı askerî bölge” olarak ilan edilmiştir. Filistin tarafındaki diğer yasak bölgelere de ancak birtakım kontrol noktalarından geçilerek girilebilmekte.
      <br/>
      <br/>
      - İsrail’in iddiasının aksine duvarın sadece % 11’i BM kararlarında İsrail sınırı olarak gösterilen Yeşil Hat’tan geçmekte % 89’luk kısmı ise BM kararlarında Filistin olarak gösterilen Batı Şeria topraklarının içinde inşa edilmekte. Yani duvar ile Batı Şeria’nın yüzde 15’ini daha gasp etmekte. Ayrıca Yeşil Hat ile duvar arasında kalan 300 bin Filistinli izole edilmektedir.
      <br/>
      <br/>
      - İşgal devleti duvarın inşaatı için 165 bin dönüm araziyi istimlak etti. Bu istimlakın bir kısmı askerî kararla bir kısmı da doğrudan el konulmasıyla gerçekleşti.
      <br/>
      <br/>
      - Duvar insanların evleriyle arsaları arasında engel koyuyor. Bu şekilde 71 köyün arazisi duvarın ötesinde kalıyor. Bu, 400 bin kişinin gündelik olarak duvarın ötesine geçmesi anlamına geliyor. Bu ise, bölge halkının % 30’u demek.
      <br/>
      <br/>
      - Duvarın sebep olduğu zarardan öğrenciler de payını alıyor. Duvar 170 bin öğrencinin yarısını okulsuz bırakıyor. Zira 320 okul duvarın öbür yakasında kalıyor.
      <br/>
      <br/>
      Şimdi bir de Filistin’i, Gazze’ye atılan binlerce tonluk bombalardan sonra düşünün.
      <br/>
      <br/>
      Çoğu çocuk ve kadınlardan oluşan 1300 Filistinlinin daha katledildiği bir Filistin’i anlamaya çalışın.
      <br/>
      <br/>
      Niye isyan ediyor Tayyip Erdoğan?
      <br/>
      <br/>
      Bunu ancak, içinde azıcık insanlık kalanlar anlayabilir?
      <br/>
      <br/>
      İsrail gözlüğü takanlara bir şey anlatmak zor.
      <br/>
