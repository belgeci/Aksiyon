   <font class="content">
    Kaç zaman var ki Türkiye, “siyasette, devlet yönetiminde temiz kalma”yı gündemden düşüremiyor.
    <br>
     <br>
      1995 seçimleri temiz kalma tartışması ile geçmişti. 28 Şubat sonrası kurulan hükûmetler, Türkiye gündemine “Hortumlama” klişesini bıraktılar. 2002 seçimlerinde AK Parti “3 Y İle Mücadele” sloganını bayraklaştırdı. “3 Y”den birisi “Yolsuzluk” idi, diğerleri de Yasaklar ve Yoksulluk...
      <br>
       <br/>
       Yıl, 2009…
       <br/>
       <br/>
       Türkiye, mahalli seçimlere gidiyor ve gündemin en sıcak tartışması yine yolsuzluklar üzerine...
       <br/>
       <br/>
       Devlet özelleştirmelerle küçüldü, şimdilerde ana rant alanı şehirler oldu. Şehirlerin, büyük şehirlerin 100 milyar dolarlarla ifade edilen büyük yatırımları var, şehirlerin imar planları büyük rantlar üretiyor, her kalem oynamasında, her cetvel kaymasında, kimileri için büyük rant doğuyor, kimisinin canına okunuyor.
       <br/>
       <br/>
       İşte oralarda bir yerlere giren kötü niyet, dünya hesabı, ya da merhum mimar Turgut Cansever Hoca’nın dediği gibi “şeytanla kol kola yürüme, günah yarışına çıkma” sapması, yolsuzluk belasını, ülke gündeminin başına oturtuyor.
       <br/>
       <br/>
       Binlerce ayak oyunu ile çalan çalana.
       <br/>
       <br/>
       Kimileri yakalıyor, kimileri yakalanmaktan korkuyor.
       <br/>
       <br/>
       Yakaladığını sananların yakınlarındakiler de bir-iki gün sonra iş üzerinde yakalanıyor.
       <br/>
       <br/>
       Anlaşılıyor ki, toplum olarak kirlenme yaşıyoruz.
       <br/>
       <br/>
       Yerel yöneticilik, bizim kadim kültürümüzde “Şehr emaneti” olarak nitelenmiş.
       <br/>
       <br/>
       Başkanın ismi “şehr emini”...
       <br/>
       <br/>
       Şehir ona emanet edilmiş.
       <br/>
       <br/>
       Yolu, suyu, temizliği, ağacı, kuşu ile...
       <br/>
       <br/>
       Fauna ve flora deniyor, şehrin kuşuna ve ağacına...
       <br/>
       <br/>
       Başkan, değil insanların, onların bile varlığına itina edecek. Onları canı gibi koruyacak. Çünkü şehrin de bir canı olduğunu unutmayacak. Şehrin canına okursanız, gün gün öldüğünü, can verdiğini görecek.
       <br/>
       <br/>
       Şehri talan etmeyecek, ettirmeyecek.
       <br/>
       <br/>
       Babasının oğluna bile...
       <br/>
       <br/>
       Şehrin havasını, suyunu, yeşilini çaldırmayacak.
       <br/>
       <br/>
       Gelirken ellerine bakacak, temiz mi diye...
       <br/>
       <br/>
       Giderken iki kere bakacak, temiz kalmış mı diye.
       <br/>
       <br/>
       Bu hassasiyet bize, inanç değerlerimizin asırlar içinde telkin ettiği şey.
       <br/>
       <br/>
       Bu hassasiyetin damarlarımıza, iliklerimize işlemiş olması lazım.
       <br/>
       <br/>
       -Arkadaş, yaptığın her işin hesabını vereceksin.
       <br/>
       <br/>
       -Bugün, yarın mutlaka...
       <br/>
       <br/>
       -İnsanlar sormazsa, bir başka yerde hesap sorulacak.
       <br/>
       <br/>
       -Tek bir kişi olsan, amel defterin kendi küçük hayatınla sınırlı olacak.
       <br/>
       <br/>
       -Bir müessesenin, bir şehrin, bir ülkenin yöneticiliği, binlerce, on binlerce, yüz binlerce, milyonlarca insanla, hatta gökle, yerle, kuşla, ağaçla bir hukuk ilişkisine girmek anlamına geliyor.
       <br/>
       <br/>
       Bir şehrin yöneticiliği, insanın kendi amel defterini yazmak gibi bir şey.
       <br/>
       <br/>
       -Söylediğiniz her şey, aleyhinize delil olabilir, denir ya hani...
       <br/>
       <br/>
       Aynen onun gibi, söylediğiniz ve yaptığınız her şey yazılıyor. Yarın önünüze çıkacak. Korkun. Ama hizmet etmekten değil, hizmet ederken içine, hesabını veremeyeceğiniz bir kir bulaşmasından korkun.
       <br/>
       <br/>
       Hazreti Ali’nin Malik bin Haris el Eşter’e gönderdiği Emirname çok ünlüdür.
       <br/>
       <br/>
       Orada diyor ki Raşit halifelerin dördüncüsü büyük sahabi:
       <br/>
       <br/>
       “Nefsin hakkında, sana yakınlığı olanlar hakkında, yönettiklerin içinde kendilerine meyil beslediklerin hakkında Allah’a ve Allah’ın kullarına karşı adaletten kat’iyyen ayrılma.
       <br/>
       <br/>
       Şayet böyle yapmazsan, zulmetmiş olursun.
       <br/>
       <br/>
       Hâlbuki, Allah’ın kullarına zulmedenin, ibadullah tarafından davacısı Allah’tır. Allah da bir kimsenin hasmı oldu mu, o kimsenin tutunabileceği bütün huccetler batıldır. Kudretullah karşısında tesiri yoktur.  Kul, zulmüne tevbe edinceye kadar husumeti ilahiyye devam eder. Zulm üzere bulunmak kadar Allah’ın nimetini gideren, gazabını çabuklaştıran hiçbir şey yoktur.
       <br/>
       <br/>
       Zira Cenabı Allah, zulüm altında inleyenin iniltisini işitmekte, zalimleri görmektedir.”
       <br/>
       <br/>
       İşte, işin gerçeği budur.
       <br/>
       <br/>
       Hazreti Ali 14 asır önceden bu yana seslenmektedir.
       <br/>
       <br/>
       Hazreti Ömer, devlet mumu ile kendi mumu arasındaki farkı vurgulayarak, devlet adamının kul hakkı üzerindeki hassasiyetini 14 asırdan beri haykırmaktadır.
       <br/>
       <br/>
       Hepsinden önce Allah Rasulü, sallallahu aleyhi ve sellem, “Birinize, kendi meramını iyi anlattığı için hakkı olmayan bir şeyi verirsem, ona ateşten bir parça vermiş olurum” diyerek, gelmiş geçmiş en büyük uyarıyı yapmaktadır.
       <br/>
       <br/>
       Bir ateş parçasını avuçlamayı kim ister?
       <br/>
       <br/>
       Ama odur işte, haksız rantlarla aldığımız, cebimize indirdiğimiz, çocuklarımıza ikram ettiğimiz, hatta kendi boğazımıza yuttuğumuz...
       <br/>
       <br/>
       Bir ateş parçası...
       <br/>
       <br/>
       Rasulullah, 14 asırdan bu yana, insanların, üzerlerine aldıkları sorumluluktan artı rantlar edinmesine meşruiyyet kapısı açan, o “Bal tutan parmağını yalar” mantığını mahkûm etmektedir.
       <br/>
       <br/>
       Zekât toplamakla görevlendirdiği bir kişi, döndüğünde, Rasulullah’ın önüne, yanında getirdiklerini koyarken, “Şu Beytülmalin şu benim, şu Beytülmalin şu benim...” diye ayrı ayrı iki öbek oluşturur da Rasulullah onu izledikten sonra üzülür ve şu tepkiyi verir:
       <br/>
       <br/>
       -Sen, zekât toplamakla görevlendirilmeseydin, “Benim” diye şuraya yığdığın şeyler yine de sana gelecek miydi?
       <br/>
       <br/>
       Buradan alınan ışık şu idi ki, devlet görevi, hiç kimse için artı rantlar üretmeyecekti.
       <br/>
       <br/>
       Şimdilerde siyaset yapan bir ilahiyatçı ilim adamımız bana şöyle söylemişti.
       <br/>
       <br/>
       “-Ben, demişti, dünyada yaptığım her işin yarın Allah huzurunda savunulabilir olmasına itina ederim.”
       <br/>
       <br/>
       Müthiş bir hassasiyet bu.
       <br/>
       <br/>
       Ama keşke her an aklımızda bulunabilse... Her an, her adımda, her sözde, her işte...
       <br/>
       <br/>
       Peki, hani nerede?
       <br/>
       <br/>
       Biz niye, bir Müslüman ülkenin çocukları, şehirlerimizi idare edecek, bize hizmet edecek insanları seçerken yolsuzlukları konuşuyoruz?
       <br/>
       <br/>
       Paçalarımızdan çamurlar yukarıya doğru yükseldiği için mi?
       <br/>
       <br/>
       Ellerimize baktığımızda, onları çok da arı duru görmediğimiz için mi?
       <br/>
       <br/>
       Temiz kalmak diyorum, onun için, her şeye rağmen temiz kalmak.
       <br/>
       <br/>
       Etrafımızı kanalizasyon atıkları götürse bile, temiz kalmak...
       <br/>
       <br/>
       Temiz kadrolarla çalışmak.
       <br/>
       <br/>
       Çocuklarımıza temiz, haramdan arınmış yiyecekler sunmak.
       <br/>
       <br/>
       El ele tutuşup yarınlara gittiğimiz insanların ellerinin temiz olmasına itina etmek.
       <br/>
       <br/>
       Hazreti Ömer, bir gün, üzerindeki devlet yükünün altında bunalıyor ve şöyle yakınıyor:
       <br/>
       <br/>
       -Ömer, Ömer, neden aldın bu barı sırtına sen!
       <br/>
       <br/>
       “Bar” yük demek.
       <br/>
       <br/>
       Hizmet yükünü taşımak kolay değil. Ama çok sevaplı bir iş.
       <br/>
       <br/>
       Ve ama, temiz kalmak şartıyla...
       <br/>
       <br/>
       Yunus Emre, “Seni sigaya çeken bir Molla Kasım gelir” diyor.
       <br/>
       <br/>
       Evet, her insanın hesaba çekileceği bir gün vardır.
       <br/>
       <br/>
       Bugün, yarın...
       <br/>
       <br/>
       Bugünkü hesaplara da hazır olmak lazım, yarınlardakine de...
       <br/>
       <br/>
       Bir Allah dostu şöyle demişti:
       <br/>
       <br/>
       -Ne mutlu yüz akı ile ahirete gidebilene...
       <br/>
       <br/>
       Söz bu. Yüz akı ile son noktayı koyabilmek...
       <br/>
       <br/>
       Ne denir? Meydana, yüz akı ile ebedi âleme göç etme iradesini yürek dokusu hâline getiren çıksın...
       <br/>
      </br>
