# Sanayi Globalizme Hazır Değil

*Birol Uzunay Bilinmeyen-tarih*

 <font class="content">
  Gazete ilanıyla iş başvurusu yaptığı şirketi satın
  <br>
   alan Dr. Murat Akdoğan, geliştirdiği pazarlama ve
   <br>
    satış stratejileriyle son dönemin en hızlı çıkış
    <br>
     yapan işadamlarından. Akdoğan, üniversitenin
     <br/>
     teorik yanını iş dünyasının pratikliğiyle
     <br/>
     birleştiriyor.
    </br>
   </br>
  </br>
 </font>
 <br/>
 <p>
  <font class="content">
   Dr. Murat Akdoğan, "sıfırdan zirveye" başarı
   <br>
    hikayeleri için prototip bir örnek... Bir dönem gazete
    <br/>
    ilanıyla iş başvurusu yaptığı Baymak Grubu'nu
    <br/>
    satın alan Murat Akdoğan, öğretmen çocuğu bir
    <br/>
    işçinin sadece "idealleriyle" patron olabileceğini
    <br/>
    gösterdi.
    <br/>
    <br/>
    <br/>
    Bilgi çağında para kazanmanın
    <br>
     çalışmakla ya da kurnazlıkla değil de "yeni bilgiyle"
     <br>
      gerçekleşeceğine inanan Murat Akdoğan, gerek ders
      <br/>
      verdiği Marmara Üniversitesi'nde gerekse sahibi
      <br/>
      olduğu Baymak Grup'ta yeni satış ve üretim teorileri
      <br/>
      üzerinde çalışıyor. Üniversitede derste anlattığı
      <br/>
      teorik bilgileri iş hayatında uygulayarak ilginç bir
      <br/>
      model geliştiren Murat Akdoğan, sanayi çağını
      <br/>
      kaçıran Türkiye'nin ancak özgün satış modelleriyle
      <br/>
      ekonomisini kurtarabileceğine inanıyor.
      <br/>
      <br/>
      26 yaşında genel müdür ve bıyık...
      <br/>
      <br/>
      Murat Akdoğan 24 yaşındaki bir mühendis adayı
      <br/>
      olarak Baymak Gruba yaptığı iş başvurusunda
      <br/>
      "Benim bu şirketi olağanüstü büyütecek modellerim
      <br/>
      var" yazıyordu. "Olağanüstü modelleri" merak eden
      <br/>
      Baymak yöneticileri, idealist gençle tanışınca
      <br/>
      hemen işe aldılar ve kısa bir süre sonra iş
      <br/>
      başvurusunda yazılanların hiç de yalan olmadığı
      <br/>
      ortaya çıktı. Sadece iki yıl sonra iş başvurusunda
      <br/>
      önerdiği çalışma modellerini uygulaması için
      <br/>
      Baymak Yönetim Kurulu, Murat Akdoğan'a genel
      <br/>
      müdürlük teklifinde bulundu. 26 yaşında bir gencin,
      <br/>
      hiçbir akrabalık bağı olmamasına rağmen
      <br/>
      uluslararası bir şirkette müdür olması hiç de kolay
      <br/>
      değildi.
      <br/>
      <br/>
      <br/>
      Tecrübenin yaşla alakasının olmadığını
      <br>
       ilk genel müdürlük deneyiminde anladığını
       <br>
        söyleyen Murat Akdoğan "Çok gençtim ve
        <br/>
        omuzlarımda milyonlarca dolarlık bir sermaye
        <br/>
        vardı. Başlangıçta sözümü dinletmek için bıyık
        <br/>
        bile bıraktım ancak imajdan çok yeni bir sisteme
        <br/>
        ihtiyaç vardı. Ben sistem geliştirdikçe saygı
        <br/>
        görmeye başladım. Hız ve bilgi kalitesinin bu kadar
        <br/>
        önemli olduğu bir çağda, hâlâ tecrübenin yaşla
        <br/>
        alakalı olduğunu söylemek bence abes" diyor.
        <br/>
        <br/>
        <br/>
        <br>
         90 metrekarelik dev sanayi tesisi!
         <br>
          <br/>
          Kalorifer kazanı, güneş enerji sistemleri, soba,
          <br/>
          şofben gibi eşyalarda satış rekorları kıran Baymak
          <br/>
          1991'de Körfez Kriziyle beklemediği bir ekonomik
          <br/>
          darbe aldı. Petrol fiyatlarının yükselmesi ve
          <br/>
          tüketimin durmasıyla zirvedeki Baymak bir anda
          <br/>
          dibe çakıldı ve şirket bankaların eline geçti.
          <br/>
          Borçların ödenemeyeceğini anlayan Baymak
          <br/>
          Yönetim Kurulu konkordato ilan etse de "Benim bu
          <br/>
          şirketi büyütecek olağanüstü modellerim var" diyen
          <br/>
          Murat Akdoğan için herşey bitmemişti. 14 bin
          <br/>
          metrekarelik Bakmak'ın karşısına 90 metrekarelik
          <br/>
          yeni bir işyeri açan Murat Akdoğan, 5 işçiyle
          <br/>
          yeniden dirilişi başlatır.
          <br/>
          <br/>
          <br/>
          5 işçiyle ilk olarak 10 tane termosifon
          <br>
           yapıp satışa sunan genç müdür, gemiyi terketme
           <br>
            niyetinde değildir. Kimsenin Baymak'ın geleceğine
            <br/>
            inanmadığı bir anda gündüzlerini işçilerle,
            <br/>
            gecelerini de pazarlama kitaplarıyla geçiren Murat
            <br/>
            Akdoğan bu dönemde kriz, pazarlama ve strateji
            <br/>
            yönetimi üzerinde yoğunlaşır.
            <br/>
            <br/>
            "Rekabet bilinmiyor, yanlış satış stratejisi
            <br/>
            uygulanıyor"
            <br/>
            <br/>
            Türkiye'deki şirketlerin rekabeti bilmediğini fark
            <br/>
            eden Murat Akdoğan, ürün, fiyat ve teknolojiyle
            <br/>
            rekabet etmenin çokta anlamlı olmadığını
            <br/>
            belirterek "Asıl rekabet iş yapma modellerindedir.
            <br/>
            Tüketici anlayışı değişti. Bugünün tüketicisini
            <br/>
            tanımak gerekiyor. Ben geceleri bile servis
            <br/>
            yaptırdım çünkü bugünün aile yapısında kadın
            <br/>
            genellikle çalışıyor. Geleneksel söylemi sürdüren
            <br/>
            şirketleri yeni pazarlama ve tüketici modelleriyle bir
            <br/>
            anda geçmeye başladık" diyor.
            <br/>
            <br/>
            <br/>
            Türkiye iş dünyasının en büyük
            <br>
             eksikliğinin yönetim ve organizasyonda olduğunu
             <br>
              söyleyen Murat Akdoğan, "Şirketlerimizde
              <br/>
              motivasyon var, plan var, hatta para da var ancak
              <br/>
              koordinasyon ve denetleme yok. Ben kriz günlerinde
              <br/>
              yeni bir modelle uğraşırken, yeni Baymak'ın küçük
              <br/>
              olması ve tüm kontrolü elime almamla farklı olmayı
              <br/>
              başardım ve organize ve denetlemeye büyük önem
              <br/>
              verdim" diyor.
              <br/>
              <br/>
              <br/>
              Evlerde kullanılan teknolojik aletlerin
              <br>
               belirli bir kaliteyi aşmış olmasına rağmen birbirine
               <br>
                çok benzediğini söyleyen Murat Akdoğan asıl
                <br/>
                farklılığın pazarlama sistemlerinde ortaya
                <br/>
                çıktığını belirtiyor: "Doktoramı pazarlama üzerine
                <br/>
                yapmam ve iş dünyasındaki modelleri akademik bir
                <br/>
                çerçeveye oturtabilmek bana büyük avantaj
                <br/>
                kazandırdı. Pazarlama merkezli bir şirket olduk ve
                <br/>
                uzmanlık alanları belirledik. Rakiplerimizle benzer
                <br/>
                teknolojideki ürünleri satmamıza rağmen,
                <br/>
                müşterilerimiz bizden çok memnundular; çünkü
                <br/>
                onların ne istediğini biliyorduk" diyor.
                <br/>
                <br/>
                Çabanın ödülü: Baymak ortaklığı
                <br/>
                <br/>
                1991'de bankaların eline geçen Baymak, Murat
                <br/>
                Akdoğan'ın atölyeden başladığı çabaları ve
                <br/>
                geliştirdiği pazarlama modelleriyle 1995'te bütün
                <br/>
                borçlarını ödeyip, bankalara borç verir hale
                <br/>
                gelmişti. Baymak ürünlerini geliştirmede ve
                <br/>
                pazarlamada geliştirdiği modellerle haklı bir
                <br/>
                başarıyı elde eden Murat Akdoğan'a hediye olarak
                <br/>
                Baymak ortaklığı verildi.
                <br/>
                <br/>
                <br/>
                İş dünyasında genç yaşlarda başarılı
                <br>
                 olmanın çok zor olduğunu söyleyen Murat Akdoğan
                 <br>
                  başarının formülünü ise ilginç biçimde açıklıyor;
                  <br/>
                  "Herkesin yaptığı işi yapmamak gerekiyor, çünkü
                  <br/>
                  çalışan değil farklı çalışan kazanıyor. Bana
                  <br/>
                  'sıfırdan bugüne gelmeyi hayal ediyor muydun?'
                  <br/>
                  diyorlar. Ben hayal etmiyordum; çünkü başarılı
                  <br/>
                  olacağımı biliyordum. Farklı olma formülünü
                  <br/>
                  bulmuştum, bu yüzden de kendime güveniyordum."
                  <br/>
                  <br/>
                  <br/>
                  Türkiye'deki işletmelerin Gümrük Birliğini
                  <br>
                   ve globalleşmeyi de yanlış anladığını söyleyen
                   <br>
                    Murat Akdoğan, "Alt yapıyı oluşturmadan,
                    <br/>
                    araştırma yapmadan herşey isteniyor. İşte Gümrük
                    <br/>
                    Birliği'nin sonuçları ortada. Şirketlerimiz globalizmi
                    <br/>
                    iyi araştırmalı, üretim ve satışı belirli ilkelere göre
                    <br/>
                    planlamalılar" diyor.
                    <br/>
                    <br/>
                    <br/>
                    "Yeni birşeyler yapmak zamanı" ilkesiyle
                    <br>
                     yabancı ortaklık modelini seçen Murat Akdoğan,
                     <br>
                      Baxi Grubu'yla ortaklık yapıyor. Özellikle kombi,
                      <br/>
                      şofben, radyatör, kat kaloriferleri, vanalar ve
                      <br/>
                      genleşme tanklarında iddialı olan Baymak
                      <br/>
                      Grubu'nun binin üzerinde bayii ve 450 yetkili servisi
                      <br/>
                      bulunuyor.
                      <br/>
                      <br/>
                      <br/>
                      "Benden neden bir başbakan olmasın"
                      <br>
                       <br>
                        <br/>
                        <br/>
                        Türkiye Genç İşadamları Derneği ve
                        <br>
                         Pazarlama Derneği Yönetim Kurulu üyesi olan,
                         <br>
                          Marmara Üniversitesi İşletme Fakültesi'nde dersler
                          <br/>
                          veren Murat Akdoğan'ın yeni hedefi ise oldukça
                          <br/>
                          iddialı... 30 yaşında sıfırdan zirveye çıkan Murat
                          <br/>
                          Akdoğan 40'lı yaşlarında da başbakanlığı
                          <br/>
                          hedefliyor. Bu hedefin her vatandaşın hakkı
                          <br/>
                          olduğunu söyleyen Akdoğan'ın başbakanlık isteği
                          <br/>
                          ise oldukça makul." Ben memur bir babanın çocuğu
                          <br/>
                          olarak hep çalıştım, hep birşeyler ürettim.
                          <br/>
                          İşletmemin başarısı ortada. Bireysel olarak da en
                          <br/>
                          genç akademisyenlerden biri oldum. Belli bir çıtayı
                          <br/>
                          aştığıma inanıyorsam, ülkeyi de kurtaracağıma
                          <br/>
                          inanmam normal değil mi? Başbakanlık yapanlar
                          <br/>
                          neyi başardı da başbakan oldular? Herkes başbakan
                          <br/>
                          olmayı hedefleyebilmeli. İşte o zaman bu ülkede
                          <br/>
                          belirli güzelliklere ulaşılabilir."
                         </br>
