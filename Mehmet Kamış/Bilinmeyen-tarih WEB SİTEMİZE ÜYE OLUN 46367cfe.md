# WEB SİTEMİZE ÜYE OLUN

*Mehmet Kamış Bilinmeyen-tarih*

<div>
 <p>
  <font>
   Sıra geldi internet sayfalarına. Uzun süredir en çok eleştiri aldığımız konuydu bu. Dünyanın dört bir yanında bulunan Aksiyon okuyucuları her fırsatta sitemlerini dile getiriyorlardı. Derginin yeni tarzı bizi çok uğraştırdığı için buna ancak fırsat bulabildiğimizi itiraf etmek zorundayım.
   <br/>
   <br/>
   Yasin Kesen uzun bir uğraştan sonra bizim de çok hoşumuza giden bir tasarım yaptı. Okuyucularımızın çok daha aktif olarak katılacağı web sitemizle birlikte, internet yayıncılığının sunduğu birçok yenilik de hayata geçirilmiş oldu.
   <br/>
   <br/>
   Okuyucularımızı öncelikle sitemize üye yapmak istiyoruz. Sitemize üye olan okurlarımız, okudukları haberle ilgili yorumlarını diledikleri anda aktarabilecekler, ilginç buldukları haberin adresini bir–iki tıklama ile bir başka kişiye gönderebilmeleri mümkün olacak.
   <br/>
   <br/>
   İnternet sitemizle ilgili bize gelen en büyük eleştiri ve istek, arama motorlarının olmamasıydı. Artık yeni düzen arşivde, isteyenlerin değişik kriterlere göre arama yapmasına imkan sağlanıyor.
   <br/>
   <br/>
   Yine, üyelik sistemi kapsamında okurlarımızın tartışma forumlarında görüşlerini paylaşmaları mümkün hale geldi. Editörlerimizin yönetimindeki forumlar vasıtası ile yine katılımcı bir okur birlikteliği amaçlanıyor. Aynı sistem içerisinde okurların birbirleriyle mesajlaşmaları, haberlere yorum yazan kişilerle görüş alışverişinde bulunmaları mümkün hale getirildi. İnternet rehberi kısmında ise daha önce Sanalpark sayfasında yer alan web sitesi tanıtımları yer alacak. Bu bölüme üyelerin link eklemeleri, web siteleriyle ilgili açıklama yazmaları sağlandı.
   <br/>
   <br/>
   Gelişim süreci devam edecek olan web sitemizde artık haftada sadece bir kez değil her gün, belki her dakika yeni şeyler bulabileceksiniz. Üyelerimizin, aksiyon.com.tr’yi farklı arayüzlerde gezmeleri, ilgilendikleri haber konularını ön plana çıkarmaları gibi seçenekler de zamanla gerçekleşecek.
   <br/>
   <br/>
   Üye kayıt aşamasında talep edilen kişisel bilgilerden çıkarılacak istatistikler ile hem dergiye, hem de web sitemize alınacak reklamlarda, reklam veren için bir kriter ortaya konacak. Bu açıdan da önemli gördüğümüz tamamen ücretsiz olan üyelik sistemine tüm okurlarımızın katılmasını bekliyoruz.
   <br/>
   <br/>
   İyi haftalar dileğiyle...
   <br/>
   <br/>
  </font>
 </p>
