# HEPSİ GERÇEK (393)

*Osman İridağ Bilinmeyen-tarih*

<div>
 <p>
  <font>
   Delilik işte
   <br/>
   Çanakkale Deliler Derneği “Dünya Deliler Günü” ilan ettikleri Haziran ayının ikinci pazar günü biraraya gelerek 4. olağan kongresini gerçekleştirdi. Kongreye davet ettikleri Kemal Derviş’in katılmaması üzerine “Sayın Derviş’i birkaç yıl içinde delirttiler, inşaallah gelecek yıl gelirler” açıklamasını yaptılar.
   <br/>
   <br/>
   Lütfen vasi tayin edin
   <br/>
   <br/>
   Nevşehirli işçi emeklisi Ali Altunbaş 65 yaşı geçen herkesin hastaneden sıhhi kurul raporu almadan kendi mülkünü bile satamadığını iddia ederek, yaptığı gaflar, unutkanlık ve hastalığı nedeniyle Başbakan Bülent Ecevit’e vasi tayin edilmesi amacıyla Nevşehir Sulh Hukuk Hakimliği kanalıyla Ankara Nöbetçi Sulh Hukuk Hakimliği’ne başvurdu.
   <br/>
   <br/>
   Komünist Ecevit
   <br/>
   <br/>
   Başbakan Bülent Ecevit’in tedavisini üstlenen Prof. Dr. Turgut Zileli geçmiş yıllarda şimdi hastası olan Ecevit’e komönüst dermiş.
   <br/>
   <br/>
   Avşar bayrağa takıldı
   <br/>
   <br/>
   Ünlü sanatçı Hülya Avşar hakkında televizyon programında Türk bayrağı desenli balonlara tekme attığı için Türk bayrağına muhalefet ettiği iddiasıyla soruşturma başlatıldı.
   <br/>
   <br/>
   At olmadı eşek verelim
   <br/>
   <br/>
   Gaziantep’te temeli 1997 yılında atılan hipodrom tamamlanmayınca çevre köylerde bulunan çobanlar bu alanı boş bırakmamak için “eşek yarışı” düzenlemeye başlamışlar.
   <br/>
   <br/>
   7’lik Shakira
   <br/>
   <br/>
   Konya’da yapılan Hakemler Gecesi’nde 7 yaşındaki bir kız çocuğu Shakira gibi giydirilerek dans ettirildi.
   <br/>
   <br/>
   İşsizlik kazandırdı
   <br/>
   <br/>
   İzmir’de her yıl gerçekleştirilen çığlık yarışmasını Dokuz Eylül Üniversitesi Muhasebe-Finansman Bölümü’nden mezun olan, ancak hâlâ iş bulamayan ve işsizliği protesto ederek çığlık atan İlyas Söğüt adlı genç kazandı.
   <br/>
   <br/>
  </font>
 </p>
