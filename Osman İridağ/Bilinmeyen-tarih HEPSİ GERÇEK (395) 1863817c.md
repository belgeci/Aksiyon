 <font class="content">
  İzin gerekiyormuş!
  <br/>
  <br/>
  Türkiye–Brezilya maçı öncesi sokaklarının parke taşlarını kırmızı beyaz renklere boyayan İzmirli 70 esnaf, belediyeden izin almadıkları gerekçesiyle karakolluk oldu.
  <br/>
  <br/>
  Bozdur bozdur harca
  <br/>
  <br/>
  Asgari Ücret Tesbit Komisyonu uzun tartışma ve toplantılar sonrası Türk halkı için belirlediği yeni asgari ücreti açıkladı: 184 milyon 251 bin 937 Türk Lirası.
 </font>
 <br/>
 <p>
  <font class="content">
   Hayvan sürücüsü cezası
   <br/>
   <br/>
   Ardahan’ın Çıldır İlçesi’ne bağlı Meryem Köyü’nde otlattığı sığırları karayoluna çıkardığı gerekçesiyle 22 yaşındaki çoban Bülent Kılıç’a, İlçe Jandarma Komutanlığı’na bağlı trafik ekibince 20 milyon 200 bin lira trafik cezası kesildi. “Hayvan Sürücüsü” olarak kesilen cezayı hayvan sahipleri ödemeyince, çoban zor durumda kaldı.
   <br/>
   <br/>
   Bir tek İsviçre
   <br/>
   <br/>
   Elektrikte dünya şampiyonluğunu kılpayı kaçırdık. Üretimde ve tüketimde olmasa bile, Milli Geliri 36 bin dolar olan İsviçre’nin ardından en pahalı elektrik kullanan ikinci ülkeyiz!
   <br/>
   <br/>
   Nuh’un dağı
   <br/>
   <br/>
   Turizm Bakanlığı, içinde müze, toplantı salonu, kütüphane ve restoranın da olacağı Nuh’un gemisinin betondan bir maketini gerçek ölçütlerine uygun olarak Ağrı Dağı’nda inşa etmeye karar verdi.
   <br/>
   <br/>
   Ayı kaçırmış
   <br/>
   <br/>
   Tunceli’nin Ovacık İlçesi’nde  ortadan kaybolduktan 5 gün sonra mezarlık yakınında bulunan Hasan Kemer’in, parçalanmış cesedi üzerinde yapılan otopsi sonucunda bir ayının saldırısına uğrayarak öldüğü belirlendi.
   <br/>
   <br/>
   Aradaki fark
   <br/>
   <br/>
   Avrupa Birliği üyesi ülkelerde 20–30 öğrenciye bir bilgisayar düşerken, Türkiye’de bir bilgisayarı tam 81 öğrenci paylaşmak zorunda kalıyor.
