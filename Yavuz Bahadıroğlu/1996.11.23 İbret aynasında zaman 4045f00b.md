# İbret aynasında zaman

*Yavuz Bahadıroğlu 1996.11.23*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   "Tarih bilmeyen diplomat, pusuladan anlamayan kaptana benzer. Her iki halde de karaya oturmak tehlikesi yüksektir."
Cevdet Paşa
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   Her ne kadar günümüzde "geçmişe mazi" deniyor ve tarihi örneklerden rahatsızlık duyuluyorsa da, tarih gerçek bir ibret aynası ve tam bir "tecrübe tahtası"dır. Devlet adamları ne kadar tarih bilirlerse, tarihin kendini tekrarlama riski o kadar azalır.
   <br/>
   <br/>
   Biliyorsunuz bugünlerde Adalet Bakanlığı'nın tayinleri konuşuluyor. O konuya hiç girmeden, herkesin tarih aynasında kendi olayını görme temennisiyle, günlük tartışmalarımızın tarih aynasına yansımalarına bir bakalım...
   <br/>
   <br/>
   * * *
   <br/>
   <br/>
   Padişahın devlet sayıldığı günlerdeyiz: Yıl 1393...
   <br/>
   <br/>
   Osmanlı tahtında Niğbolu kartalı Yıldırım Bayezid oturuyor...
   <br/>
   <br/>
   Sefer dönüşü bir solukluk uğradığı yerde "Ayak Divanı" (Padişahın doğrudan halkın şikâyetlerini dinlemesi) kurdurup halkın dertlerini dinlerken, yaşlı bir kadın bağıra bağıra Padişahı azarlamaya başlıyor:
   <br/>
   <br/>
   "Padişahım! Yularını gevşek tuttuğun hademelerinden biri, destur dilemeden sütümü içti. Bedelini talep ettiğimde bağırıp çağırdı. İmam efendinin himmeti, ahalinin gayretiyle herifi yakalayıp kadı efendiye götürdüm. Lâkin senin kadı, herifin lehine hükmetti. Mağdur oldum. Hakkımı isterim."
   <br/>
   <br/>
   Hademe aranıp bulunuyor. Getirilip Padişahın huzuruna çıkarılıyor. Padişah bizzat sorguluyor:
   <br/>
   <br/>
   "Böyle iken böyle yaptın mı?"
   <br/>
   <br/>
   Adam boynunu bükmüş yalvarıyor:
   <br/>
   <br/>
   "Affediniz Hünkârım şeytana uydum."
   <br/>
   <br/>
   Suç sabit. Hademe cezalandırılacak ve konu kapanacak. Hayır! Padişahın derdi bu işin içindeki işte: Acaba şahitliispatlı bir suçu kadı neden cezalandırmamış? Yoksa bazı kadıların rüşvet yediği söylentisi doğru mu?
   <br/>
   <br/>
   Hademeye sual;
   <br/>
   <br/>
   "Kadıya rüşvet vererek mi serbest kaldın?"
   <br/>
   <br/>
   Genç hademenin boynu bükük, elleri önüne bağlı:
   <br/>
   <br/>
   "Şevketlüm, billahi rüşvet vermedim, sadece senin maiyetinde bulunduğumu söyledim. O da kabahatımı bağışladı."
   <br/>
   <br/>
   Eyvah ki eyvah!..
   <br/>
   <br/>
   Yıldırım Bayezid yıldırım gibi düşüyor:
   <br/>
   <br/>
   "Kul hakkını Mevlâ bile bağışlamazken, kadılar bu selâhiyeti nereden alır? Tiz o kadı bulunup huzurumuza getirile!"
   <br/>
   <br/>
   Baltacılar mahkemeye yollanırken, Bostancıbaşı, Padişah hükmünü alıyor:
   <br/>
   <br/>
   "Adamlarını topla. Ev ev bütün şehri dolaş. Kadılardan ve mahkemelerden şikâyetçi olanları tek tek tespit et. Sonra  da gel bana bildir. Bildir ki, bozuk mizaçların kârını itmam idub adaleti tekrar mülkün temeli yapalum."
   <br/>
   <br/>
   Hakimlerin bozulması adalet terazisinin bozulması demekti; adalet terazisinin bozulması ise mülkün zevaline delildi. En şiddetli tedbirleri alacak, devri saltanatında mülkün zeval bulmasına izin vermeyecekti.
   <br/>
   <br/>
   Padişah buyruğunu alan Bostancıbaşı birkaç gün içinde tahkikatını tamamlayıp Padişah huzuruna çıkıyor. Hazırladığı listeyi sunuyor. Padişahın anlıyor ki mahkemelerden ve kadılardan yana yoğun şikayetler var. Yüreği kavruluyor Padişahın, inliyor:
   <br/>
   <br/>
   "Biz bitmişiz!"
   <br/>
   <br/>
   Bursa'ya döner dönmez tüm beylere hitaben bir ferman yazdırıyor:
   <br/>
   <br/>
   "Kalenüzde, yahut şeherünüzde, yahut keryenüzde, şer'i şerife mugayir hareket ittiği, rüşvet ile hükmettiği şuyu bulmuş kadıların derderst Beyşeheri'ne gönderilmesi... fermanımızdır."
   <br/>
   <br/>
   Veziri âzâm Çandarlı Ali Paşa Padişaha, kadıların suçu sabit olması halinde ne yapacağını sorunca, yüreğini ürperten bir cevap alıyor:
   <br/>
   <br/>
   "Adâletin bozulması mülkün zevaline işarettir. Mülkümüzün zevalini hazırlayan kadıları bir eve doldurup evi ateşe vereceğiz!"
   <br/>
   <br/>
   Hüküm korkunç! Başta Çandarlı olmak üzere bütün vezirler telâşta. Ama genç Padişaha o anda itiraz edip söz dinletmeye imkân yok.
   <br/>
   <br/>
   Böyle durumlarda Padişaha söz söyleyebilecek tek kişi vardır: Habeşli maskara. O, komik hareketlerle konuyu yumuşatıp Padişahı eğlendirirken bazı doğruları söylemekte ustadır.
   <br/>
   <br/>
   Çandarlı Paşa, Habeşliyi bulup derdini anlatıyor.
   <br/>
   <br/>
   "O iş kolay" diyor Habeşli, "şimdi hallederim."
   <br/>
   <br/>
   Yol kıyafetini giyip huzura çıkıyor. Yıldırım Padişah, Habeşli maskarayı yol kıyafetinde karşısında görünce, gülmekten kendini alamıyor. Sonra da soruyor:
   <br/>
   <br/>
   "Bre maskara yolculuk mu var?"
   <br/>
   <br/>
   "Beli Hünkârım, gitmek için ruhsat dilemeye geldim."
   <br/>
   <br/>
   "Nereye?"
   <br/>
   <br/>
   "Bizans'a."
   <br/>
   <br/>
   "Ne yapmaya?"
   <br/>
   <br/>
   "Bizans'tan Bursa'ya yüz papaz getirmeye gidiyorum, Hünkârım."
   <br/>
   <br/>
   Padişahın kaşları kalkıyor:
   <br/>
   <br/>
   "Bre Köle" Müslüman mülkünde papazın işi ne?"
   <br/>
   <br/>
   "Kadılık edecekler Şevketlüm."
   <br/>
   <br/>
   Padişah işin  özünü ve özetini anlar gibi. Fakat bir yandan da sohbetin ne şekilde gelişeceğini, sonunun nereye varacağını merak etmekte; tekrar soruyor:
   <br/>
   <br/>
   "Ya bizde kadılık edecek âdem yok mudur da papaz getiriyorsun?"
   <br/>
   <br/>
   "Sayenizde kalmayacak Hünkârım. Kadıları yakacağınıza göre, bari dâvâlarımıza papazlar baksın da ümmetin işi aksamasın. Malûm, kadılık ilim işidir: Eh, papazlar da bir nevi âlim sayılır."
   <br/>
   <br/>
   Hünkâr hükmün ağırlığı altında ezilerek gülmeye çalışıyor.
   <br/>
   <br/>
   "Tamam tamam vaz geçtik. Belli ki ifrat etmişiz. Söyle seni huzurumuza gönderen vezirlerimize müsterih olsunlar."
   <br/>
   <br/>
   Sadece suçluların cezalandırılmasıyla yetiniyor...
   <br/>
   <br/>
   Bu bir derstir. Dersini alan Padişah kurmaylarına danışıp rüşvete çare arıyor. Rüşvet kapısını kapatmak için tarihimizde ilk defa "mahkeme rûsumu" adı altında davayı kaybedenlerden alınmak üzere bir ücret konuyor. Hakimlere bu paradan pay verilmeye başlanıyor.
   <br/>
   <br/>
   Değil yetişmiş insanların, yerine göre bir kölenin doğrularına bile sahip çıkmak, "Hikmet mü'minin yitiğidir" diyen bir dine sımsıkı sarılı Osmanlı Devleti'ni altıyüz sene imparatorluk burcunda tutan sırrın belki de özüdür.
   <br/>
   <br/>
   Ve birkaç uyumsuz yüzünden bir camianın cezalandırılamayacağı, ayrıca devletin her zaman haklı olamayacağı prensibi de adâlet anlayışının temelidir.
   <br/>
   <br/>
   * * *
   <br/>
   <br/>
   Aynı Padişah, bir başka zaman kendisi mahkeme huzuruna çıkıyor. Yer, Başkent Bursa. Emir Sultan Hazretleri de Bursa Kadısı. Yıldırım Bayezid şahitlik edecek.
   <br/>
   <br/>
   Evvela hüviyet tespiti. Ardından Bursa Kadısı Emir Sultan'ın gürül gürül sesi:
   <br/>
   <br/>
   "Padişahum: Terki cemaat baisi cerh idüğün şuyu bulmağilen... şehadetün caiz değildür."
   <br/>
   <br/>
   Yani. "Namazlarını cemaatle kılmadığın söylendiği için şahitliğini kabul etmiyorum.
   <br/>
   <br/>
   Osmanzâde Taib'in Hadikat'üsselâtin isimli eserine göre: "Hünkâr, sarayı hümayünları pişgâhında bir camii şerif bina idüb evkâtı hamsede cemaate müdavemet buyurdular."
   <br/>
   <br/>
   Evet ya: "Padişah, sarayının avlusuna bir cami yaptırdı ve beş vakit namazını burada cemaatle kılmaya başladı."
   <br/>
   <br/>
   Ancak ondan sonra şahitliği kabul edilmiş olmalı.
   <br/>
   <br/>
   * * *
   <br/>
   <br/>
   Ne diyor bazıları: "Geçmişe mazi" diyorlar.
   <br/>
   <br/>
   Peki Thucydides ne diyor? "Tarih sürekli bir başlangıçtır."
   <br/>
   <br/>
   Kim haklı dersiniz?
