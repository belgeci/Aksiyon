# B2B danışmanlığı

*Yasin Kesen 2000.12.16*

<div class="news-detail-text-todays">
 <div>
 </div>
 <div>
 </div>
 <div id="newsSpot">
  <font class="detail-spot">
   Genelde ABD'deki girişimlerden bahsettiğimizin farkındasınızdır. Teknolojiyi en iyi kullanan ülkelerdeki modelleri gözlemleyip, kendimize göre uyarlayıp uygulamaya koyma metodunun "Amerika'yı yeniden keşfetmek"ten daha akıllıca olacağı gerçeği, bu ısrarımızın temelini oluşturuyor.
  </font>
 </div>
 <div id="newsText">
  <font class="detail-text">
   Bu hafta ise teknolojinin diğer merkezi Uzak Doğu'ya yöneleceğiz. Özellikle donanım piyasasının can damarını oluşturan Japonya, Kore, Tayvan, Singapur gibi ülkeler, milli gelir seviyeleri ile de zaten en önde gelen devletler arasında. Dolayısıyla teknolojik başarılara imza atan bu ülkelerde olup bitenleri de yakından takip etmek gerek.
   <br/>
   <br/>
   Japonya'da kurulan Emarket Concepts Asia Inc. adlı firma internet üzerinden B2B modeli hizmet veren firmalara danışmanlık yapmak amacını taşıyor. (B2B: Business to Business, yani sadece firmalara mal veya hizmet pazarlama modeli)
   <br/>
   <br/>
   Teknoloji firmalarının birçoğu artık internet üzerinden dağıtıcılarına veya bayilerine ulaştığı için B2B modelini aktif olarak kullanıyor. Lakin, bu modelin etkin kullanılması sırasında birçok firma da zorluklarla karşılaşıyor, çünkü bu model halen herkesin zihninde belli bir kavrama oturabilmiş değil.
   <br/>
   <br/>
   CCI ve Nec firmalarının 10 milyon Japon Yeni (100 bin dolar) sermaye ile kurduğu şirketin başına IBM, Compaq ve Mitsui gibi önde gelen teknoloji firmalarında üst düzey yöneticilik yapmış bir isim getirilmiş.
   <br/>
   <br/>
   İnternet hizmetleri konusunda Japonya'nın haricinde diğer Asya ülkelerine de danışmanlık hizmeti vermeyi hedefleyen Emarket Concepts danışmanlık firması, 2002 yılına kadar 2.5 milyar dolarlık kâr hedefliyor. 2003 yılında ise bu rakamı 5 milyar dolara kadar çıkarmayı planlıyor.
   <br/>
   <br/>
   Firmanın hizmet vereceği konular genel olarak:
   <br/>
   <br/>
   Eticaret firması kuruluş danışmanlığı
   <br/>
   <br/>
   İş sahası ve kâr modelleri danışmanlığı
   <br/>
   <br/>
   Pazar araştırması
   <br/>
   <br/>
   Sistem kurulumu danışmanlığı
   <br/>
   <br/>
   Emarket Concepts, bunların haricinde hali hazırda internet üzerinden işlem yapan ancak bunu etkinleştiremeyen firmalara da danışmanlık hizmeti verecek.
   <br/>
   <br/>
   Darısı başımıza...

   <br/>

   <br/>
   <br/>
   Sanal sözlükler
   <br/>
   <br/>
   İnternet gezginlerinin çektikleri sıkıntıların başında yabancı dil eksikliği geliyor (Teknik problemleri aştıktan sonrasını kasteddim, yoksa bağlantı hızı herkesin sorunu). Yabancı dili olanlar da bilmedikleri kelimeler karşısında zaman zaman takılabiliyor. İşte böyle durumlarda sanal sözlükler web sörfçüleri için can simidi oluyor.
   <br/>
   <br/>
   İşte size bir kısım sanal sözlükler:
   <br/>
   <br/>
   Seslisozluk.com
   <br/>
   <br/>
   Sitenin en başında arama kutucuğuna anlamı öğrenilecek kelime yazılıp hangi dilden hangisine çevrileceği seçiliyor. Kelimenin okunuşunu duymak isteyen ayrıca yandaki "Sesli" kutucuğunu da seçiyor. Ara komutu verildikten sonra, istenen kelimenin anlamları, kullanılma şekilleri ve o kelimenin geçtiği deyimler listeleniyor.
   <br/>
   <br/>
   Bu site ayrıca, çeşitli sözlüklerin satışını da yapıyor.
   <br/>
   <br/>
   Hazar.com
   <br/>
   <br/>
   İngilizce, Almanca, Fransızca, İtalyanca, İspanyolca, Rusça ve Japonca dillerine kelime çevirisi yapılabiliyor. Bunların haricinde bilgisayar terimleri sözlüğü de bulunuyor.
   <br/>
   <br/>
   http://hattusas.cs.bilkent.edu.tr/Sozluk
   <br/>
   <br/>
   Yaklaşık 55.000 sözcük, 90.000 sözcük anlamı ve 30.000 kadar da kullanım örneği bulunuyor. TDK'nın 1988 yılında çıkardığı sözlükten yararlanılarak hazırlanan bu sözlük Bilkent Üniversitesi tarafından sağlanıyor.
   <br/>
   <br/>
   http://www.kadifeli.com/cgibin/compdict.pl
   <br/>
   <br/>
   Bilgisayar terimleri de artık bir dil sayılabileceği için sözlük ihtiyacı doğuruyor. Bu sayfa bilgisayar terimleri için zengin bir arşiv niteliğinde.
   <br/>
   <br/>
   Yourdictionary.com
   <br/>
   <br/>
   Eğer ülkesinin adını bile zor telaffuz ettiğiniz dillerle karşı karşıya kalırsanız, başvuracağınız adres bu site olmalı. Site 200'den fazla dildeki tüm sözlüklere link veriyor. Bu site aracılığı ile en çok ulaşılan diller şöyle: İngilizce, İspanyolca, Fransızca, Almanca, Japonca, Çince, Hinduca, Arapça, Rusça ve Thai dili.
   <br/>
   <br/>
   Sitede sözlükler haricinde, kelime oyunları ve kütüphane de bulunuyor. Sitenin sık ziyaretçileri için de her gün bir kelime ön plana çıkarılarak kullanım şekilleri anlatılıyor.
   <br/>
   <br/>
   Teknik terimler bölümünde de hukuk, işletme, finans, tıp, bilgisayar ve spor terimleri başta olmak üzere 50'den fazla konuda özel sözlük bulunuyor.
