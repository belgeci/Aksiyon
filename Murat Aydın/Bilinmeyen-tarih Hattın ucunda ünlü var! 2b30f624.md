# Hattın ucunda ünlü var!

*Murat Aydın Bilinmeyen-tarih*

 <font class="content">
  Bir telefon şirketi lüzumlu—lüzumsuz anonslar için ünlülerin sesini kullanıyor ya... Bu esprileri de beraberinde getiriyor. Alp Ekin kardeş konuyla ilgili bir metin göndermiş, nerede yayınlandığını bilemiyorum ama gülmekten perişan oldum:
 </font>
 <br/>
 <p>
  <font class="content">
   Alaattin Çakıcı: Alooo, yanlış numara çevirdin aslanım. Bize  yanlış olmaz. Fıtık etmeyin lan beni!.. Hadi naşşş..
   <br>
    <br>
     Kamer Genç:  Kapsadığınız numara arama alanı dışındadır. Yok. Olmadı. Yani aradığınız alan kapsama numarası dışındadır. Yok bu da değil. Kapsama numarası.. Eeeh.. Ben çiçek sulamaya gidiyorum ya. Aradığınız yönde bütün çiçekler sulanmaktadır, işte o kadar.
     <br>
      <br>
       Mustafa Denizli: Mustafa Denizli— Aradığınız numara yanlış  diyorsam yanlıştır. Bana ne medyanın ne dediğinden kardeşim. Kafamı bozmayın sadece Fenerbahçe’yi arayanları bağlarım işte o kadar. Bu takımın patronu benim.
       <br/>
       <br/>
       Mustafa Topaloğlu: Aradığınız numara sistemimizde yer almamaktadır. Hatta güneş sistemimizde bile yer almamaktadır. Hasta mısınız kardeşim?
       <br/>
       <br/>
       Nuriş Kardeşler: Aramakta olduğunuz kişinin telefonu meşguldür ama isterseniz biz hemen bir iki eleman yollayıp bu meşguliyet durumunu  ortadan kaldıralım. Onu kontör manyağı yapalım... Bu arada, alooooo boyalı saçlım, sanal Alaattinim bittin oğlum sen...
       <br/>
       <br/>
       Selçuk Parsadan: Aradığınız numara yanlıştır. İlk görüş günü gelin de size numara nasıl olurmuş göstereyim. Bende daha ne numaralar var. Bu arada Tansu’ya söyleyin paşanın selamı var (hihohahaha!).
       <br/>
       <br/>
       Hakan Şükür: Eksik rakam çevirdiniz. Bu numaralar jiplerimin plakalarına bile yetmez. Elin oğlu genetik şifre çözüyor, siz daha bir numara çeviremiyorsunuz. Vatan hainleri.
       <br/>
       <br/>
       Demirel: Eksik rakam çevirdiniz. Binaenaleyh telaşa mahal yoktur. Devlet böyüktür. Tuşlar basmakla aşınmaz. 2 tuşu basmıyorsa 1’e 2 kere basmakta fayda vardır.
       <br/>
       <br/>
       İbrahim Tatlıses: Aradığınız kıroya şu anda ulaşılmıyor. Ben de ariyem ariyem bir türlü ulaşamiyem... Şeappiii.. The number you’ve dialed cannot be reached at the moment. Ne dedim üle ben?
       <br/>
       <br/>
       Sergen: Aradığınız numara yanlıştır. Ben sizin yerinizde olsam 2’yi son ayakta tek geçerdim. Numarayı Mustafa Hoca’dan aldıysanız çağ dışıdır.
       <br/>
       <br/>
       Alpay: Aradığınız numara yanlıştır. Yav, daha bir numarayı çeviremiyorsunuz sonra da yumruk attım diye bana laf atıyorsunuz.
       <br/>
       <br/>
       M.Ali Erbil: Enee... Aradığınız kişi kapsama alanı dışında belki de Çanakkale’dedir. Orada değilse Kıbrıs’tadır. Biraz bekleyip tekrar arayın. Parmaktan sonra...
       <br/>
       <br/>
       M. Ali Ağca: La şantemi kantareeee... Yav,   memleketten gittik gideli neler çıkmış be. Hapishane bile cep telefonu dolu. Aradığınız numaranın ne olduğunu bilmiyorum. Bekir Çelenk’in ölümüyle birlikte bütün sırlar da gitti. Odamda bir kara Fatima gördüm az önce. Acele onun sırrını çözmem lazım. Şimdi telefonu kapatın ve beni yalnız bırakın. Sıkıldım. Hadee.
       <br/>
       <br/>
       Akif’le görüştük
       <br/>
       <br/>
       Çağdaş İslam, En Bi Gerçek İslam tartışmaları bir yerlerde sürüyor ya, biz de konuyla ilgili olarak bu fikri mücadeleyi 70 sene önce vermiş Akif merhumla görüşelim dedik.
       <br/>
       <br/>
       Bakın bize neler söyledi:
       <br/>
       <br/>
       Bakın ne günlere kaldık: Ya beş, ya altı kopuk,
       <br/>
       <br/>
       Yamaklarıyla beraber ki hepsi kılkuyruk,
       <br/>
       <br/>
       Utanmadan çıkıyor, içtihada kalkışıyor!
       <br/>
       <br/>
       Bu hâle karşı tahammül hakîkaten pek zor.
       <br/>
       <br/>
       Harîm—i şer’i mübînin ahır değil... Oradan
       <br/>
       <br/>
       Çekil de kendine bir sâha bul, behey nâdan!
       <br/>
       <br/>
       Kilitli bir kapı var orta yerde anlasana:
       <br/>
       <br/>
       Harem—sarây—ı şerîat değil dalan dalana.
       <br/>
       <br/>
       Nasıl ki her kapının ayrı bir anahtarı var,
       <br/>
       <br/>
       Onun da var. Bunu idrak eder birinci nazar.
       <br/>
       <br/>
       Nedir mi? Anlatayım: Sizde olmayan irfan.
       <br/>
       <br/>
       Biraz hayâ edin öyleyse şaklabanlıktan!
       <br/>
       <br/>
       Kilitlidir kapı “ümmi duhât” için, amma
       <br/>
       <br/>
       Kıyam—ı haşre kadar içtihad eder “ulemâ”.
       <br/>
       <br/>
       Evet, şerâiti mevcûd olunca insanda;
       <br/>
       <br/>
       Ne kaldı men’edecek içtihadı, meydanda?
       <br/>
       <br/>
       İle’l—ebed yetişir müçtehid bu ümmetten;
       <br/>
       <br/>
       Şu var ki: Çıkmalı ferdâ—yı nûra zulmetten.
       <br/>
       <br/>
       Kıyas—ı fâside bir kerre eyleyin dikkat:
       <br/>
       <br/>
       Süveyş’i açtı heri... Doğru.. Neyle açtı fakat?
       <br/>
       <br/>
       Omuzlamakla mı? Heyhat! Öyle bir fenle,
       <br/>
       <br/>
       Ki, bir ömür telef etmiş o fenni tahsile.
       <br/>
       <br/>
       Düşünmüyor bu kopuklar ki, müçtehid geçinen,
       <br/>
       <br/>
       Zamanın olacak muktedası irfânen.
       <br/>
       <br/>
       Kitâb’ı, Sünnet’i, İcmâ’ı sağlam anlayacak,
       <br/>
       <br/>
       Hilâfı yoklayacak, ihtiyacı kollayacak.
       <br/>
       <br/>
       Ne içtihadı yapar, yoksa, bir alay—zımmi,
       <br/>
       <br/>
       Kadar nasîbe—i fıkhîsi olmayan— ümmi,
       <br/>
       <br/>
       Kuzum, eşek nalı yapsan: Bir usta çingenenin,
       <br/>
       <br/>
       Yanında uğraşacaksın, başında mengenenin.
       <br/>
       <br/>
       Peki! Liyâkât—i fıtrîsi âdemin sâde,
       <br/>
       <br/>
       Kifâyet eylemiyorken bu en hasis işde,
       <br/>
       <br/>
       Ya içtihada nasıl kalkıyor bu sersemler?
       <br/>
       <br/>
       O içtihada ki: Dünya kadar ulûm ister!
       <br/>
       <br/>
       Sokarsa burnunu herkes düşünmeden her işe,
       <br/>
       <br/>
       Kalır selâmet—i milliyemiz öbür gelişe.
       <br/>
       <br/>
       İşin reculleri kimlerse çıksın orta yere,
       <br/>
       <br/>
       Ne var ne yok bilelim, hiç değilse, bir kerre.
       <br/>
       <br/>
       Sabahleyin mütefelsif, ikindi üstü fakîh,
       <br/>
       <br/>
       Sular karardı mı pek yosma bir edîb—i nezih,
       <br/>
       <br/>
       Yarın müverrih; öbür gün siyasetin kurdu;
       <br/>
       <br/>
       Bakarsın, ertesi gün içtihada pey vurdu!
       <br/>
       <br/>
       Hülâsa bukalemun fıtratında züppelerin,
       <br/>
       <br/>
       Elinde maskara olduk... Deyin de hükmü verin!
       <br/>
       <br/>
       Bak şimdi
       <br/>
       <br/>
       Diyanet Abi’nin bazı sözlerinin yankısı bitmiş değil. Üç—beş günlüğüne arazi olmuştuk, peşimizi oralarda da bırakmamıştı. “Müslüman ülkelerin şiddet, işkence, çağdışı görüntülerden oluşan kötü imajını silme görevini Türkiye’nin üstlendiğini” söylemiş. Güzel de, bakınız yaz geldi geçiyor Kur’an Kurslarında ilkokul çocukları elif cüzü öğrenemiyor. Hiç savtınızı duymadık. Olmuyor ama. Hoş görünmeye çalışmak havada kalıyor. Hafızlık müessesesi bir garip gonca gibi kuruyor ama Diyanetin bu konuyla ilgili bir teşebbüsünü duyamıyoruz. Ne acîb değil mi?
