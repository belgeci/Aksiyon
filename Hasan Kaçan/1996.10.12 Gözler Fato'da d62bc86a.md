# Gözler Fato'da

 <p>
  <font class="content">
   Bir zamanlar Türk filmlerinin unutulmaz yıldızlarından olan Fatma Girik, değişime uğrayıp da “Fato” olunca, insanların korkulu rüyası haline geldi.
  </font>
  <!--- Baslik, Spot, Resim kutusu ---------->
  <p>
   <font class="content">
    Her programında yargısız infaz yapmaya başladı. Geçtiğimiz hafta mahkemece yasaklanmasına rağmen, haklılığını kanıtlamak için yanına “birkaç iyi adam”ı da alarak, yaptığı işi kendince ahlaki bir zemine oturttu ve bir Kur’an kursunda geçen “çirkin olaylar”ı aktararak kamuoyunu aydınlattı.
    <br>
     <br>
      Hem insanları kanunsuzlukla suçla, hem de mahkeme kararını hiçe sayarak kanunları çiğne.
      <br>
       <br/>
       Eski Türk filmlerinde sevgi, aşk,  ahlâk ve adalet kavramları işlenirdi. Yeni temalarımız; adaletsizlik ve iftira.
       <br/>
       <br/>
       Hanimiş dipnot: Programa görüşlerini bildiren Cumhuriyet gazetesi yazarı Hikmet Çetinkaya’nın “Kur’an kurslarında cumhuriyet düşmanları yetiştiriliyor” sözü olmasa yüzüm asık kalacaktı. Cumhuriyet yazarının bu sözleri beni gülümsetti.
       <br/>
       <br/>
       GİGA!
       <br/>
       <br/>
       Her şey karlı bir kış günü başlamıştı. Televizyondaki ses; “Evet, otuz kupon değerinde süper kupon” diye bağırdığında, güzel Türkçemize bir turist kelime daha girmiş oluyordu. “Süper kupon”un açtığı yoldan 40 kuponluk Mega’lar, 50 kuponluk Ultra’lar ve 60 kuponluk Ultra—Mega’lar gizlice sızdı lügatimize.
       <br/>
       <br/>
       Az zamanda Türkçemize müthiş kelimeler kazandıran “birtakım” medyamız son bombayı patlattı: “GİGA!”
       <br/>
       <br/>
       Evet yanlış duymadınız, 160 kupon değerinde “Giga” kupon. “Bu fırsat her zaman ele geçmez!”.
       <br/>
       <br/>
       Pekii, bu “Giga” kelimesinin menşei nedir acaba?
       <br/>
       <br/>
       Eski Yunanca mı, Sanskritçe mi, Farsça mı?
       <br/>
       <br/>
       Kuponlar “Giga” laşıp devleştikçe, bizim zavallı beyinlerimiz de bu bombardıman karşısında cüceleşiyor.
       <br/>
       <br/>
       NETLİK AYARI
       <br/>
       <br/>
       Televizyonlarında dekodersiz “Cine—5”i seyretmeye çalışanların yüzüne bir bakın hele.
       <br/>
       <br/>
       Nasıl da gözlerini kısıp bir şeyler görmeye çalışıyorlar.
       <br/>
       <br/>
       O kargacık burgacık çizgiler arasından futbolcuları seçmeye çalışıyorlar.
       <br/>
       <br/>
       Arada bir, hiç bir şey görmedikleri halde, “Aah, o da kaçar mıydı!” diye bağrışıveriyorlar.
       <br/>
       <br/>
       Böyle durumlarda, insanların hayıflanmalarına bakınca aklıma âmâ kardeşlerimiz geliyor.
       <br/>
       <br/>
       Bizler Cine—5 ekranında maçı göremediğimize üzülürken, onlar bir ömrü görmeden geçiriyor.
       <br/>
       <br/>
       Aralarında, bu eksiklerine rağmen üniversite mezunu, hatta master, doktora tezi seviyesinde eğitim almış olanlar bile var.
       <br/>
       <br/>
       Allah’ın bize lüftettiği iki gözün kıymetini bilelim.
       <br/>
       <br/>
       O gözlerle kötü şeylere bakıp da canımızı sıkmayalım.
       <br/>
       <br/>
       “Her şey o kadar çirkinleşti ki etrafa bakamıyorum.”
       <br/>
       <br/>
       Kör olsaydın daha mı iyiydi.
       <br/>
       <br/>
       Çirkini görme o zaman.
       <br/>
       <br/>
       Eğer insan istese aynı “Cine—5” gibi gözünde çirkin’i görmeyip, sadece güzellikleri görecek bir şifre geliştirebilir.
       <br/>
       <br/>
       İTİRAF
       <br/>
       <br/>
       Eğilin, kulağınıza bir şey söyleyeceğim... Eğilin, eğilin...
       <br/>
       <br/>
       Ben... ben... benim artık bir cep telefonum var biliyor musunuz?
       <br/>
       <br/>
       İki yıllık direniş sonunda teknolojiye pes ettim.
       <br/>
       <br/>
       Kanal 7’ye çekeceğimiz yeni program dolayısı ile sürekli dışarıda olacağımız için Fakir’e bir cep telefonu verdiler zorla.
       <br/>
       <br/>
       Ama neticede artık bir cep telefonum var.
       <br/>
       <br/>
       İsterseniz küsebilirsiniz bana!
       <br/>
