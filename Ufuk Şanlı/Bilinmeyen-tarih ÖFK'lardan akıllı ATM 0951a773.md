# ÖFK'lardan akıllı ATM

*Ufuk Şanlı Bilinmeyen-tarih*

 <font class="content">
  ÖFK"lar ATM hizmetinin yaygın olamamasından
  <br>
   dolayı potansiyel müşterilerini bile başka bankalarla
   <br>
    çalışmak zorunda bırakıyordu. Para yatırmak ve
    <br>
     çekmek için şubeyi kullanan müşterilerin bu konuda
     <br/>
     yaşadığı sıkıntıyı önce şube sayısını artırarak
     <br/>
     çözmeye çalışan ÖFK'lar şimdilerde rotayı
     <br/>
     bankamatiklere çevirdi
    </br>
   </br>
  </br>
 </font>
 <br/>
 <p>
  <font class="content">
   Türkiye'de ilk kez 1987 yılında kullanılmaya
   <br>
    başlayan ATM (Automated Teller Machine) ya da
    <br/>
    bilinen adıyla 'bankamatikler' artık günlük
    <br/>
    yaşamımızın vazgeçilmez bir parçası haline geldi.
    <br/>
    Bireysel bankacılığın bu vazgeçilmez
    <br/>
    enstrümanından uzun süre yararlanamayan Özel
    <br/>
    Finans Kurumu (ÖFK) müşterileri ise 2002 yılından
    <br/>
    itibaren artık 'ATM'li' olacaklar. Yaklaşık 500 bin
    <br/>
    ÖFK müşterisinden ilk etapta sadece bu konuda
    <br/>
    yatırım yapan finans kurumlarının müşterileri
    <br/>
    yararlanacak, bir sonraki aşamada ise ortak kullanım
    <br/>
    yani 'Altın nokta' uygulumasına geçilecek.
    <br/>
    <br/>
    <br/>
    ÖFK"lar ATM hizmetinin yaygın
    <br>
     olamamasından dolayı potansiyel müşterilerini bile
     <br>
      başka bankalarla çalışmak zorunda bırakıyordu.
      <br/>
      Para yatırmak ve çekmek için şubeyi kullanan
      <br/>
      müşterilerin bu konuda yaşadığı sıkıntıyı önce
      <br/>
      şube sayısını artırarak çözmeye çalışan ÖFK'lar
      <br/>
      şimdilerde rotayı bankamatiklere çevirdi. Özellikle
      <br/>
      para çekme ve yatırma işlemlerinde yaşanan
      <br/>
      sıkıntıyı gidermeyi amaçlayan Özel Finans
      <br/>
      Kurumları"ndan Family ve Kuveyt Türk bu konuda
      <br/>
      düğmeye bastı. 2003 yılına kadar ciddi olarak
      <br/>
      yapılanmayı düşünen bu iki kurum aynı zamanda
      <br/>
      ATM'lerini diğer finans kurumu müşterilerinin de
      <br/>
      kullanımına açmaya hazırlanıyor. Böylece
      <br/>
      müşteriler daha geniş bir alandan hizmet alma
      <br/>
      imkanına kavuşarak farklı bankaların
      <br/>
      ATM"lerinden para çekebilecek, finans kurumları ise
      <br/>
      bu yatırımın maliyetini azaltmış olacaklar. Krizin
      <br/>
      ağır tahribatını yavaş yavaş üzerlerinden atarak
      <br/>
      geleceğe dönük plan yapmaya başlayan finans
      <br/>
      kurumları rakiplerini şaşırtacak adımlar atmaya
      <br/>
      kararlı görünüyorlar.
      <br/>
      <br/>
      <br/>
      İhlas Finans'ın iflası ile büyük bir
      <br>
       sıkıntıya düşen ÖFK'lar hem parasal olarak hem de
       <br>
        imaj açısından zor günler geçirmelerinin ardından
        <br/>
        kaybettikleri müşterilerine yeniden ulaşabilmek ve
        <br/>
        bankacılık sistemi içindeki paylarını artırabilmek
        <br/>
        için düğmeye bastı. ÖFK'lar bireysel bankacılığa
        <br/>
        ağırlık verince ATM makinaları da kaçınılmaz
        <br/>
        olarak gündeme geldi. ATM makinaları üzerinden
        <br/>
        yapılabilen işlemlerin sayısının artması ve
        <br/>
        bankamatik olarak anılan bu makinalarda yapılan
        <br/>
        işlemlerin maliyetlerinin şubeden yapılan işlemlere
        <br/>
        nazaran daha ucuz olması ise şubeleşme konusunda
        <br/>
        sıkıntı yaşayan ÖFK'ların bu konuya daha da ciddi
        <br/>
        eğilmesine neden oldu.
        <br/>
        <br/>
        <br/>
        Daha ucuza daha çok işlem
        <br>
         <br/>
         Türkiye'de ilk dönemlerde sadece para çekmek
         <br>
          için kullanılan ATM'ler aradan geçen zaman içinde
          <br>
           havale işlemlerinden, borsadan hisse senedi almaya
           <br/>
           kadar pek çok avantajı da beraberinde getirdi. Çeşitli
           <br/>
           bankacılık hizmetlerine olanak sağlaması ve her
           <br/>
           yerde ulaşılabilen küçük lokasyonlar ile küçük bir
           <br/>
           şubeye dönüşen ATM makinaları şubeleşme
           <br/>
           konusunda sıkıntı yaşayan finans kurumları için
           <br/>
           yeni bir dağıtım kanalı haline geldi.
           <br/>
           <br/>
           <br/>
           Family Finans Genel Müdürlüğü"nün
           <br>
            yanısıra Özel Finans Kurumları Birliği"nin de
            <br>
             başkanlığını yürüten Can Akın Çağlar, alternatif
             <br/>
             dağıtım kanallarının şubeleşme konusunda
             <br/>
             sıkıntı yaşayan ÖFK'lara yeni olanaklar
             <br/>
             sağladığına dikkat çekiyor. Çağlar, "ÖFK'ların
             <br/>
             çoğunun yeterli sayıda şubeye ulaşamaması
             <br/>
             müşteriler için ciddi sorunlar doğuruyor. Buna
             <br/>
             karşın teknolojinin getirdiği olanaklar çerçevesinde
             <br/>
             ÖFK'lar bu sorunu alternatif dağıtım kanallarıyla
             <br/>
             çözmeye çalışıyorlar. Bu dağıtım kanalları aynı
             <br/>
             zamanda şubeleşme konusundaki çabaları da pozitif
             <br/>
             etkiliyor" diyor.
             <br/>
             <br/>
             <br/>
             Yapılan uluslararası araştırmalara göre
             <br>
              şubeden yapılan bir bankacılık hizmeti 1.1 dolara
              <br>
               mal olurken, aynı işlemin telefonla yapılması işlem
               <br/>
               maliyetini yüzde 50 daha azaltıyor. ABD'de yapılan
               <br/>
               araştırmalara göre otomatik para çekme
               <br/>
               makinalarında (ATM) gerçekleştirilen işlemlerin
               <br/>
               maliyeti ise sadece 37 cent. İşlem maliyeti en düşük
               <br/>
               hizmet ise internet bankacılığı; buradan yapılan
               <br/>
               işlemlerin maliyeti sadece 0.13 cent. Alternatif
               <br/>
               dağıtım kanalları içinde nakit işlem yapabilen (para
               <br/>
               yatırma veya çekme gibi) tek kanal ATM'ler. Ayrıca
               <br/>
               ATM'lerin fiyatları da ekonomik krizle birlikte
               <br/>
               önemli oranda düştü.
               <br/>
               <br/>
               <br/>
               Batan bankalar finans kurumlarına
               <br>
                yaradı
                <br>
                 <br/>
                 Bu kadar marifetli ve ucuz olan bankamatikler
                 <br>
                  halen bankacılık işlemlerinin yüzde 50'sinin
                  <br/>
                  gerçekleştirildiği bir platform ama bu özellikleri
                  <br/>
                  onların maliyetlerini biraz pahalı kılıyor. 10 bin ile
                  <br/>
                  22 bin dolar arasında değişen fiyatlardan alıcı
                  <br/>
                  bulan bankamatikleri satın almakla iş bitmiyor.
                  <br/>
                  "ATM'yi satın aldıktan sonra yeni bir çok masraf
                  <br/>
                  ortaya çıkıyor. Network, para nakli, fiziksel
                  <br/>
                  lokasyon ve koruma masraflarıyla birlikte bir
                  <br/>
                  ATM'nin maliyeti Avrupa'da yapılan araştırmalara
                  <br/>
                  göre 42 bin dolara yaklaşıyor" diyen ATM üreticisi
                  <br/>
                  NCR Genel Müdürü Yalçın Orhon, Türkiye'de ise bu
                  <br/>
                  maliyetlerin 30 bin dolar civarında olduğuna işaret
                  <br/>
                  ediyor.
                  <br/>
                  <br/>
                  <br/>
                  NCR Genel Müdürü Yalçın Orhon'un
                  <br>
                   yaptığı açıklamalara göre Tasarruf Mevudatı
                   <br>
                    Sigorta Fonu'na (TMSF) devredilen bankalar
                    <br/>
                    yüzünden Türkiye'de ilk kez ikinci el ATM pazarı
                    <br/>
                    oluştu. Bankaların mallarının tasfiyesi sonucunda
                    <br/>
                    bazı banka ve finans kurumlarının batan bankaların
                    <br/>
                    ATM'lerini aldığını söyleyen Yalçın Orhon,
                    <br/>
                    Oyakbank'ın Sümerbank'tan kalan 1000 ATM ile
                    <br/>
                    önemli bir avantaj kazandığını belirtiyor. İkinci el
                    <br/>
                    ATM pazarından OYAK grubunun dışında
                    <br/>
                    yararlanan bir diğer finans kurumu da Family Finans
                    <br/>
                    oldu. Konuyla ilgili açıklama yapmak istemeseler de
                    <br/>
                    kurumun batan bankalardan satın aldığı ATM'leri
                    <br/>
                    en geç yaz sonunda hizmete sokacağı belirtiliyor.
                    <br/>
                    <br/>
                    <br/>
                    Kuveyt Türk: Altın Nokta'ya hazırız
                    <br>
                     <br/>
                     Otomatik para çekme makinalarını
                     <br>
                      müşterilerinin hizmetine sunacak bir diğer kurum ise
                      <br>
                       Kuveyt Türk Finans Kurumu. Genel Müdür Ufuk
                       <br/>
                       Uyan 2001 yılında verdikleri stratejik bir kararla 25
                       <br/>
                       tane ATM makinası satın aldıklarını açıklıyor.
                       <br/>
                       İlk etapta şubelerde hizmet verecek bankamatikleri
                       <br/>
                       daha sonra merkezi noktalara taşımak istediklerini
                       <br/>
                       anlatan Uyan, haziran ayından itibaren bu hizmeti
                       <br/>
                       müşterilerine sunacaklarını vurguluyor. Bu
                       <br/>
                       teknolojinin doğal uzantısı olan kredi kartı
                       <br/>
                       hizmetleri konusunda da finans kurumlarının uzun
                       <br/>
                       süre büyük bir sıkıntı yaşadığını hatırlatan Uyan,
                       <br/>
                       "ÖFK'lar bugüne kadar 'Third Party' olarak anılan
                       <br/>
                       'üçüncü el' yani başka kurumlar aracılığıyla kredi
                       <br/>
                       kartı temin ediyordu. Kuveyt Türk VİSA yönetim
                       <br/>
                       kuruluna başvurarak kredi kartlarını doğrudan
                       <br/>
                       temin etme yolunda önemli bir kazanım elde etmeyi
                       <br/>
                       de hedefliyor" diyor.
                       <br/>
                       <br/>
                       <br/>
                       Altın nokta veya ortak nokta olarak
                       <br>
                        anılan bankalararası ATM'lerin ortak kullanım
                        <br>
                         projesine sıcak baktıklarını söyleyen Ufuk Uyan,
                         <br/>
                         bu gelişmenin aslında bir bakıma zorunluluk
                         <br/>
                         olduğuna işaret ediyor. "Şube kurmanın maliyeti
                         <br/>
                         100-150 bin dolar arasında değişiyor; bir de bunlara
                         <br/>
                         yıllık maliyeti 40 bin doları bulan ATM'leri
                         <br/>
                         eklediğinizde bu maliyet daha da artacaktır. Biz bu
                         <br/>
                         şekilde bir paylaşıma hazırız çünkü finans
                         <br/>
                         kurumları hâlâ bankacılık sektöründe istenen payı
                         <br/>
                         alabilmiş değil. Hâlâ yapılması gereken çok
                         <br/>
                         yatırım var."
                         <br/>
                         <br/>
                         <br/>
                         <br>
                          Asya Finans Genel Müdürü Ünal Kabaca: "Altın
                          <br>
                           nokta veya ortak nokta konusunda henüz bir
                           <br/>
                           çalışmamız olmadı. İleride belki olabilir çünkü
                           <br/>
                           hâlâ bu konuda yatırımın boyutlarını ve alanlarını
                           <br/>
                           tartışabileceğimiz bir gelişme olmadı. Finans
                           <br/>
                           kurumlarının bireysel bankacılık konusunda
                           <br/>
                           rekabet edebilmeleri için ATM teknolojisi şart; ama
                           <br/>
                           anlaşamazsak bakarsınız biz de kendi ATM'lerimizi
                           <br/>
                           kurarız.
                           <br/>
                           <br/>
                           <br/>
                           <br/>
                           <br/>
                           Bankaların ortak noktaları
                           <br/>
                           Altın Nokta, 1993'te Akbank, Garanti Bankası ve
                           <br/>
                           Vakıfbank tarafından kuruldu. Daha sonra Halbank
                           <br/>
                           ve Osmanlı Bankası da sisteme katıldı. Bunlara
                           <br/>
                           Dışbank ve Koçbank da eklendi. Bu altı bankanın
                           <br/>
                           toplam ATM sayısı 3 bin 500 civarında bulunuyor.
                           <br/>
                           Orta ölçekli bankaların dahil olduğu Ortak
                           <br/>
                           Nokta'nın üye sayısı ise 15. Alternatifbank,
                           <br/>
                           Anadolubank, MNG Bank, Oyakbank, Sitebank,
                           <br/>
                           Şekerbank, Tarişbank, Tekstilbank, HSBC,
                           <br/>
                           Tekfenbank, TEB, Toprakbank, Finansbank,
                           <br/>
                           Citibank ve Denizbank 'tan oluşan grubun 1500
                           <br/>
                           ATM makinası bulunuyor.
                          </br>
