     <br>
      Kayda geçmek gerek.
      <br>
       <br/>
       Üç gelişme yaşandı geçen hafta…
       <br/>
       <br/>
       Bir:
       <br/>
       <br/>
       Ergenekon davasının 2. iddianamesi tamamlandı ve mahkemeye iletildi. İddianame henüz resmen kabul edilmemekle ve tüm yönleriyle bilinmemekle birlikte, basına yansıdığı yönleriyle ciddi ve ağır suçlamalarda bulunuyor.
       <br/>
       <br/>
       Emekli orgeneraller Şener Eruygur ve Hurşit Tolon, “terör örgütü yöneticisi olmak”, “cebir ve şiddet kullanarak TBMM’yi ve hükûmeti ortadan kaldırmaya veya görevlerini yapmasına kısmen veya tamamen engellemeye teşebbüs”, “hükûmete karşı silahlı isyana tahrik”, “devletin güvenliğine ilişkin belgeleri temin etmek ve kullanmak”, “kişisel verileri hukuka aykırı olarak kaydetmek” ile suçlanıyorlar…
       <br/>
       <br/>
       Tuğgeneral Bahtiyar Aydın ve Albay Rıdvan Özden’in öldürülmesi, Sivas ve Gazi olayları, Başbağlar katliamı, Bingöl’de 33 erin şehit edilmesi Ergenekon’a mal ediliyor…
       <br/>
       <br/>
       İddialara göre, Ergenekon Terör Örgütü Türkiye’de faaliyet gösteren başka ve birçok terör örgütünü de amaçları doğrultusunda kullanıyor.
       <br/>
       <br/>
       PKK ve DHKP-C, El Kaide, Hizbullah, Hizbu’t-Tahrir, TİT bunlar arasında.
       <br/>
       <br/>
       Bu örgütler, yine iddianameye göre, eylem ve faaliyetlerini Ergenekon yapılanmasının amaçları ve hedeflerine uygun şekilde yapıyorlar.
       <br/>
       <br/>
       İki:
       <br/>
       <br/>
       Ergenekon soruşturması kapsamında emekli Albay Mehmet Ülger gözaltına alındı. Emekli Kurmay Albay Mehmet Ülger, Malatya İl Jandarma Alay Komutanı iken, geçtiğimiz yıl görev süresinin bitmesinin ardından emekliye ayrılmıştı. Malatya’daki 3 kişinin öldürüldüğü Zirve Yayınevi davasında dinlenmesine karar verilen tanıklar arasında yer alıyor.
       <br/>
       <br/>
       Üç:
       <br/>
       <br/>
       Şırnak’ın Silopi ilçesi Sinan Tesisleri’ndeki kuyularda, 9 kemik parçası bulundu. Silopi Cumhuriyet Başsavcılığı’nın, 1990’lı yıllarda öldürülen bazı kişilerin kuyulara atıldığı iddiasıyla ilgili yaptığı soruşturma kapsamında, Silopi-Cizre karayolunun 15. kilometresinde yer alan Sinan Tesisleri’nde başlatılan kuyulardaki kazı çalışmaları devam ediyor.
       <br/>
       <br/>
       Başka kuyularda daha önce iki kemik parçasına rastlanmıştı.
       <br/>
       <br/>
       Tüm bu iddialar ve gelişmeler aslında tüyler ürperticidir.
       <br/>
       <br/>
       Son derece vahim olaylara ve vahim yapılanmalara işaret ederler.
       <br/>
       <br/>
       Ne var ki, Türkiye’nin bir kısmı, bir bölümü tüm gelişmelere sanal muamelesi yapıyor. Bir yandan ülke tarihinin en karanlık dönemleri elden geçiriliyor, en karanlık yapılarına ulaşılıyor, öte yandan bu durum ısrarla ya sıradanlaştırılmaya çalışılıyor ya da örtbas edilmeye…
       <br/>
       <br/>
       Yine geçen hafta, Ergenekon davası çerçevesinde darbe hazırlamak ve hazırlıkları içinde yer almak iddiasıyla tutuklanan gazeteci Mustafa Balbay’ın tutukluluğu, düşünce ve ifade özgürlüğü çerçevesinde telin ediliyordu.
       <br/>
       <br/>
       Birçok gazeteci, Balbay yerine kitaplarını imzalayarak okurlarına veriyordu, bir imza gününde…
       <br/>
       <br/>
       Doğru olan ne, hangisi?
       <br/>
       <br/>
       Doğru olan kafaların karıştığıdır…
       <br/>
       <br/>
       Doğru olan darbe girişimlerinin Kemalizm’den sonra şimdi de demokrasi kavramı arkasına gizlenmeye çalışmasıdır.
       <br/>
       <br/>
       Yukarıda anılan gelişmeler ve iddialar doğru olamaz kanısında bazıları. Kimimiz ordunun üst düzey subayları bu tür ilişkilere girmez, giremez, terör örgütlerini kullanamaz diye düşünür.
       <br/>
       <br/>
       Keşke öyle olsa…
       <br/>
       <br/>
       Keşke başka olaylar da öyle olsaydı.
       <br/>
       <br/>
       Örneğin, darbeler yapılmamış olsaydı.
       <br/>
       <br/>
       Yüz binlerce insan tutuklanmamış olsaydı.
       <br/>
       <br/>
       Binlerce insan işkence görmemiş, binlercesi faili meçhullere kurban edilmemiş olsaydı…
       <br/>
       <br/>
       28 Şubat’ta toplumun bir bölümü, diğer bölümü tarafından fişlenmemiş olsaydı keşke…
       <br/>
       <br/>
       Ama öyle değil, öyle olmadı…
       <br/>
       <br/>
       Bize ne kadar zor, acı, kabul edilmez gelse de bazı hâller, bazı gerekçelerle yaşandı ve yaşanmaya devam ediyor.
       <br/>
       <br/>
       Kabul edelim:
       <br/>
       <br/>
       Ergenekon davası bu karanlığın izinin sürülmesi konusunda en büyük şanstır, belki de son şanstır. Ergenekon davası ve soruşturmasının başarısızlığa uğraması ve uğratılması, kimi iddia ve dosyaları hafifleterek gerçekten bilinmezliğe ve yaptırımsızlığa itecektir.
       <br/>
       <br/>
       Seçimler, evet, önemlidir…
       <br/>
       <br/>
       Türkiye’nin ekonomik durumu önemlidir…
       <br/>
       <br/>
       Seçim sonrası ortam önemlidir…
       <br/>
       <br/>
       Ancak kanımız odur ki, en önemlisi Ergenekon meselesidir…
       <br/>
       <br/>
       Ergenekon davasının gidişatı, Türkiye’de yaşanmakta olan sivilleşmenin pratik ve politik sonuçlarını kuşatacaktır. Vesayet rejimi ile bu rejimin zemininde büyüyen devasa bir hikmet-i hükûmet anlayışı, bu yolla ya baskı altına alınacak ya da aksi durumda yeniden ve daha güçlü biçimde yapılanacaktır…
       <br/>
       <br/>
       Kemikler…
       <br/>
       <br/>
       Cesetler…
       <br/>
       <br/>
       Şehitler…
       <br/>
       <br/>
       Kimse bunları hafife almasın…
       <br/>
       <br/>
       Bunlar Türkiye’nin tarihini yazdılar; şimdi o tarihi yeniden yazma fırsatı var Türkiye’nin elinde…
       <br/>
       <br/>
       Bu tarih yazımı sanmayın ki ülkeyi ve kurumları yıpratacak, tersine güçlendirecektir…
       <br/>
