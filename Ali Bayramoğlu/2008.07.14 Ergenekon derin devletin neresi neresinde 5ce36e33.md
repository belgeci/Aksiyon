   <font class="content">
    Türkiye’de devletin içinde hep iki kademe olmuştur.
    <br>
     <br>
      Birinci kademe askerdir.
      <br>
       <br/>
       Asker, devlet ve siyasette egemenliğini yasalar, olmadı yönetmelikler ve protokollerle sağlar ya da fiili hareketler gerçekleştirir. Darbeler yapar, muhtıralar verir.
       <br/>
       <br/>
       Bu ülkede yaşayan reşit her Türk vatandaşı öykünün bu yönünü ana hatlarıyla bilir…
       <br/>
       <br/>
       Devletin içindeki, daha doğrusu devlete enformel olarak bağlı ikinci kademeye gelince, bu, seyyar vurucu gruplardan oluşan yasa dışı bir yapıdır.
       <br/>
       <br/>
       Doğal olarak kurumlardan değil, kişilerden oluşur.
       <br/>
       <br/>
       Gerektiğinde devreye girer.
       <br/>
       <br/>
       Bu ikinci kademe Osmanlı’da İttihatçılardan, Teşkilat-ı Mahsusa’dan başlayan bir öyküye işaret eder.
       <br/>
       <br/>
       Sıradan bir öykü, bir kanun dışılık, bir siyasi hata meselesi değildir bu.
       <br/>
       <br/>
       Öylesine öykü devletin derin politikalarının taşıyıcısı dahi olmuştur.
       <br/>
       <br/>
       Örneğin Anadolu topraklarının dinî açıdan standardize edilmesinde bu yapıların önemli payı görülür. Bu anlamda, derin teşkilatlar mevcut düzenin kurucuları arasında yer alırlar. Örnek pek çok…
       <br/>
       <br/>
       Balkan Savaşı sonrası Ege bölgesindeki Rumları “katledileceksiniz” söylentileriyle kaçıran, yüzbinlerle anılacak rakamlarla göçe iten Teşkilat-ı Mahsusa olmuştur.
       <br/>
       <br/>
       Ermeni tehcirinde bilinir bu yapının işlevi...
       <br/>
       <br/>
       İşlev ve yapının Cumhuriyet’le de devam ettiğini biliriz.
       <br/>
       <br/>
       1950’lerde NATO bağlantısı oluşmasıyla Özel Harp Dairesi bağlantısı ortaya çıkmıştır. Kıbrıs’ta Mukavemet Teşkilatı’nın kurulması, Irak Türkmenleri ve Balkan Türkleri arasında teşkilatlanma bu yolla gerçekleştirilmiştir…
       <br/>
       <br/>
       Aynı yapı özellikle Soğuk Savaş’ın sert yıllarında, 1970’lerde ülkeye büyük acılar yaşatmıştır.
       <br/>
       <br/>
       Kahramanmaraş, Çorum katliamlarında yüzlerce insan ölmüş, mahalleler birbirine karşı kışkırtılmıştır.
       <br/>
       <br/>
       Ardından ortaya büyük Susurluk skandalı ve arkasındaki yapı çıkmıştır.
       <br/>
       <br/>
       Şu açıktır: Osmanlı’dan Cumhuriyet’e, 60’lardan 90’lara, Maraş’tan Susurluk’a, temel olarak “engelleme” işlevi üstlenmiştir. Başka bir ifadeyle sistemin, düzenin, yerleşik ideolojinin istemediği, tehlikeli gördüklerini, siyasi hareket toplumsal kesim ayrımı yapmadan susturmak, devre dışı bırakmak, imha etmek işini yerine getirmiştir.
       <br/>
       <br/>
       Faili meçhul cinayetler, bombalı saldırılar, kışkırtmalar ortadadır.
       <br/>
       <br/>
       Gazi Mahallesi’nde bir kahvenin taranmasının yol açtığı olaylar Türkiye’nin büyük bir kentinde ayaklanmaya dönüşmüştür, örneğin…
       <br/>
       <br/>
       Bugün neler oluyor diye bakıldığında, bu yapı ve öyküden bağımsız bir durumla karşı karşıya olmadığımız açıktır…
       <br/>
       <br/>
       Dink cinayeti, Malatya’daki misyoner vahşeti, ülkenin çeşitli yerlerindeki rahiplere yapılan ölümlü, yaralamalı saldırılar kâğıt üzerinde mahalle çetelerinin işi gibi görünse de bağlantılar oldukça derinlere inmektedir.
       <br/>
       <br/>
       Dink cinayetine, cinayetin gerek emniyet gerek jandarma tarafından önceden haber alındığı hâlde, önlem alınmaması nasıl açıklanır?
       <br/>
       <br/>
       Bir Malatya zanlısının sık sık telefonla aradığı bir savcının, bir dönem Silopi’de iki HADEP’linin sırra kadem bastığı dönemde görev yapmış olması, aynı dönemde, aynı şehirde jandarma alay komutanı olan kişinin Ergenekon soruşturmasında aranıyor olması tesadüf müdür?
       <br/>
       <br/>
       Ergenekon soruşturması etrafında yaşanan gelişmeler, ortaya çıkan gerçekler, “derin devlet” olarak tabir ettiğimiz bu yapının izdüşümünden başka ne olabilir?
       <br/>
       <br/>
       Dün ile bugün arasında elbet farklar var…
       <br/>
       <br/>
       İlk fark şu: Derin devlet, düne kadar değişmeyi engelleyen vurucu bir timdi.
       <br/>
       <br/>
       Şimdi ise, derin devlet sadece değişmeyi engelleyen değil, bunun da ötesinde, toplumu ve siyaseti yeniden inşa etmeye soyunan büyük bir güç.
       <br/>
       <br/>
       Ergenekon meselesine böyle bakmak gerekir…
       <br/>
       <br/>
       Ergenekon’un amacı kimilerinin sandığı gibi sadece AK Parti’yi alaşağı etmek değildi, Türkiye’de Putinvari bir militer sistem inşa etmekti.
       <br/>
       <br/>
       İkinci farka gelince…
       <br/>
       <br/>
       Dünün derin yapıları dar alanda, dar kadrolarla hareket ederken, bugün Ergenekon’la ortaya çıkan resim bu yapının âdeta bir siyasi parti gibi hareket ettiğini, devlet içinde devlet gibi örgütlendiğini göstermektedir.
       <br/>
       <br/>
       Nitekim bugün Ergenekon dikkatle izlendiğinde ortaya farklı katmanlar çıkmaktadır. Ergenekon’un bir kademesinde vurucu timler vardır.
       <br/>
       <br/>
       Bir diğer kademesinde Anadolu’ya yayılmış binlerce üyesi olan militarize türlü dernekler bulunmaktadır.
       <br/>
       <br/>
       Bir başka kademesinde darbeye soyunan generaller vardır.
       <br/>
       <br/>
       Bir dördüncü kademesinde ise yeni otoriter bir düzeni arzulayan, kimi gazeteler, gazeteciler, iş adamları bulunmaktadır.
       <br/>
       <br/>
       Deşifre olan, tasfiye edilen ve edilecek yapı budur…
       <br/>
       <br/>
       Türkiye bu yapıyla tam olarak hesaplaştığı takdirde demokrasi istikametinde hiç almadığı kadar büyük yol almış olacaktır…
       <br/>
       <br/>
       Kaldı ki yol alınıyor…
       <br/>
       <br/>
       Emekli Genelkurmay Başkanı Hilmi Özkök’ün 2003 ve 2004 yıllarındaki darbe girişimlerini yalanlamaması, ifade verebileceğini ima etmesi bile, okun yaydan çıktığını gösteriyor…
       <br/>
